### 2.3 路径 D 关键约束声明 + 电影预告专属

⚠️ **路径 D 必加约束**：
- 不复刻分镜板的 2x2 网格边框/分隔线/格子边距
- 4 格不是同时显示的并列画面，而是**先后串联**的 4 个独立场景

⚠️ **电影预告专属约束**：
- 不走抖音/小红书 vlog 画质退化路线，走**胶片电影感画质**
- 镜头语言专业（推/拉/摇/移可用，但每段单一镜头运动不超过两轴）
- 每段独立场景但**主角同一气质 + 同一色温心境**保持连贯
- 段间转场可用慢速溶解或匹配剪辑（非硬切，预告片高级感）

## 3. 范式选择 + 拼接结构

**Hook 范式**：**电影预告范式 · 大气连贯 + 脉冲式剪辑（路径 D-1 拼接版）**

**剪辑公式**：**脉冲式（哒哒哒—哒哒哒哒—轰！）混合静默锤击** —— 前 3 段 3-3.5s 慢推 + 最后 1 段 5s 静默凝视收束

### 拼接方案 A：心境流动版（默认）

按"心境从准备 → 独处 → 等待 → 抵达"的情绪流动串联：
- **0-3.5s** = 第 1 格（公寓换衣准备出门）= **建立人物 · 起势**
- **3.5-7s** = 第 2 格（厨房独处回眸）= **建立独居孤独感 · 暗涌**
- **7-10.5s** = 第 3 格（雨夜窗边回望）= **情绪高点 · 张力**
- **10.5-15s** = 第 4 格（夜市烟雾凝视）= **氛围收束 · 余韵**

### 拼接方案 B：倒叙版（备选）

从夜市开场倒推："她为什么到这里" → 雨夜回望 → 厨房独处 → 公寓出门

→ **默认选 A**（心境流动版更适合王家卫式预告片节奏，倒叙适合悬疑片但本图氛围偏文艺）

## 6. 微调建议

如果想调整：
- **想要更激烈的预告片节奏**：把"缓慢溶解"改成"匹配剪辑"为主+少量硬切，加入帧率变化（120fps 慢动作镜头）
- **想要更安静的纯静默版**：改用静默锤击剪辑公式——第 1-3 段无 BGM 只环境音+第 4 段配乐才进入
- **想要倒叙版**：用拼接方案 B 从夜市开场，节奏更悬疑
- **想要 30s 版**：每段扩展到 6-8s+加入更多镜头细节（玻璃水珠特写/烟雾粒子慢动作/胶片划痕过场），需走路径 D-2 多条独立 prompt
- **想要去掉雨夜潮湿氛围改成晴天**：本图 4 格氛围已锁定潮湿感，硬改会破坏整体调性，不建议

## 7. 路径 D-1 电影预告范式方法论沉淀（本 case 是首个 case）

本 case 是 **路径 D-1 + 电影预告范式（非凝视模式）的首个完整实战 case**，验证以下范式假设：

### 7.1 ✅ 验证通过的设计假设

1. **路径 D-1 同时支持 vlog 和电影预告两种风格**——case-10 走 vlog，case-11 走电影预告，同样的"4 格拼接"框架可承载完全不同的范式
2. **电影预告范式不走凝视模式 + 抖音味的 4 层退化模板**——需用**史诗大制作架构**（品质锚定/大气连贯/光影三层/收束句）
3. **段间转场是范式区分关键**：vlog 用硬切（自然感）；电影预告用缓慢溶解或匹配剪辑（高级感）
4. **路径 D-1 拼接 = 路径 D 框架 + 内部范式选择**：网格规避+段间一致性是路径 D 通用约束；范式（vlog/预告/MV/动漫）是内部具体选择

### 7.2 ⚠️ 待验证的设计假设

1. **缓慢溶解 vs 匹配剪辑**哪个更适合 Seedance 的执行能力——本 case 选缓慢溶解为主，待出片验证
2. **35mm 胶片质感**能否被 Seedance 准确还原（vs 普通"电影感"泛词）——首个测试
3. **主角同一气质连贯性**——4 段独立但主角是同一人的描述能否让模型保持人物一致性
4. **静默锤击 + 脉冲式混合剪辑**是否会让节奏断裂——前 3 段稍快+第 4 段长收束，待验证
