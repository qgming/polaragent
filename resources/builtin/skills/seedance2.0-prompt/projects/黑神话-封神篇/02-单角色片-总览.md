## 🎬 整体概念

```
立项片（选角）结束后 →【白屏转场】→ 单角色片开场

单角色片（每条 10-15 秒）：
  00:00-00:03 菜单界面·呼吸感（玩家看着菜单，环境微动，菜单项闪烁）
  00:03-00:04 「繼續遊戲」高亮 + 按 A（按钮闪烁反馈）
  00:04-00:06 加载转场（UI 淡出 + 水墨笔触/金色加载条 + 短暂黑屏）
  00:06-00:15 实机游戏画面（角色第三人称 + 场景展示 + UI 弹入）

单角色片结束 → 可无缝衔接下一段剧情/玩法演示
```

**核心设计哲学**（延续潘金莲系列）：
**一张 Prompt 造出一款不存在的 3A 大作** —— 每角色有专属的菜单图 + 实机图 + 过渡视频。

## 🎬 5 条单角色片的视频结构（共用时间轴）

每条单角色片 **10-15 秒**，分 3 幕：

```
▶ 第一幕（0-3 秒）：菜单画面 · 玩家浏览
  - 画面保持在菜单图状态（首帧 = @菜单图 X）
  - 角色自然呼吸 + 衣袍微动 + 背景环境持续活动
    · 姜子牙：白须飘、封神榜金册缓慢转动、四不像呼吸
    · 二郎神：战袍微动、瀑布水花、哮天犬呼吸
    · 妲己：九尾末端轻摆、烛火摇曳、红色帷幔微动
    · 聞仲：紫电微闪、墨麒麟呼吸、符文光流
    · 紂王：冕旒流苏轻晃、血月微浮、酒面涟漪
  - 白色鼠标光标/手柄光标平滑移动到「繼續遊戲」
  - 2.5 秒处点击，按钮闪烁（金光脉冲）

▶ 第二幕（3-6 秒）：加载过渡
  - 3 秒：UI 元素逐一优雅淡出（LOGO → 菜单 → 血条 → 匾 → 诗词）
  - 4 秒：画面中央浮现繁体「載入中…」+ 金色中式纹样加载图标
  - 4-5 秒：一张水墨风加载插画短暂浮现（对应角色专属元素）
  - 5-6 秒：插画淡出 → 画面短暂漆黑 0.3 秒 → 急速变亮

▶ 第三幕（6-15 秒）：游戏实机 · 前进 + 招式
  - 6 秒：尾帧画面瞬间呈现（= @实机图 X）
  - 6-8 秒：MMO UI 元素逐一弹入
    · 头像血条滑入（从左上）
    · 任务列表从左边展开
    · 技能栏从下方升起
    · 小地图从右上角展开
    · 世界公告横向滑入
    · 聊天频道消息浮现
  - 8-12 秒：角色**持续向画面深处迈步前行**，镜头跟随推进
    · 角色背影自然大步迈动（专属步态）
    · 衣袍/披风/发丝/尾羽随步伐向后扬动
    · 场景向镜头两侧后退（透视推进感）
    · 环境持续活动（根据角色场景不同）
  - 12-14 秒：**触发该角色招牌招式**（2 秒 VFX）
    · 角色停步，释放招牌技能
    · 对应 VFX 特效爆发（详见各角色详情文件）
    · 底部技能栏对应图标高亮冷却
    · 画面整体泛起该角色主色调高光
  - 14-15 秒：招式收尾
    · VFX 淡出
    · 角色继续向前迈步
    · 世界频道滚动新消息：「〔系統〕角色釋放 XX 招式」

【5 位角色专属招式】
  · 姜子牙 → 五雷訣（金色雷符光圈）
  · 二郎神 → 天眼通（第三眼金色光柱）
  · 妲己   → 千狐陣（金色狐狸虚影环绕）
  · 聞仲   → 九天雷訣（紫色闪电从天而降）
  · 紂王   → 王氣震懾（金色龙形王气盘旋）
```

**视频 prompt 通用框架**（每角色详情文件里只改差异化部分）：
首帧 = 对应菜单图，尾帧 = 对应实机图。

## 📐 衔接设计：和立项片选角视频的关系

立项片**选中效果视频**的尾帧 = **角色就绪态纯特写（无 UI）**
单角色片**菜单图**首帧 = **角色专属菜单界面（有完整 UI）**

两者之间的衔接（剪辑处理）：

```
选角片尾帧（纯角色定格 + 金光）
    ↓
0.3-0.5s 白屏/金光爆闪过场（剪辑加 "闪白" 特效）
    ↓
0.3-0.5s 画面快速淡入
    ↓
单角色片首帧（对应菜单图完整状态）
```

**转场音效**：「咚」低频共鸣（确认键重击）+ 金属 whoosh + 风铃渐弱

