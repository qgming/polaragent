# 穿越万象 EP03：维度穿越

## 视频提示词

**主题**：一镜到底从真实世界穿过镜子进入卡通世界
**时长**：15秒  |  **比例**：16:9

---

### 版本一：镜面穿越（写实→吉卜力）

#### 提示词

```
15秒一镜到底维度穿越，前半段电影级写实摄影质感，后半段渐变为吉卜力手绘动画风格，全程不切镜头，

0-4秒：Smooth Tracking Shot跟随一个背书包的年轻人走在阳光斑驳的真实街头，梧桐树影投在人行道上，自行车经过叮铃作响，浅景深虚化背景，电影胶片颗粒质感，温暖的午后自然光；

5-8秒：年轻人走到街边一面巨大的复古穿衣镜前停下脚步，Subtle Push In缓缓推向镜面，镜中的倒影开始微妙变化——人物轮廓线条逐渐变得柔和圆润，色彩变得更加鲜艳饱和，镜面如同水面般泛起轻微涟漪波纹；

9-13秒：镜头穿过镜面涟漪，Smooth Dolly Forward推进，画面完全转变为吉卜力风格的手绘动画世界，同一条街道变成了色彩明丽的动画小镇，天空中飘着棉花糖般蓬松的白云，屋顶上有一只圆滚滚的卡通猫咪在打盹，街角花店门口鲜花盛开颜色浓郁而梦幻；

13-15秒：卡通风格的年轻人转过身来面对镜头微微一笑，身后蒲公英种子在风中飘散成光点，阳光温柔得像水彩晕染。

一镜到底全程不切镜头。禁止：任何文字、字幕、LOGO或水印。
```

#### 首帧建议

> **首帧图片生成提示词**：阳光斑驳的城市街头午后，一个背书包的年轻人走在梧桐树下的人行道上，前方街边立着一面大型复古铜框穿衣镜，温暖自然光，电影胶片质感，写实摄影风格，浅景深

#### 设计意图

用一面镜子作为两个维度的"入口"是最直觉的隐喻——镜子本身就是现实与幻想的边界。写实的午后街头 vs 吉卜力的动画小镇形成强烈的风格反差，但情绪基调保持统一的温暖治愈。

---

### 版本二：画框穿越（写实→3D皮克斯）

#### 提示词

```
15秒一镜到底维度穿越，前半段写实电影摄影画质，后半段转为皮克斯风格3D动画渲染，全程不切镜头，

0-4秒：美术馆内部，Smooth Dolly Forward推进穿过安静的展厅，白色墙壁上挂满画框，大理石地面反射天窗洒下的柔和自然光，一个年轻女孩站在一幅巨大的画作前驻足凝视，画中是一座色彩浓郁的奇幻城市；

5-8秒：Subtle Zoom In缓缓推进向画框，画中的奇幻城市开始有了微妙的动态——屋顶的风车缓缓旋转、街道上的小人开始走动、天空中飘过一只纸飞机，画面边缘画框逐渐模糊消失，画中世界的色彩开始溢出画框；

9-13秒：镜头完全进入画中，Smooth Dolly Forward继续推进，置身于皮克斯风格的3D动画奇幻城市之中，圆润可爱的建筑高低错落，色彩饱和明亮，一个穿红色围巾的卡通女孩从面前跑过追逐着那只纸飞机，屋檐上一排彩色小鸟齐刷刷转头看向镜头；

13-15秒：Gentle Crane Up轻轻升起，俯瞰整座动画城市的奇妙全景，远处天空中漂浮着鲸鱼形状的云朵，夕阳把一切染成橙粉色。

一镜到底全程不切镜头。禁止：任何文字、字幕、LOGO或水印。
```

#### 设计意图

美术馆→画中世界的穿越逻辑非常自然——"走进一幅画"是人人都能共情的幻想。画框作为维度入口比镜子更有"艺术感"，画中世界从静态到动态的渐变过程也给模型更充裕的过渡空间。皮克斯风格的3D动画相比吉卜力手绘可能对模型更友好（3D渲染风格是 Seedance 的强项）。

---

### 提示词解析

| 对比项 | 版本一（镜面穿越） | 版本二（画框穿越） |
|--------|-----------------|-------------------|
| 穿越介质 | 街头穿衣镜 | 美术馆画框 |
| 目标风格 | 吉卜力手绘2D | 皮克斯3D动画 |
| 情绪基调 | 温暖治愈、日常感 | 奇幻冒险、童话感 |
| 模型友好度 | 中等（2D手绘风格过渡难度较高） | 较高（3D渲染是Seedance强项） |
| 适合平台 | 小红书/视频号（治愈向） | 抖音/B站（视觉冲击向） |

> **迭代建议**：维度穿越的核心难点在于"风格过渡的丝滑度"。如果第一次生成时两种风格割裂感太强，可以尝试在提示词中增加过渡描述的篇幅（5-8秒区间写得更细），给模型更多渐变空间。也可以考虑用即梦的「视频编辑」能力，先生成一段写实视频，再编辑后半段为动画风格。
