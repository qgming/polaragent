# 穿越万象 EP01：时间穿越

## 视频提示词

**主题**：一镜到底从现代都市穿越到大唐长安
**时长**：15秒  |  **比例**：16:9（建议 2.35:1 宽银幕比例更有电影感）

---

### 版本一：旋转门穿越（史诗质感）

#### 提示词

```
15秒一镜到底时间穿越，8K超高清，ARRI ALEXA摄影机质感，前半段现代都市冷蓝灰色调，后半段大唐盛世暖金色调，全程不切镜头，一个连贯的穿越长镜头，

0-4秒：Smooth Dolly Forward推进穿过现代都市繁忙的雨夜街头，玻璃幕墙反射霓虹灯光，行人撑伞匆匆而过，雨后柏油路面倒映城市灯火，浅景深虚化远景，镜头前附着细密雨珠；

5-8秒：镜头推入街角一扇青铜色旋转门，门扇转动的瞬间画面光线扭曲，金色时间粒子如沙砾般在空气中漂浮旋转，旋转门内壁浮现模糊的朝代剪影，色调从冷蓝灰开始向暖金渐变；

9-13秒：穿过旋转门，Smooth Dolly Forward继续推进，画面完全转为暖金色调，出来是大唐长安朱雀大街的繁华盛景，骆驼商队缓行、绫罗绸缎铺面、飞檐斗拱的酒楼、红色灯笼高悬两侧，golden hour暖光洒满整条长街，丁达尔光线穿过飞檐间隙；

13-15秒：Gradual Crane Up缓缓升起，俯瞰长安城恢弘全景，远处大明宫巍峨耸立，万家灯火初上，黄昏暮色笼罩整座城池。

一镜到底全程不切镜头。禁止：任何文字、字幕、LOGO或水印。
```

> **首帧图片生成提示词**：现代都市雨夜街头，霓虹灯光反射在湿润的柏油路面上，行人撑伞走过，远处一扇青铜色旋转门隐约可见，冷蓝灰色调，电影级写实摄影，浅景深，ARRI摄影机质感

#### 设计意图

旋转门是一个很巧妙的穿越介质——它本身有"旋转"的物理动作，和"时空扭曲"的感觉天然契合。冷蓝→暖金的色调渐变让穿越感更强烈。

### 版本二：水坑倒影穿越（诗意梦幻）

#### 提示词

```
15秒一镜到底时间穿越，电影级写实质感渐变至中国风古典工笔画质感，全程不切镜头，

0-4秒：Smooth Tracking Shot跟随一个穿深色风衣的背影在现代雨夜街头行走，脚步踩过积水溅起微小水花，地面水坑中倒映着霓虹招牌和高楼轮廓，电影胶片颗粒质感；

5-8秒：镜头Subtle Tilt Down缓缓下移聚焦地面水坑，水坑倒影中的城市建筑开始扭曲变形，现代建筑的直线条逐渐弯曲成飞檐翘角，霓虹灯光渐变为温暖的灯笼光晕，涟漪在水面一圈圈扩散；

9-13秒：镜头穿入水坑倒影如同穿过水面，Smooth Dolly Forward在水纹涟漪中穿越到唐朝黄昏集市，色调完全转为暖金，绸缎飘扬在微风中，远处传来马蹄声和商贩吆喝声，木质建筑和石板路在夕阳下泛着温润光泽，画面质感从写实摄影过渡为中国风工笔画的细腻质感；

13-15秒：Slow Crane Up从集市缓缓升起，暮色中长安城宫殿群的轮廓在金色晚霞中若隐若现，远处城墙绵延。

一镜到底全程不切镜头。禁止：任何文字、字幕、LOGO或水印。
```

#### 设计意图

水坑倒影是一个非常诗意的穿越介质——"低头看水中倒影，抬头已是千年前"。通过倒影中建筑轮廓的渐变来过渡，比直接切换更丝滑。画面质感从写实摄影→工笔画的转变也暗示了从"现代纪实"到"古典意境"的跨越。

### 提示词解析

| 对比项 | 版本一（旋转门） | 版本二（水坑倒影） |
|--------|-----------------|-------------------|
| 穿越介质 | 旋转门（物理空间过渡） | 水坑倒影（虚实交融过渡） |
| 情绪基调 | 大气、震撼、有冲击力 | 诗意、梦幻、有意境 |
| 技术难度 | 中等（场景直切） | 较高（倒影渐变+质感过渡） |
| 适合平台 | 抖音/B站（视觉冲击强） | 小红书/视频号（文艺调性） |

> **迭代建议**：先试版本一，效果更可控；如果版本一出来效果不错，再试版本二挑战更高难度的诗意过渡。
