# 一站式测试清单

> **加长版 / 清明脑洞 / ASMR 真人向 / AI 脱口秀**：见同目录 **[11-用户反馈延展.md](11-用户反馈延展.md)**（按你的试片反馈整理，含分段与视频延长写法）。

> 按优先级排列，每个提示词直接复制到即梦使用。
>
> **操作流程：** 先生成图片（图片3.0，9:16），选满意的图片后再生成视频（视频3.0）。
>
> **时长选择：** 每个提示词标题旁标注了建议时长（`5秒` 或 `10秒`）。大多数用 5 秒即可——动作型和微距型 5 秒刚好，名画活化类建议 10 秒给足展开空间。效果好的可以追加"视频延长"拼到 10~15 秒。

---

## 第一批：动作触发场景变化（最快验证，每个~300积分）

> 建议时长：5秒 | 比例：9:16竖屏（适配抖音）或 16:9横屏

### ① 打喷嚏→吹飞一切 `5秒`
```
一位穿连帽卫衣的年轻人站在秋天的街头，猛地打了一个大喷嚏，喷嚏的气流像飓风一样把周围所有东西都吹飞——树叶路人雨伞报纸全部被吹向远方，只有他一个人站在原地，3D动漫风格，夸张喜剧效果
```

### ② 弹指→季节切换 `5秒`
```
一只手在画面中央弹响手指，弹指的瞬间整个画面的环境从白雪皑皑的冬天瞬间切换为樱花盛开的春天，树枝上的雪瞬间变为粉色花瓣纷飞，地面积雪化为绿草如茵，天空从灰白变为湛蓝，电影质感，慢镜头
```

### ③ 吹蒲公英→变星空 `5秒`
```
微距特写一个人轻轻吹一朵蒲公英，绒毛种子飘散到空中后每一颗种子变成一颗发光的星星，星星越飘越远越飘越高最终填满整个画面变成一片浩瀚星空，从微距到宏观的尺度变化，梦幻色调，慢镜头
```

### ④ 击掌→世界定格 `5秒`
```
两个人在篮球场击掌的瞬间，冲击波从掌心爆发向四周扩散，冲击波经过的地方一切都被冻结定格——篮球悬停在半空、跳跃的观众定格在腾空状态、水花凝固成冰晶，只有这两人还在动，3D动漫渲染，慢镜头
```

### ⑤ 拍桌子→画面碎裂 `5秒`
```
一只手用力拍在桌面上，拍击的瞬间整个画面如同玻璃屏幕一样从拍击点向四周裂开碎片，碎裂的画面碎片脱落后露出背后一个完全不同的世界——从办公室碎裂后露出星空宇宙，碎片在太空中飘浮，科幻风格
```

### ⑥ 翻书页→场景穿越 `5秒`
```
特写一只手翻开一本旧书的书页，书页翻过的瞬间画面跟随书页转动，翻过后整个场景从现代书房变成了书中描述的古代战场，盔甲骑兵在远处冲锋扬起尘土，书页的边缘仍然可见框住画面如同一扇窗户，史诗古风
```

## 第二批：视觉ASMR + 满足感（高完播率）

> 建议时长：5秒 | 比例：9:16竖屏 | 这类内容慢镜头+微距是核心，5秒足够

### ⑦ 蜂蜜倾倒 `5秒`
```
微距特写，金色蜂蜜从勺子边缘缓缓倾泻而下，蜂蜜拉出一条完美的粘稠细线后落在烤面包表面缓慢摊开，面包的每一个气孔都被蜂蜜缓慢填满，全程极致慢镜头，暖色侧光，背景纯黑，视觉满足感拉满
```

### ⑧ 雨滴打荷叶 `5秒`
```
微距特写，雨滴击打碧绿荷叶表面的瞬间，水珠在荷叶疏水表面弹跳滚动如同水银，多颗小水珠汇聚成一颗大水珠后沿叶脉滚落坠入池塘，荷叶微微震颤，慢镜头120帧，自然光，绿色调
```

### ⑨ 墨入水绽放牡丹 `5秒`
```
纯白背景中透明玻璃容器装满清水，一滴浓黑墨水从上方坠入水中，墨水在水中精确绽放成一朵盛开牡丹花形态，每一片花瓣是墨水涡旋结构，花蕊处最浓向边缘渐淡如水墨渲染，慢镜头120帧，均匀柔光
```

### ⑩ 液态金属流动 `5秒`
```
一团银色液态水银在纯黑背景上缓慢流动变形，表面完美反射周围一切如同镜面，水银分裂成数颗小球后又汇聚成一团，全程极致慢镜头，微距特写，金属表面反射清晰，ASMR般的视觉满足感
```

## 第三批：经典改编（名画活化，无版权风险）

> 建议时长：5~10秒 | 比例：16:9横屏（长卷更适合宽画幅）| 效果好可追加10秒延长

### ⑪ 清明上河图活起来 `10秒`
```
清明上河图风格的长卷画面，水墨工笔画中的人物开始有了微妙的动作——河上的船夫摇橹划水，桥上的行人缓步走动，沿街小贩招手揽客，茶楼二楼有人推窗观景，画面保持工笔画的纸质纹理和墨色质感，微风吹动酒旗飘扬
```

### ⑫ 千里江山图化为3D `10秒`
```
以千里江山图的青绿色调为基础，平面的山水画开始膨胀生长为3D立体地形，山峰从纸面上拔起，河流开始流淌反光，云雾在山谷中弥散，一群白色仙鹤从画中飞出穿过云层，从2D古画到3D仙境的连续变化，中国风极致美学
```

### ⑬ 梵高星月夜流动 `10秒`
```
一幅梵高星月夜风格的画面，天空中旋涡状的云层和星星开始缓缓旋转流动，金色的月亮散发脉动的光晕，远处小镇的窗户亮起暖光，柏树在夜风中轻轻摆动，所有笔触保持油画肌理质感但都在活动，如同画中世界苏醒
```

## 第四批：微缩小人国（先做图再做视频）

> 流程不同：先用下面的提示词生成图片，选满意的图后再点"生成视频"。
> 建议时长：图片→视频 `5秒` | 比例：9:16竖屏 | 效果好可用"视频延长"追加到10秒

### ⑭ 包饺子 `图片→5秒视频`
```
微观世界微距摄影视角，俯瞰巨大的饺子皮和肉馅，五六个穿围裙的微缩小人合力将饺子皮对折包起来，有的在搅拌馅料有的在擀皮，木质案板质感，温暖厨房光线，微缩景观风格，质感细腻高清
```
→ 图片满意后，视频动态描述：`小人们协力包饺子，动作自然流畅`

### ⑮ 烤棉花糖 `图片→5秒视频`
```
微观世界微距摄影视角，一颗巨大的棉花糖被架在篝火上烤，三个穿露营服的微缩小人举着长签子翻转棉花糖，棉花糖表面微微焦黄冒着热气，篝火火光照亮小人的脸，户外野营氛围，微缩景观风格，高清细腻
```
→ 视频动态描述：`小人翻转棉花糖，篝火跳动，热气升腾`

### ⑯ 冲咖啡 `图片→5秒视频`
```
微观世界微距摄影视角，巨大的咖啡杯中深褐色咖啡液面如同湖泊，两个微缩小人站在杯沿用小桶往下倒牛奶形成拉花，一个小人坐在方糖上钓鱼，咖啡馆温暖灯光，微缩景观风格，质感细腻高清
```
→ 视频动态描述：`小人倒牛奶形成拉花，钓鱼小人甩竿`

### ⑰ 寿司制作 `图片→5秒视频`
```
微观世界微距摄影视角，巨大的紫菜铺在竹帘上，一排微缩小人穿着厨师服在紫菜上铺米饭和鱼片，有的推着小推车运送三文鱼块有的在用小铲子抹饭，日式简约光线，微缩景观风格，高清细腻
```

## 第五批：风格突变 + 萌系改编

> 建议时长：5秒 | 比例：9:16竖屏

### ⑱ 写实→水墨变身 `5秒`
```
一位穿汉服的女子站在竹林中微风吹动衣带，她缓缓抬起手臂，从指尖开始整个人和背景逐渐化为中国水墨画风格，墨色从边缘向中心晕染蔓延，最终整个画面变为一幅活的水墨画
```

### ⑲ 黑白→彩色觉醒 `5秒`
```
一朵玫瑰花的黑白特写画面，花瓣上有露水，画面从花蕊中心开始有颜色渗出，鲜红色如同墨水扩散般沿着花瓣纹脉蔓延，逐渐整朵花恢复鲜艳色彩，背景也从黑白变为彩色花园，微距，慢镜头
```

### ⑳ 武侠猫咪对峙 `5秒`
```
两只穿着迷你古装的猫咪在竹林中对峙，竹叶飘落，一只橘猫握着牙签剑摆出起手式，对面的黑猫单爪背后冷酷凝视，风吹动它们的小披风，突然两猫同时弹跳交锋在空中对砍，Q版武侠风格，慢镜头
```

### ㉑ 像素→3D写实 `5秒`
```
一个像素风格的8-bit游戏画面，角色在像素方块组成的世界中奔跑跳跃，镜头逐渐推近，像素方块开始膨胀变得圆润细腻，色块填充为真实材质纹理，最终整个画面从像素变为高清3D写实游戏画面，角色动作保持一致
```
