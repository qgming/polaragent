## 第一部分：IP 设计

### IP 名称：丁总（Chairman Ding）

**物种**：英国短毛猫（蓝灰色）

### 人设定位

| 项目 | 内容 |
|------|------|
| **职业** | 霸道总裁 / 某集团董事长 |
| **性格** | 表面高冷傲娇，内里小心眼又嘴硬，时刻维持"霸总"人设但总在最后一秒破功 |
| **口头禅** | "这种小事，不需要我亲自处理。"（然后亲自去处理） |
| **反差萌点** | 穿西装打领带一本正经地做猫的事情（踩纸箱、追逗猫棒、舔爪子），且全程保持霸总表情 |
| **人际关系** | 有一个"秘书"（橘猫，戏份可扩展），员工们都是各种小动物 |

### 外形描述（角色一致性锚点）

- **毛色**：标准英短蓝灰色，毛发浓密蓬松，胸口有一小撮稍浅的灰白绒毛
- **眼睛**：铜黄色大圆眼，永远半眯着，自带"你不配"的凝视感
- **标志性细节**：
  - 脖子上系一条深藏蓝色迷你领带（始终佩戴，是角色标志）
  - 圆脸扁鼻，两腮微鼓（天生的霸总"不屑脸"）
  - 右耳尖有一小块深色毛斑（辨识度细节）
- **体型**：微胖圆润，前爪粗短有力，坐姿永远挺胸端正

## 使用指南

### 生成顺序

1. **先生成角色卡片图**：用"角色卡片图生成提示词"在即梦图片生成中出图，挑选最满意的一张
2. **生成首帧图**：用每条短剧的"首帧图生成提示词"生成，确保风格与角色卡一致
3. **生成视频**：将角色卡片图作为 @图片1 上传，首帧图可作为 @图片2 辅助引导首帧画面，粘贴视频提示词生成

### 运营建议

- **发布节奏**：每天 1-2 条，建立"丁总日常"固定栏目
- **系列化**：5条短剧可作为第一季"丁总的办公室"，后续扩展"丁总出差""丁总相亲""丁总过年"
- **互动钩子**：每条视频结尾可配文案"丁总今天又破防了，你猜下一次是什么？"
- **跨平台**：竖屏9:16适配抖音/小红书/视频号同步分发
- **追热点**：节假日/热点事件出丁总特别版（丁总过春节、丁总看世界杯）
