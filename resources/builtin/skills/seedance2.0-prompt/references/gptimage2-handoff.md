# GPT Image 2 前期资产交接

用于短剧、短片、电影项目中，先用 GPT Image 2 生成角色/道具/场景/首帧图片，再把这些图片交给 Seedance 2.0 做图生视频、角色一致性视频或多镜头分段生成。

## 什么时候先用 GPT Image 2

遇到以下需求时，先调用 `gptimage2-prompt` 生成图片资产，再回到本技能生成 Seedance 视频提示词：

- 用户要短剧、短片、电影、预告片、剧情片段。
- 用户要保持人物、服装、道具或场景一致。
- 用户需要角色设定图、人物三视图、角色卡、物品设定图、道具设定图。
- 用户需要首帧图、关键帧图、分镜参考图、场景板。
- 用户只给创意，但没有可用于图生视频的参考图。

## 推荐资产顺序

短剧/短片生产优先按这个顺序准备：

1. **角色设定图**：主角、反派、重要配角。用于锁定外貌、服装、体态和辨识点。
2. **关键道具图**：戒指、报告、合同、武器、手机、录音笔、玉佩等剧情物件。
3. **场景设定图**：医院走廊、出租屋、婚宴后台、县城街口、宫殿内室等核心空间。
4. **首帧图**：每条 Seedance 图生视频的起点画面。
5. **关键帧图**：转折、揭示、动作高潮或情绪爆点。

## 与 Seedance 的引用分工

把 GPT Image 2 生成出的图片导入 Seedance 后，使用官方引用编号：

- `@图片1`：主角角色图或首帧图。
- `@图片2`：对手角色图、双人关系图或关键道具图。
- `@图片3`：场景图或关键帧图。
- `@视频1`：参考运镜、动作节奏或成片质感。

文字提示词只补图片没有的信息：

- 时长、画幅、镜头运动和节奏。
- 角色动作、情绪变化、台词和音效。
- 首帧之后发生什么。
- 需要避免的漂移：不要换脸、不要换服装、不要改道具、不要改变场景结构。

## Seedance 提示词写法

当图片来自 GPT Image 2 设定图时，Seedance 提示词要强调“引用图片锁定一致性”，不要重新长篇描述外貌。

```text
参考@图片1的人物外貌、服装层次、体态和气质保持一致，参考@图片2的场景结构和光线氛围。9:16竖屏，8秒短剧镜头。镜头从中近景开始，角色站在病房门外低头看文件，手指轻微发抖，随后慢慢抬头看向门内。摄影机轻微手持前推，焦点从文件边缘转到角色眼神。走廊冷白顶灯闪烁，远处护士站虚化，环境安静压抑。禁止换脸、换服装、改变文件道具、生成字幕、LOGO、水印。
```

## 交接检查

生成 Seedance 提示词前检查：

- 是否明确哪个 `@图片N` 锁定角色，哪个锁定场景或道具。
- 是否减少了重复外貌描写，避免和图片冲突。
- 是否写清首帧之后的动作、情绪和镜头运动。
- 是否加入一致性约束：不换脸、不换服装、不改道具、不改变场景结构。
- 是否控制在 Seedance 可理解的自然中文范围内，不堆砌图片提示词里的材质细节。
