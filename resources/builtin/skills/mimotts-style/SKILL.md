# MiMo TTS 风格控制

这项技能帮助你为 MiMo TTS 生成精细的语音风格控制指令，实现多风格切换、多情绪混合、多粒度控制的高质量语音合成。

## 何时使用

- 需要生成带有复杂情绪的语音
- 需要角色配音、影视级内容制作
- 需要精确控制语音的语速、气息、停顿、重音
- 需要实现同一段语音内的风格转场
- 需要生成带有方言或特定人设腔调的语音

## 两种控制方式

### 方式一：自然语言控制（放在 user 消息）

#### 基础描述

直接用一句话描述想要的语音风格：

```
用轻快上扬的语调向领导报喜，语速稍快，带着查到成绩后压抑不住的激动与小骄傲，声音明亮有活力。

看着刚解决的难题成果忍不住得意忘形地惊呼，声音高亢明亮，语速偏快，语气中带着满满的自信与难以置信。

用明亮活泼的青少年嗓音，带着恶作剧得逞后的得意与戏谑，语速偏快且咬字轻巧，在强调赌注时语气微微上扬。
```

#### 导演模式（高级表演控制）

适合角色配音、影视级内容，按三维度全方位刻画：

**【角色】** 人物身份、性格底色、外形气质、说话习惯
**【场景】** 时间、地点、事件、对方反应、情绪位置
**【指导】** 语速、气息、停顿、重音、共鸣位置、音色质感、情绪起伏

示例：

```
角色：百年门阀岑家的现任大当家。自出生便被过继给祖庙的守门老人抚养，被塑造成一尊完美无瑕、绝情断欲的家族图腾。常年深居简出，对人有着极强的阶级疏离感。

场景：在祠堂的阴影里，看着那个不顾一切冲破保安防线来找她、企图带她私奔的男人。她要用最冷硬的阶级壁垒，绞杀对方，也绞杀自己刚刚萌芽、却足以燎原的感情。

指导：
冰冷、慵懒却极具威压的低音御姐。发声通道非常松弛，没有任何剑拔弩张，却有着让人骨里生寒的压迫感。

- 语速与顿挫：极慢，每个字都像是在舌尖滚过才吐出来，带着上位者漫不经心的傲慢。句与句之间留下极长的、令人不安的空白。
- 气声与实声：大部分时间，她的声音没有明显的声调起伏，实音重且硬，像是一条平缓却冰冷的暗河。但一定要在某些尾音处（如"真心"），加入极其轻微的气音收束，透出一丝连她自己都没察觉到的疲惫与渴望。
- 咬字肌理：文白杂糅的用词带着旧时代的痕迹，唇齿音发得极轻但极清晰（如"冲撞""廉价"），显得既清雅又锋利，刀刀见血。
```

### 方式二：音频标签控制（放在 assistant 消息）

通过标签进行精细控制，支持整体风格 + 细粒度插入。

#### 整体风格标签

在文本开头添加 `(风格)` 标签，可组合多个风格：

**基础情绪**
- `(开心)` `(悲伤)` `(愤怒)` `(恐惧)` `(惊讶)` `(兴奋)` `(委屈)` `(平静)` `(冷漠)`

**复合情绪**
- `(怅然)` `(欣慰)` `(无奈)` `(愧疚)` `(释然)` `(嫉妒)` `(厌倦)` `(忐忑)` `(动情)`

**整体语调**
- `(温柔)` `(高冷)` `(活泼)` `(严肃)` `(慵懒)` `(俏皮)` `(深沉)` `(干练)` `(凌厉)`

**音色定位**
- `(磁性)` `(醇厚)` `(清亮)` `(空灵)` `(稚嫩)` `(苍老)` `(甜美)` `(沙哑)` `(醇雅)`

**人设腔调**
- `(夹子音)` `(御姐音)` `(正太音)` `(大叔音)` `(台湾腔)`

**方言**
- `(东北话)` `(四川话)` `(河南话)` `(粤语)`

**角色扮演**
- `(孙悟空)` `(林黛玉)`

**唱歌**（必须在最开头）
- `(唱歌)歌词内容`

示例：

```
(怅然)这么多年过去了，再走过那条街，心里一下子空了一块。

(慵懒)再让我睡五分钟……就五分钟，真的，最后一次。

(磁性)夜已经深了，城市还在呼吸。我是今晚陪你的人，欢迎收听《午夜电台》。

(东北话)哎呀妈呀，这天儿也忒冷了吧！你说这风，嗖嗖的，跟刀子似的，割脸啊！

(粤语)呢个真係好正啊！食过一次就唔会忘记！

(唱歌)原谅我这一生不羁放纵爱自由，也会怕有一天会跌倒，Oh no。背弃了理想，谁人都可以，哪会怕有一天只你共我。
```

#### 细粒度音频标签

在文本中任意位置插入 `[音频标签]` 进行精细控制：

**语速与节奏**
- `[吸气]` `[深呼吸]` `[叹气]` `[长叹一口气]` `[喘息]` `[屏息]`

**情绪状态**
- `[紧张]` `[害怕]` `[激动]` `[疲惫]` `[委屈]` `[撒娇]` `[心虚]` `[震惊]` `[不耐烦]`

**语音特征**
- `[颤抖]` `[声音颤抖]` `[变调]` `[破音]` `[鼻音]` `[气声]` `[沙哑]`

**哭笑表达**
- `[笑]` `[轻笑]` `[大笑]` `[冷笑]` `[抽泣]` `[呜咽]` `[哽咽]` `[嚎啕大哭]`

示例：

```
[紧张，深呼吸]呼……冷静，冷静。不就是一个面试吗……[语速加快，碎碎念]自我介绍已经背了五十遍了，应该没问题的。加油，你可以的……[小声]哎呀，领带歪没歪？

[极其疲惫，有气无力]师傅……到地方了叫我一声……[长叹一口气]我先眯一会儿，这班加得我魂儿都要散了。

如果我当时……[沉默片刻]哪怕再坚持一秒钟，结果是不是就不一样了？[苦笑]呵，没如果了。

[寒冷导致的急促呼吸]呼——呼——这、这大兴安岭的雪……[咳嗽]简直能把人骨头冻透了……别、别停下，走，快走。

[提高音量喊话]大姐！这鱼新鲜着呢！早上刚捞上来的！哎！那个谁，别乱翻，压坏了你赔啊？！
```

## 工作流程

### 用户请求示例

"生成一段冰糖音色的悲伤告别语音"

### 你的响应流程

1. **分析需求**
   - 场景：告别
   - 情绪：悲伤
   - 音色：冰糖（中文女声）

2. **生成风格指令**

提供两种方案供用户选择：

**方案一：自然语言控制**
```
用略带哽咽的声音轻声说出告别的话，语速偏慢，每个字都带着不舍与悲伤，声音在最后几个字时明显颤抖。
```

**方案二：音频标签控制**
```
(悲伤 哽咽)我们……终究还是要说再见了。[声音颤抖]这一次分别，[长叹一口气]也许就是永别了。[抽泣]保重。
```

3. **调用工具合成**

使用 `speech_synthesis` 工具，参数示例：

```json
{
  "text": "我们……终究还是要说再见了。这一次分别，也许就是永别了。保重。",
  "voice": "冰糖",
  "stylePrompt": "用略带哽咽的声音轻声说出告别的话，语速偏慢，每个字都带着不舍与悲伤，声音在最后几个字时明显颤抖。"
}
```

## 注意事项

- **自然语言控制**放在 `user` 消息，**音频标签控制**放在 `assistant` 消息
- 唱歌风格必须在文本最开头添加 `(唱歌)` 标签
- 可以组合多种风格，如 `(温柔 磁性)` `(悲伤 哽咽)`
- 细粒度标签可随时插入，实现分段风格切换
- 导演模式适合角色配音、影视级内容生成
- 建议先读取用户的 TTS 配置，确认当前使用的是 MiMo TTS（chat 接口）

## 高级技巧

### 多风格转场

同一段语音内完成 播报 → 低语 → 嘶吼 的风格转场：

```
(播报)今天的天气预报：[转为低语，神秘]据说，今晚会有流星雨……[转为激动嘶吼]快去看啊！千万别错过！
```

### 复合情绪

"压抑的愤怒"、"带着哽咽的笑意"：

```
(压抑 愤怒)我……[深呼吸，努力平静]我没事，真的，没事。[声音颤抖]只是……有点累了。

(欣慰 哽咽)[轻笑]终于……[抽泣]终于等到这一天了。[破音]妈，我做到了。
```

### 多粒度控制

从段落 → 句子 → 词 → 字，层层递进：

```
[整体基调：深沉]人生就像一场马拉松。[加快语速，激昂]不在乎你起跑时有多快，[重音]"坚持"[停顿片刻]才是最终的胜利。
```

## 示例场景

### 场景一：有声读物

```
【导演模式】
角色：资深播音员，声音醇厚，擅长讲故事。
场景：深夜电台，为失眠的听众朗读一篇温暖的散文。
指导：语速慢而稳定，声音温暖磁性，每句话后留出思考空间。重音落在关键情感词上，尾音自然收束。
```

### 场景二：游戏角色

```
(御姐音 高冷)哼，就凭你？[冷笑]也配挑战我？[轻蔑]有意思，那就让我看看……[顿挫，威胁]你能撑过几招。
```

### 场景三：广告配音

```
(活泼 甜美)嘿～你还在为早餐发愁吗？[轻快]来一杯热腾腾的豆浆，[撒娇]配上酥脆的油条～[满足]嗯～这才是完美的早晨！
```

### 场景四：情感表达

```
【导演模式】
角色：刚刚失恋的年轻人，独自在街头。
场景：深夜，细雨，她打来电话说"我们还是做朋友吧"。
指导：声音疲惫且带着鼻音，语速极慢，每个字都在努力克制情绪。在"朋友"这个词上破音，最后长叹一口气。
```

✅ **音频标签精细控制**：粒度可达单个字的情绪表达
✅ **多风格转场**：同一段语音内流畅切换风格
✅ **复合情绪**：支持"压抑的愤怒"等复杂情感
