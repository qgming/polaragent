# 输出模板

本文件放可直接套用的输出结构。用户给定字段时，必须保留用户字段；用户没有给字段时，按这里选择模板。

## 电影截帧图

```text
【最终提示词】
[主体] 出现在 [具体场景] 中，[人物关系或事件压力]，[动作前后状态]，[环境和道具提供的叙事线索]。镜头位于 [观众位置]，[机位与角度]，[焦距与构图]，[前景遮挡或参照]，[中景空间]，[远景压力]。光线来自 [可信光源]，[空气介质与光线互动]，[色彩基调]。真实电影机拍摄，[画幅]，[摄影质感]，[曝光与高光行为]，[胶片颗粒与镜头细节]，超清电影画面，物理可信。

【负面约束】
避免廉价特效感、塑料表面、过度光滑、干净空白背景、主体与环境没有接触、过亮轮廓边缘、不自然比例、过度锐化、均匀平光、海报式摆拍、主体完整居中平铺、没有环境叙事、没有人类尺度参照、颜色过度鲜艳、材质过度完美。
```

## 角色设定图

```text
【最终提示词】
[角色身份] 的角色设定图，横版宽画幅，左侧占主要面积放置正面、侧面、背面三视图，三视图比例统一、站姿统一、基线对齐；右侧排列面部特写、表情变化、手部特写、服装细节、道具细节和材质局部，方便视频前期制作参考。角色轮廓清晰，五官和体型辨识度高，服装结构、层次、旧化痕迹、纹理和配饰关系完整可读，背景保持浅灰或中性纯净背景，不抢主体信息。光线均匀克制，便于观察形体和材质，整体像专业视频前期设定板而不是情绪海报。真实电影级质感，清晰、干净、可拆分、可复用。

【负面约束】
避免复杂剧情场景、强烈戏剧光、烟雾遮挡、过度动态姿态、主体完整居中单张摆拍、背景过于花哨、文字说明块过多、信息密度失衡、材质过度光滑、比例失真、轮廓不清、细节糊成一团。
```

## 场景设定图

```text
【最终提示词】
[场景名称] 的场景设定图，强调空间结构、建筑比例、光位、动线和可拍摄性，画面中清楚展示前景、中景、远景和关键道具位置，适合后续视频分镜和置景参考。整体保持结构清晰、信息完整、材质可信，避免把场景写成单一情绪海报。真实电影级写实质感，构图干净，空间层次明确，便于视频创作者拆解。

【负面约束】
避免主体抢场景、过强氛围遮挡结构、过度虚化、过度戏剧化、信息不完整、空间关系混乱、比例失真、无用装饰过多。
```

## 首帧图与关键帧图

```text
【最终提示词】
[画幅比]，[风格锚定]，[构图与主体位置]。[角色状态和关键辨识特征]，[动作前摇或动作后果]，[环境细节]，[光影层次]，[色调氛围]。这是一张用于视频生成的首帧或关键帧，保留后续接镜头的空间余量，主体辨识度清楚但不把画面塞满。真实电影级质感，物理可信。

【负面约束】
避免文字、字幕、LOGO、水印、信息过满、动作已经结束无延展空间、主体过小、主体被遮挡到无法识别、场景关系混乱。
```

## 视频提示词

当用户要求视频、动态镜头、运镜、短片、AI 视频提示词时，输出视频提示词而不是静态图提示词。

```text
【镜头目标】
用真实目击者视角表现 [主体] 的 [情绪、尺度或叙事目标]，让观众感觉自己站在现场。

【视频提示词】
[时长] 连续镜头，场景位于 [现实地点]，[天气与时间]。镜头从 [人类尺度前景] 开始，[摄影机初始运动]，[环境异常或剧情线索逐步出现]，随后 [主体动作或关键揭示] 引发 [物理反馈或人物反应]。前景的 [人物、车辆、道具、门框、窗户] 提供尺度或叙事线索，中景的 [建筑、房间、街道、山体、海浪] 承接动作，远景由 [主体、天象、城市或事件压力] 占据。光线来自 [可信光源]，穿过 [雨、雾、尘、水汽、烟、雪] 形成真实散射，整体色彩为 [低饱和色彩基调]，真实电影机拍摄，物理可信。

【摄影机运动】
[低机位、手持、缓慢前推、轻微抬镜、最后停住]，运动克制，有真实重量感。

【主体运动】
[主体动作] 带有明确开始和结束，动作速度符合体量和情绪。

【环境运动】
雨线、尘埃、水面、云层、衣物、树冠、火光、灯光和人群反应随着主体出现发生延迟变化。

【光线与色彩】
可信自然光与现实小光源共同塑造画面，暗部保留层次，高光不过度溢出。

【真实摄影质感】
真实电影机拍摄，自然曝光，真实镜头高光滚降，细腻空气透视，轻微胶片颗粒，运动处保留自然模糊。

【负面约束】
避免廉价特效感、塑料表面、过度光滑、漂浮无重量感、干净空白背景、主体与环境没有接触、过度锐化、均匀平光、海报式摆拍、颜色过度鲜艳、材质过度完美。
```

## 用户字段纪律

当用户提供已有字段结构、标题列表或示例结构时，必须严格保留该结构。除非用户明确要求新结构，否则不要新增、重命名、删除、重排或扩展字段。可以优化字段内部内容。
