# 图像生成参考

这份文档专门放视频创作者常用的图像资产模板。主文件 `SKILL.md` 只保留入口规则和总原则，具体版式和写法放在这里，便于后续继续扩展更多素材类型。

## 先判断类型

先判断用户要的是哪一类图。

- **电影镜头图**：强调情绪、叙事、尺度和现场感。
- **制作素材图**：强调结构、辨识点、拆分能力和后续视频可用性。

如果用户提到角色设定图、三视图、角色卡、角色参考图、首帧图、关键帧图、场景板、道具板、群像图或阵容图，就优先走制作素材写法。

角色设定图、人物三视图、物品设定图、道具设定图、武器设定图的详细版式见 [`design-sheets.md`](design-sheets.md)。当用户明确要“左侧三视图、右侧特写”时，以该文件为最高优先级。

## 角色设定图

角色设定图的目标是让后续视频制作省事，而不是让单张图本身最戏剧化。

### 默认布局

- 默认使用横版宽画幅，常见为 `16:9` 或 `3:2`。
- 左侧放主视图区域，优先安排正面、侧面、背面三视图。
- 三视图站姿统一、比例统一、脚下基线对齐。
- 右侧放特写区，安排面部近景、表情变化、手部特写、服装纹理、鞋子、武器、配饰或局部材质。
- 背景使用浅灰、米白、冷白或轻微中性渐变，避免复杂场景抢信息。
- 光线均匀克制，便于观察形体、结构和材质，不要把图做成情绪海报。

### 写法重点

- 角色身份必须清楚。
- 轮廓要有识别度。
- 服装结构、层次、旧化痕迹、材质和配饰关系要可读。
- 设定图要便于后续拆成角色卡、动作参考和换装参考。

### 角色设定图模板

```text
[角色身份] 的角色设定图，横版宽画幅，左侧占主要面积放置正面、侧面、背面三视图，三视图比例统一、站姿统一、基线对齐；右侧排列面部特写、表情变化、手部特写、服装细节、道具细节和材质局部，方便视频前期制作参考。角色轮廓清晰，五官和体型辨识度高，服装结构、层次、旧化痕迹、纹理和配饰关系完整可读，背景保持浅灰或中性纯净背景，不抢主体信息。光线均匀克制，便于观察形体和材质，整体像专业视频前期设定板而不是情绪海报。真实电影级质感，清晰、干净、可拆分、可复用。

负面约束：避免复杂剧情场景、强烈戏剧光、烟雾遮挡、过度动态姿态、主体完整居中单张摆拍、背景过于花哨、文字说明块过多、信息密度失衡、材质过度光滑、比例失真、轮廓不清、细节糊成一团。
```

## 群像与阵容图

群像图、阵容图、角色关系图和短剧选角图，重点是让角色关系一眼能读懂。

### 默认布局

- 主角放在视觉重心。
- 其他角色围绕站位、视线方向和关系展开。
- 用身高差、服装差异、道具差异和姿态差异区分人物。
- 如果后续要做视频，每个人都要能继续拆成独立角色卡。

### 群像模板

```text
[作品名称或角色阵容] 的群像设定图，横版构图，主角位于视觉重心，其他角色围绕关系展开，所有角色身份和站位清晰可读。通过身高差、视线方向、服装差异、配饰差异和道具差异区分人物，保证后续视频制作可以单独拆分每个角色。背景简洁，光线克制，整体像专业前期阵容板，不要做成剧情海报或情绪拼贴。

负面约束：避免过度戏剧化、角色互相遮挡、关系不清、信息过载、背景复杂、人物比例混乱、装饰元素抢戏。
```

## 场景设定图

场景板、空间设定图、环境参考图和道具板，重点是空间结构和拍摄可用性。

### 默认布局

- 清楚展示前景、中景、远景。
- 让建筑比例、光位、动线和关键道具位置一目了然。
- 保持空间层次完整，方便后续分镜和置景。

### 场景模板

```text
[场景名称] 的场景设定图，强调空间结构、建筑比例、光位、动线和可拍摄性，画面中清楚展示前景、中景、远景和关键道具位置，适合后续视频分镜和置景参考。整体保持结构清晰、信息完整、材质可信，避免把场景写成单一情绪海报。真实电影级写实质感，构图干净，空间层次明确，便于视频创作者拆解。

负面约束：避免主体抢场景、过强氛围遮挡结构、过度虚化、过度戏剧化、信息不完整、空间关系混乱、比例失真、无用装饰过多。
```

## 首帧图与关键帧图

首帧图和关键帧图不是完整成片，它们是视频链路里的接镜头素材。

### 默认写法

- 保留动作前摇和视线方向。
- 预留空间余量，方便后续接镜头。
- 主体辨识度要足够高，但不要把画面塞满。
- 如果是连续动作的关键帧，优先锁定姿态、光位和空间关系。

### 首帧与关键帧模板

```text
[画幅比]。[风格锚定]，[构图+主体位置]。[角色状态+关键辨识特征]。[环境细节]。[光影层次]。[色调氛围]。[画质]。把它当成视频第一眼画面来写，保留后续接镜头的空间余量，不要塞满所有信息。禁止：任何文字、字幕、LOGO、水印
```

## 默认补充原则

- 如果用户没有说明具体模板，优先给出能直接用于制作的版本。
- 如果用户只给了一个角色名，也不要追问太多，直接补齐身份、结构、版式和可拆分细节。
- 如果用户在做视频，就始终优先考虑一致性、拆分性和后续镜头可用性。

## 短剧短片图片资产清单

面向短剧、短片和电影前期制作时，优先把图片需求拆成以下资产。不要只给一张“好看的图”，要判断这张图在生产链路里的用途。

| 资产类型 | 解决什么 | 默认画幅 |
|:---|:---|:---|
| 角色设定图 | 锁定人物外貌、服装、比例、辨识点 | 16:9 或 3:2 横版 |
| 角色单人卡 | 给演员/模型/后续图生视频提供一致性参考 | 9:16 竖版 |
| 双人关系图 | 锁定 CP、对手、亲属、上下级的站位和权力关系 | 16:9 横版或 9:16 竖版 |
| 群像阵容图 | 展示主角团、反派团、短剧选角阵容 | 16:9 横版 |
| 首帧图 | 作为图生视频的第一帧，引导情绪和动作 | 跟视频比例一致 |
| 关键帧图 | 锁定关键动作、转折、揭示、反应 | 跟视频比例一致 |
| 分镜参考图 | 为某个镜头提供构图、景别、动作和空间参考 | 16:9 或 9:16 |
| 场景设定图 | 锁定空间结构、动线、光位、道具位置 | 16:9 横版 |
| 道具设定图 | 锁定关键道具形状、材质、磨损、剧情线索 | 1:1 或 4:3 |
| 氛围色调板 | 锁定系列视觉基调、季节、年代、冷暖关系 | 16:9 横版 |

## 双人关系图

双人关系图适合短剧里最常见的关系：情侣、前任、夫妻、继承人和管家、老板和员工、嫌疑人和警察、母女、兄弟、师徒。

写法重点：

- 站位体现权力关系，而不是两个人并排合照。
- 用视线、距离、手势、遮挡和道具表达关系。
- 服装差异要服务身份差异。
- 保留后续拆成单人角色卡的清晰度。

模板：

```text
[两名角色关系] 的双人关系设定图，[画幅]，两人处于 [具体场景]，站位体现 [权力关系或情绪关系]。角色 A 的外貌、服装、姿态和道具清晰可读，角色 B 的外貌、服装、姿态和道具清晰可读。通过视线方向、身体距离、手部动作、前景遮挡和环境线索表现两人的关系张力。光线克制，背景不抢主体，整体适合短剧前期人物关系参考，也可拆成单人角色图继续使用。
```

## 道具设定图

短剧和悬疑片里的道具经常承担剧情信息。道具图不要做成电商产品图，要保留使用痕迹和故事线索。

常见道具：

- 亲子鉴定报告、离婚协议、旧手机、录音笔、婚戒、玉佩、血迹衣角、钥匙、药瓶、车票、合同、遗书、照片、旧箱子。

模板：

```text
[关键道具] 的道具设定图，清楚展示正面、侧面、局部磨损、材质纹理和被使用过的痕迹。道具放在 [符合剧情的表面或环境] 上，周围有少量叙事线索但不喧宾夺主。光线来自 [可信小光源]，材质真实，细节可读，适合短剧或电影分镜中的特写镜头参考。
```

## 分镜参考图

分镜参考图不是最终成片，也不是漂亮海报。它服务于“这一镜怎么拍”。

每张分镜参考图要包含：

- 镜头编号或内部顺序感，但最终图像提示词里不要要求生成文字编号。
- 景别：远景、全景、中景、近景、特写。
- 角度：平视、低角度、高角度、过肩、门缝、车内、监控视角。
- 运动意图：前推、后退、横移、抬镜、下摇、手持跟拍。
- 角色动作：开始姿态、结束方向、视线和手部动作。
- 空间余量：为视频延展和运镜留下画面边界。

模板：

```text
[镜头用途] 的分镜参考图，[画幅]，[景别]，[机位角度]，[主体位置]，[主体动作前摇或动作后果]，[前景遮挡]，[中景空间关系]，[远景压力或出口方向]。画面服务于后续视频生成，保留运镜空间和动作延展空间，不要做成完成度过高的海报。
```

## 氛围色调板

当用户要“系列视觉风格”“短剧统一调性”“电影感参考图组”时，输出氛围色调板，而不是单张剧情图。

色调板应包含：

- 主场景氛围。
- 角色服装色彩。
- 关键光源。
- 道具材质。
- 天气和空气介质。
- 系列的冷暖对比。

模板：

```text
[项目名称] 的电影氛围色调板，横版构图，由多个小画面组成但整体统一，展示主场景、角色服装色彩、关键光源、道具材质、天气空气介质和整体冷暖关系。不要出现文字、LOGO 或水印。画面像电影前期视觉开发板，帮助统一短剧或短片的视觉方向。
```
