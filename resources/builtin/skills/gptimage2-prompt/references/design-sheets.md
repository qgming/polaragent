# 角色与物品设定图

用于生成短剧、短片、电影前期制作中的角色设定图、人物三视图、服装设定图、物品设定图、道具设定图、武器设定图和关键物件参考图。

这类图片的目标不是“最好看的海报”，而是在一张图里把后续制作需要的信息交代清楚：外形、比例、材质、结构、磨损、特写、可复用辨识点。

## 总原则

- 默认横版宽画幅，优先 `16:9` 或 `3:2`。
- 背景保持浅灰、米白、冷白或中性渐变，避免复杂场景抢信息。
- 光线均匀克制，便于观察结构、比例和材质。
- 画面可以有分区布局，但不要要求生成文字标签、编号、表格、水印或 LOGO。
- 所有视图必须保持同一对象、同一服装、同一比例、同一材质，不要生成多个版本混在一起。
- 设定图里的主体姿态要稳定，避免大幅动作、情绪表演、风吹烟雾、强逆光和剧情场景。

## 人物设定图默认版式

当用户要“角色设定图”“人物设定图”“三视图”“角色参考图”“服装设定图”时，默认使用这个布局：

- **左侧 65%**：角色正面、侧面、背面三视图。
- **右侧 35%**：局部特写区。
- **三视图要求**：同一角色、同一服装、同一身高比例，脚下基线对齐，站姿统一，双臂自然下垂或微微离开身体，方便看清轮廓。
- **右侧特写内容**：面部近景、2-3 个表情、手部、发型、服装层次、布料纹理、鞋子、腰带、配饰、随身道具、污渍或破损细节。
- **短剧人物重点**：年龄感、社会身份、生活痕迹和可识别的小道具，比夸张造型更重要。
- **古装/奇幻人物重点**：发冠、领口、袖口、腰封、鞋靴、武器、纹样、旧化和材质层次必须清楚。

### 人物设定图模板

```text
【最终提示词】
[角色身份] 的人物设定图，横版宽画幅，专业影视前期设定板布局。画面左侧约 65% 放置同一角色的正面、侧面、背面三视图，三视图身高比例一致、服装一致、站姿统一、脚下基线对齐，角色轮廓清晰，便于后续视频制作保持一致性。画面右侧约 35% 是局部特写区，展示面部近景、2-3 个克制表情、手部、发型、服装层次、布料纹理、鞋靴、配饰、随身道具和磨损污渍细节。背景为浅灰中性背景，光线均匀克制，材质真实，细节清楚，整体像专业电影/短剧角色设定板，而不是剧情海报。

【负面约束】
避免复杂剧情背景、强烈戏剧光、烟雾遮挡、动态打斗姿势、三视图比例不一致、服装前后不一致、多个不同角色混入、文字标签、编号、水印、LOGO、表格、过度美颜、塑料皮肤、细节糊成一团。
```

## 角色单人卡版式

当用户明确要“角色卡”“单人角色图”“9:16 角色参考图”时，优先做单人竖版，而不是三视图。

- 默认 9:16 竖版。
- 角色全身或膝上构图。
- 背景简单，但可以带少量符合身份的环境线索。
- 必须写清面部、发型、服装、鞋子、道具、体态、气质和生活痕迹。

### 角色单人卡模板

```text
【最终提示词】
9:16 竖版角色参考图，[角色身份] 单人站立，[年龄感、面部特征、发型、体态气质]，[服装逐层描述、材质、颜色、新旧程度、污渍或磨损]，[鞋子、配饰、随身道具]。背景简洁，只保留少量符合身份的环境线索，光线真实克制，角色辨识度高，适合后续短剧/短片图生视频保持人物一致性。

【负面约束】
避免多人入镜、复杂背景、夸张摆拍、过度美颜、服装结构不清、手部畸形、文字、LOGO、水印。
```

## 物品设定图默认版式

当用户要“物品设定图”“道具设定图”“武器设定图”“关键物件参考图”时，默认使用这个布局：

- **左侧 60%**：物品主视图，展示正面、侧面、背面或俯视图。
- **右侧 40%**：局部特写，展示材质、边缘磨损、接口结构、开合结构、刻痕、污渍、使用痕迹、握持状态或收纳状态。
- **比例参照**：可以放一只手、桌面、布料、盒子或人体局部作为尺度参照，但不要放文字尺码。
- **剧情道具重点**：使用痕迹和故事线索比干净产品摄影更重要。
- **武器装备重点**：整体轮廓、握柄、刃口、磨损、连接处、挂绳、皮套、铭刻纹理必须清楚。

### 物品设定图模板

```text
【最终提示词】
[物品名称] 的物品设定图，横版宽画幅，专业影视道具设定板布局。画面左侧约 60% 展示同一物品的正面、侧面、背面或俯视图，比例一致，结构清晰，轮廓准确。画面右侧约 40% 展示局部特写，包括材质纹理、边缘磨损、接口结构、开合方式、污渍、刻痕、使用痕迹、握持状态或收纳状态。加入一个克制的尺度参照，例如手部、桌面或布料，但不出现文字尺码。背景为浅灰中性背景，光线均匀，材质真实，适合短剧/短片/电影前期道具设计和特写镜头参考。

【负面约束】
避免电商广告感、过度干净、漂浮无重量感、结构不清、多个不同版本混在一起、文字标签、编号、水印、LOGO、表格、强烈戏剧光、复杂剧情背景、材质塑料感。
```

## 文件、证据、纸面道具

亲子鉴定报告、离婚协议、遗书、旧照片、车票、合同、病历、录音笔、手机聊天记录这类道具要谨慎处理文字。GPT Image 2 容易生成乱码文字，提示词应强调“文字不可读但版式可信”。

### 纸面道具模板

```text
【最终提示词】
[纸面道具名称] 的影视道具设定图，横版构图，展示道具的整体版式、折痕、纸张厚度、边角磨损、污渍、水痕、手指按压痕迹和放置环境。文字内容不需要清晰可读，只需要呈现真实文件的排版感和剧情可信度。右侧展示局部特写：纸角、印章痕迹、手写划线、折痕、夹子或文件袋。光线来自真实小光源，材质自然，适合短剧特写镜头参考。

【负面约束】
避免清晰可读文字、真实机构标识、LOGO、水印、乱码大字、过度干净、纸张像塑料、背景复杂抢戏。
```

## 常见默认补全

如果用户只说“给我做一个角色设定图”，默认补：

- 横版 16:9。
- 左侧正面、侧面、背面三视图。
- 右侧面部、表情、手部、服装材质、鞋子、道具特写。
- 浅灰背景，均匀光，专业影视前期设定板。

如果用户只说“给我做一个物品设定图”，默认补：

- 横版 16:9。
- 左侧正面、侧面、背面或俯视图。
- 右侧材质、磨损、接口、使用痕迹、握持状态特写。
- 加尺度参照，但不生成文字尺码。
