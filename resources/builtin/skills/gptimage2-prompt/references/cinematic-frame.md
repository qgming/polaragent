# 电影截帧图规则

用于用户要“电影感图片”“短片画面”“短剧剧照”“电影截帧”“真实拍摄质感”“某个瞬间的一张图”时。

## 核心目标

把极简概念扩展成像真实摄影队拍到的一帧电影画面。最终结果不追求漂亮插画，而追求叙事瞬间、真实空间、可信物理、可见光源和现场感。

## 极简输入补全

当用户只输入一个词或短句，例如“中国龙”“古神”“雨夜追车”“落魄皇帝”“废墟婚礼”“海底神明”，不要先问一堆问题。直接补全：

- 主体身份、年龄感、体量或社会位置。
- 场景地点、时间、天气和空间层次。
- 观众位置和权力关系。
- 镜头角度、焦距、景别、画幅和构图。
- 主体动作、环境反馈和物理尺度。
- 材质、纹理、磨损、污渍和真实生活痕迹。
- 光源、空气介质、色彩基调和曝光逻辑。
- 负面约束。

默认视觉方向：史诗尺度、现实目击、神秘压迫、纪录片式灾难真实感、好莱坞大片级宽银幕画面。

## 现场感优先

电影截帧不要像孤立角色展示。即使主体非常奇幻，也要让空气、光线、重力、材质、天气和镜头反应可信。

优先写出这些可见信息：

- 前景有人、车、伞、窗框、门缝、残墙、船舷或肩部遮挡。
- 中景有街道、桥梁、柱廊、楼体、山脊、浪面、车队或人群。
- 远景由主体、风暴、山体、城市、天象或神迹压住。
- 空气中有雨、雾、尘、水汽、烟、雪、灰烬、逆光颗粒。
- 地面有积水、泥土、玻璃、碎石、脚印、车灯反射。

## 短剧短片场景补全

短剧和短片图片通常不需要巨物，但更需要“这一秒有戏”。补全时优先考虑：

- 人物关系：谁掌握主动权，谁在隐瞒，谁刚意识到真相。
- 动作前后：画面像某个动作的前一秒或后一秒，而不是僵硬摆拍。
- 表情克制：避免夸张哭喊，使用眼神、手势、站位和沉默表达冲突。
- 环境叙事：旧出租屋、医院走廊、便利店夜班、地下停车场、婚宴后台、县城街口、办公室玻璃隔间都可以提供故事。
- 道具线索：手机未读消息、皱掉的合同、湿掉的外套、半杯冷茶、摔碎的口红、掉在地上的婚戒。

## 三层尺度

每张电影截帧都至少建立三层：

1. 人类尺度：人物、手、门、车、椅子、手机、伞、台阶。
2. 环境尺度：房间、楼道、街道、桥梁、山体、海浪、建筑。
3. 事件尺度：神迹、灾难、冲突、秘密、权力、巨物或情绪压迫。

不要只写“大”“震撼”“电影感”。用参照物表达尺度和情绪。

## 电影截帧模板

```text
【最终提示词】
[主体] 出现在 [具体场景] 中，[人物关系或事件压力]，[动作前后状态]，[环境和道具提供的叙事线索]。镜头位于 [观众位置]，[机位与角度]，[焦距与构图]，[前景遮挡或参照]，[中景空间]，[远景压力]。光线来自 [可信光源]，[空气介质与光线互动]，[色彩基调]。真实电影机拍摄，[画幅]，[摄影质感]，[曝光与高光行为]，[胶片颗粒与镜头细节]，超清电影画面，物理可信。

【负面约束】
避免廉价特效感、塑料表面、过度光滑、干净空白背景、均匀平光、海报式摆拍、主体完整居中平铺、没有环境叙事、没有前中后景、颜色过度鲜艳、材质过度完美。
```
