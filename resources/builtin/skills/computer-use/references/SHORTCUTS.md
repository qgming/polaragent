## 使用建议

### 常用操作的推荐方式

#### 1. 打开开始菜单
```typescript
await windows_keypress({ keys: ["Ctrl", "Esc"] });
```

#### 2. 切换窗口
```typescript
await windows_keypress({ keys: ["Alt", "Tab"] });
```

#### 3. 复制粘贴
```typescript
// 复制
await windows_keypress({ keys: ["Ctrl", "C"] });

// 粘贴
await windows_keypress({ keys: ["Ctrl", "V"] });
```

#### 4. 保存文件
```typescript
await windows_keypress({ keys: ["Ctrl", "S"] });
```

#### 5. 关闭窗口
```typescript
await windows_keypress({ keys: ["Alt", "F4"] });
```

## 快捷键组合规则

### 修饰键顺序
按键数组中的顺序：
```typescript
["Ctrl", "Shift", "Alt", "字母/数字/功能键"]
```

### 示例
- `Ctrl+Shift+T`: `["Ctrl", "Shift", "T"]`
- `Alt+F4`: `["Alt", "F4"]`
- `Ctrl+Alt+Delete`: `["Ctrl", "Alt", "Delete"]`

### 特殊按键名称
- 功能键: `F1`, `F2`, ..., `F12`
- 方向键: `Up`, `Down`, `Left`, `Right` 或 `↑`, `↓`, `←`, `→`
- 其他: `Enter`, `Tab`, `Esc`, `Space`, `Delete`, `Backspace`, `Home`, `End`, `PageUp`, `PageDown`

## 注意事项

1. **大小写敏感**: 按键名称不区分大小写，`Ctrl` 和 `ctrl` 都可以
2. **Win 键不可用**: 所有包含 Win 键的快捷键都需要替代方案
3. **应用特定**: 某些快捷键只在特定应用中有效
4. **延迟**: 连续按键之间可能需要短暂延迟
