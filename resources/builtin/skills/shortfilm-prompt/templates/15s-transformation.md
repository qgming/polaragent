# 15 秒单镜头变身模板

> 适用于：特摄式变身、武器激活、能量场展开等单镜头一镜到底类型。
> 写法 A（按秒切片），4-5 个时间分段。

---

## 适用场景

- **视频类型**：单镜头变身 / 能量激活 / 装甲展开
- **推荐时长**：10-15 秒（Seedance 最长 15s，矮于 10s 节奏太赶）
- **写法**：A（按秒切片）
- **推荐模型**：Seedance 2.0、可灵 3.0、Veo 3
- **参考运镜**：极缓推近 + 环绕 + 退步拉远（见 [camera-move-library.md](../references/camera-move-library.md) #1/#12/#5）

---

## 五段式骨架

### 段 1 · 核心主题

```
核心主题：{写实/暗黑/赛博等风格标签} | {核心动作标签} | {美学标签} | {战损/瑕疵标签} | {场景标签}
```

标签用 `|` 分隔，3-6 个。从"画面类型 → 题材 → 美学风格"层层递进。

### 段 2 · 人物与基础设定

```
【人物与基础设定】
面部：参照上传图片，五官、脸型、发型百分百还原，杜绝美化。
      {面部瑕疵：轻微擦伤/淤青/雨水/疲惫纹}，{表情：眉心微蹙/目光冷}。
服装：{质地描述：哑光深绿钛合金/皮质/布料}，{服装瑕疵：关节油污/磨损掉漆/划痕}，
      {核心装备描述：胸甲核心/能量装置}，整体远非平整，磨损痕迹清晰可见。
场景：{动态场景描述：暴雨砸金属面/硝烟弥漫/闪电间歇}，
      {标志性环境元素}。
```

⚠️ 注意：
- 服装写**质地**（哑光黑色皮质）而非名称（黑色皮衣）
- 场景写**动态**（微风/硝烟/陨石）而非静态
- 面部/服装/场景三行必须各含 ≥1 处瑕疵

### 段 3 · 氛围与画质

从 [atmosphere-prefabs.md](../references/atmosphere-prefabs.md) 选「史诗/大场面」或「暗调赛博」预制件：

```
【氛围与画质】
视觉基调：变形宽荧幕电影质感。模拟 IMAX 胶片摄影机，
        搭配 Panavision C 系列镜头（焦段 35mm，光圈 f4）。
色彩与影调：{色调词}，高对比度，暗部信息压缩保留细节，
          边缘添加适度柔焦与胶片颗粒感。
风格核心：{具体美学风格}。强调{物理反馈描述}，
        营造{具体情绪描述}。
声音：不需要配乐，仅保留同期声（{雨声}/{雷声}/{金属摩擦}/{能量低频嗡鸣/...}）。
```

### 段 4 · 运镜规则

```
【运镜规则】
镜头：一镜到底，无剪辑。
角度：开场仰拍人物右侧 30 度面部特写，
     {激活段：极缓速度环绕至正面胸甲核心}，
     {展开段：环绕至全身大全景}，逐渐拉远。
呼吸感：手持拍摄，全程保持极其轻微的、如呼吸般的镜头浮动，增强临场感。
```

### 段 5 · 分镜（按秒切片 · 写法 A）

```
0-3 秒 · {阶段名：凝视/等待/准备}
动作：{具体动作，非空泛词}
镜头：{运镜描述}
特效：{物理效果，非纯视觉叠加}
[声音：{环境音}]

3-6 秒 · {阶段名：启动/激活/触发}
动作：{触发动作，避 IP 词}
特效：{能量/变形/物理反馈}
镜头：{运镜描述}
[声音：{启动声效}]

6-9 秒 · {阶段名：裂变/展开/撕裂}
{核心变身过程}
身体：{身体变化描述}
装备：{装备变化描述，含瑕疵：部分单元故障般闪烁}
面部：{面部反应}
镜头：{运镜含应激微颤}

9-12 秒 · {阶段名：生长/成型/凝聚}
{变身接近完成}
{{新结构}}：{从哪里延伸/展开，覆盖速率不均匀，部分单元故障般闪烁}
{环境反应}：{能量场对环境的影响：推开雨水/搅动碎屑/等}
镜头：{环绕至正面，捕捉完成观}

12-15 秒 · {阶段名：完成}
{{变身产物}}：{描述最终状态，保留 1-2 个"故障"区域不发光}
装备：{发光部位维持脉冲频率描述}
环境：{环境恢复/持续}
最终镜头：环绕至人物侧面，镜头以极缓速度向后拉远，展示{主角在环境中的全景}。

结尾：没有台词，没有爆炸，没有光芒万丈。只有{主角}{动作}，{环境细节}。
```

---

## 此类型的特殊规则

1. **结尾留白**（硬规则 6）：最后一段**禁止**写光芒万丈/爆炸/胜利姿态，用留白模板结束
2. **触发动作避 IP**（硬规则 7）：用"按压胸甲核心"代替"像钢铁侠一样"，用"低吟 + 自创音节"代替特定变身台词
3. **物理反馈**：能量场/护盾/变身效果要有**环境反应**——推开雨水、搅动碎屑、瞬间汽化——而非纯视觉效果叠加
4. **故障瑕疵**：变身完成时保留"部分单元故障般闪烁"或"1-2 个区域不发光"，完美=假
5. **单镜头不超过 15 秒**：超过 15 秒，AI 视频模型成功率暴跌
