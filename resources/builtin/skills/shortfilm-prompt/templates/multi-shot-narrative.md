# 多分镜剪辑叙事模板

> 适用于：多镜头叙事短片、人与动物相遇、末日公路、悬疑短片等。
> 写法 B（按镜头切片），4-8 个分镜。

---

## 适用场景

- **视频类型**：多镜头剧情叙事 / 人与动物相遇 / 短片故事
- **推荐时长**：20-30 秒（4-6 个分镜，每镜 4-7 秒）
- **写法**：B（按镜头切片）
- **推荐模型**：可灵 3.0（物理运动好）、Sora 2 Pro（最长 25s）、Veo 3
- **参考运镜**：多机位 + 正反打 + 空镜 + 平行跟拍（见 [camera-move-library.md](../references/camera-move-library.md) #19/#27/#39）

---

## 五段式骨架

### 段 1 · 核心主题

```
核心主题：{画面类型标签} | {题材标签} | {美学标签} | {叙事标签} | {场景标签}
```

例：`原子朋克 | 末日废土 | 电影级质感 | 超写实 | 废墟偶遇`

### 段 2 · 人物与基础设定

```
【人物与基础设定】
面部：{角色 A}：参照上传图片，五官、脸型、发型百分百还原，杜绝美化。
      {面部瑕疵}，{表情描述}。
      [{角色 B}：同上格式，若有多角色]
服装：{角色 A 服装质地 + 瑕疵}，{角色 B 服装质地 + 瑕疵}。
场景：{动态环境描述}，{标志性环境元素}，{天气/光线/氛围细节}。
```

⚠️ 多角色时每个角色各写面部+服装。动物角色用"外观"代替"面部+服装"。

### 段 3 · 氛围与画质

从 [atmosphere-prefabs.md](../references/atmosphere-prefabs.md) 任选适合的预制件，替换 `{占位符}`。

**多分镜特别注意**：全片锁一档调色，不要每个分镜换色调/滤镜。如果需要时间推进感，用光线/季节变化（同暖调档位内变化），而非换滤镜。

### 段 4 · 运镜规则

```
【运镜规则】
镜头：按分镜剪辑，{分镜数量} 个镜头。
风格：{运镜风格描述：手持纪实/稳定器平滑/混合}。
呼吸感：手持拍摄，全程保持极其轻微的、如呼吸般的镜头浮动，增强临场感。
```

### 段 5 · 分镜（按镜头切片 · 写法 B）

```
分镜一：
景别：{远景/全景/中景/近景/特写}
构图：{画面位置/三分法/居中/留白方向}
运镜手法：{固定/推近/横移/跟拍/…}
画面内容：{具体画面描述，含动作/环境/氛围}

分镜二：
景别：…
构图：…
运镜手法：…
画面内容：…

...

分镜 N：
景别：…
构图：…
运镜手法：…
画面内容：…
```

**每个分镜必须包含 4 件套**：景别 / 构图 / 运镜手法 / 画面内容。

**必须包含**：
- 至少 1 个**空镜**（无人物，只有环境/物件，用于场景建立或情绪停顿）
- 至少 1 个**过肩镜头**或**主观视角**（建立空间关系/人物关系）
- 动物角色：写**具体动作**而非拟人化情绪（"猫耳朵竖起"而非"猫感到好奇"）

---

## 此类型的特殊规则

1. **写法 B 严格 4 件套**：每个分镜必须含景别/构图/运镜/画面内容，不要偷懒只写动作
2. **不要死守写法 A**：多分镜叙事不能按 0-3s/3-6s 切，必须按镜头切
3. **动物不拟人**：猫的表情写"耳朵竖起/慢慢探出半个头/尾巴轻甩"，不写"惊讶/犹豫/信任"
4. **空镜不可省**：空镜是叙事的呼吸点，省了会让节奏像跑步不停
5. **情绪用画面传达**：不堆"温馨/感人/治愈"这类情绪空洞词，用具体动作/画面传达
6. **不得引用 IP 名**：《机器人总动员》→"60 年代原子朋克 + 后末日废土"
7. **物理动作要具体**：可灵容易把"投喂罐头"做成飞罐头特效——写明"手缓慢推过去，罐头停在地面 0.5 米外"

## 反向提示词追加项

在标准前缀基础上追加：

```
jarring transitions, mismatched lighting between shots, inconsistent character appearance, changing face between shots, cartoon animal expressions, anthropomorphic animal poses
```
