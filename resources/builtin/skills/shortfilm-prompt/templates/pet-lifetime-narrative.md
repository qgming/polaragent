# 情感叙事模板（亲情 · 萌宠 · 离别 · 一生陪伴）

> 适用于：宠物一生陪伴、亲人离别、成长回顾、时光流逝等催泪叙事。
> 写法 B（按镜头切片），时间靠季节+光线推进。
> 这是 5 段式在情感叙事上的特殊适配——同样的"瑕疵 + 克制"纪律，让情感片打中人。

---

## 适用场景

- **视频类型**：情感叙事 / 萌宠亲情 / 离别催泪 / 时光陪伴
- **推荐时长**：25-40 秒（5-8 个分镜，每镜 4-6 秒）
- **写法**：B（按镜头切片）
- **推荐模型**：可灵（宠物运动稳）、Veo 3、Sora 2 Pro
- **参考运镜**：极缓推近 + 空镜静物固定 + 手持近景（见 [camera-move-library.md](../references/camera-move-library.md) #3/#41/#24）

---

## 情感叙事三条适配规则

这三条是情感叙事的核心纪律，每一条都来自 5 段式方法在情感片中的验证：

### 1. 时间靠季节 + 光线推进，调色锁一档

每个分镜换一种滤镜是情感多分镜**最常见的翻车点**。反过来做：

> "窗外季节在变，屋内暖光不变。"

- 时间推进靠：窗外树叶颜色 / 光线角度 / 地面影子长度 / 窗外雨或晴
- 调色锁一档：全片暖琥珀，不换滤镜，不换色调
- **可接受的变化**：同一暖色调内的光线强弱（清晨光→正午光→黄昏光），但**不能**从暖调跳到冷调

### 2. 克制本身在催泪（硬规则 6 用在情感上）

不要闪回蒙太奇、不要配乐渐强、不要慢推泪脸。

> 空 位替你 哭：空门槛上的旧项圈、一片落叶、被风吹动的空牵引绳、茶几上残留半杯水的旧杯子。

**给「缺席」，而不是「对缺席的反应」。**

结尾可以这样写：
- "只有旧项圈挂在门槛上，门半开着，风穿过客厅。"
- "只有空狗窝旁的落叶缓慢旋转，院子里的风继续吹。"

### 3. 每个主体 2 处瑕疵锚点 = 一致性锁

AI 视频最容易做的错事：**中途把你的狗/主角换成了另一只**。

解决方案：每个主体给 ≥2 处固定的、跨镜头一致的外观瑕疵锚点：

- 狗：磨旧皮项圈 + 灰白口鼻 + 左前爪干泥
- 人：右膝擦伤旧疤 + 左手虎口老茧 + 疲惫纹

这些锚点**每个镜头都要出现**，它们把"同一只狗/同一个人"钉死在每个画面里。

> **建议：先生成第一镜和最后一镜，锁定主体外观后，再补中间镜头。**

## 五段式骨架

### 段 1 · 核心主题

```
核心主题：{写实日常} | {陪伴/离别标签} | {暖调胶片标签} | {克制叙事} | {时光标签}
```

例：`写实日常 | 一生陪伴 | 暖调胶片叙事 | 克制催泪 | 季节交替`

### 段 2 · 人物与基础设定

```
【人物与基础设定】
{动物主角}：
外观：{品种描述 + 毛色 + 体型}，
     瑕疵锚点 1：{如：磨旧皮项圈}，瑕疵锚点 2：{如：左耳有个缺口}。
     幼年期→老年期变化：{体型渐大→毛色渐灰→步态变慢}。
{人类主角}：
面部：参照上传图片，五官、脸型百分百还原，杜绝美化。
     瑕疵锚点 1：{如：右膝擦伤→旧疤}，瑕疵锚点 2：{如：左手虎口老茧}。
     年龄变化：{矮→高→肩膀渐宽→有疲惫纹}。
场景：{家庭/院落/城市空间}，{光线特征}，{标志性物件：旧狗窝/门槛/窗台}。
```

### 段 3 · 氛围与画质

强烈建议使用 [atmosphere-prefabs.md](../references/atmosphere-prefabs.md) 的「温暖胶片 / 情感叙事」预制件：

```
【氛围与画质】
视觉基调：模拟 ARRICAM ST 胶片摄影机，
        搭配 Cooke S4 Mini 镜头，Kodak Vision3 250D 胶片。
色彩与影调：全片暖琥珀一档调色。窗外季节光变化，
          但室内光源色调不变，贯穿全片锁一档。
风格核心：日常真实温度感。克制冷色，
        用自然光和窗外光线塑造时间推进感。
声音：不需要配乐，仅保留同期声（{奶狗呜咽}/{雨打窗}/{老狗缓慢呼吸}/{牵引绳扣声}/{院子里的风}）。
```

⚠️ **显式写**"无配乐、无闪回、仅同期声"——AI 会默认给情感戏加配乐和闪回。

### 段 4 · 运镜规则

```
【运镜规则】
镜头：按分镜剪辑，{N} 个镜头。
风格：缓推慢移为主，间以固定空镜。节奏比其他类型慢 30%。
呼吸感：手持拍摄，全程保持极其轻微的、如呼吸般的镜头浮动，增强临场感。
```

⚠️ **不要写**快切/闪回蒙太奇/慢推泪脸/配乐渐强。

### 段 5 · 分镜（按镜头切片 · 写法 B）

```
分镜一 · {标签：幼年/春/初遇}
景别：{近景/中景}
构图：{人物与动物的空间关系}
运镜手法：{极缓推近/手持跟拍}
画面内容：{具体互动：动作 + 瑕疵锚点出现 + 环境光线季节}

分镜二 · {标签：成长/夏}
...

...

分镜 N · {标签：晚年/冬/空位}
景别：{空镜，无人物}
构图：{空门槛/旧狗窝/遗留物居中}
运镜手法：{三脚架固定，3-5 秒零运动}
画面内容：{空位/遗留物/风/光线——无声告别}

结尾：没有闪回，没有配乐，没有泪脸特写。只有{空位描述}。
```

**分镜结构要点**：
- 前 1-2 镜：幼年初遇（暖光 + 活力 + 两个主体靠在一起）
- 中间 3-5 镜：相处日常（季节递进 + 各自瑕疵锚点持续出现）
- 后 1-2 镜：老去/缺席（光线变暗 + 空镜 + 克制告别）
- **最后一镜必须是空镜**：旧项圈/空牵引绳/遗留物——三个以上的"无"（无人、无声、无反应）

## 此类型的特殊规则

1. **全片锁一档调色**：最常翻车点是在不同镜头间换滤镜/色调。全片暖琥珀一档，靠光线强弱（同档位内）变化传递时间
2. **结尾三禁**：禁止闪回蒙太奇 / 配乐渐强 / 慢推泪脸。用空镜 + 缺席代替
3. **瑕疵 = 一致性锁**：每个主体 ≥2 处固定瑕疵锚点，且**每个镜头都要出现**。AI 最常犯的错是换了一只狗
4. **先生成首末两镜**：锁定两个主体的外观后再补中间镜头
5. **动物不拟人**：写具体动作（"老狗缓慢把灰白口鼻搁在膝上，不动"）而非情绪词（"狗不舍地看着主人"）
6. **"老去"≠换狗**：明确写"灰白口鼻/浑浊眼/步态变慢"是**同一只**狗的变化而非换狗
7. **不堆情绪空洞词**：不写"温馨/感人/治愈/泪目"，用画面传递
