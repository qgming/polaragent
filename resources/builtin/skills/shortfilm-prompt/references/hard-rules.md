# 七条硬规则（写完自检）

> 这是用 TDD 方法反推出来的「未加载 skill 的 Claude 最容易翻车的 7 个点」。
> **每次输出前在脑子里过一遍这 7 条**，不合规的话改了再交付。

---

## 规则 1：每段都必须有具体名词，禁用空泛美化词

| ❌ 禁用 | ✅ 替换 |
|---|---|
| 电影感 / 史诗感 / 大片感 | "IMAX 胶片摄影机 + Panavision C 系列镜头 35mm f4" |
| 震撼 / 炫酷 / 史诗 / 完美 | 删掉或换成具体物理效果（"画面边缘被轻微拉伸"） |
| 帅气 / 冷峻 / 凛冽 | "眉心微蹙" / "目光中带一丝轻蔑" / "脊背紧绷" |
| 高级感 / 质感拉满 / 细节满满 | "釉面光泽" / "金属拉丝" / "胶片颗粒感" |
| 4K / 高清 / 高画质 | 不写。写"低饱和灰蓝主调，胶片颗粒感"等具体视觉描述 |
| 神秘 / 奇幻 / 未来感 | 写具体视觉："暗部藏有未点亮的灯管"、"金属表面有冷凝水珠" |
| 温馨 / 感人 / 治愈 | 写具体动作/画面："手指缓慢合拢，掌心留有狗毛"、"空门槛上挂着旧项圈" |
| 科技感 / 未来感 / 赛博感 | 写具体细节："每 1 秒一次翠绿色脉冲发光"、"颈后散热孔排出细线热气" |

**自检方法**：从输出里随便挑 3 个形容词，问自己"这个词 AI 看了能产生具体画面吗？" 不能 → 删 / 替换。

## 规则 2：每个视频提示词必须包含摄影机型号 + 镜头型号

选配详见 [camera-lenses.md](camera-lenses.md)。

**核心组合（快速参考）**：
- 史诗感大场面：**IMAX 胶片摄影机 + Panavision C 系列镜头**（35mm，f4）
- 暗调赛博 / 写实硬核：**索尼威尼斯电影机 + 佳能 K-35 系列镜头**
- 港片武侠：**柯达 35mm 复古胶片**，跳过漂白胶片质感
- 商业人像（用于生图）：**Canon EF 85mm f/1.2**
- 温暖胶片 / 情感叙事：**ARRICAM ST + Cooke S4 Mini**，Kodak Vision3 250D
- 冷调纪实：**索尼威尼斯 2 + Zeiss Super Speed MK III**
- 赛博朋克夜景：**ARRI ALEXA Mini LF + Arri Signature Prime**
- 自然风光：**RED V-RAPTOR + Leica Thalia**

**自检方法**：搜输出里有没有上述任一组合名 —— 没有就补。

## 规则 3：永远加"呼吸感"那一行

精确句式（**不能简化**）：

> "手持拍摄，全程保持极其轻微的、如呼吸般的镜头浮动，增强临场感。"

- "如呼吸般"和"极其轻微"两个限定词缺一不可
- 省略任何一个，AI 会理解为剧烈摇晃
- 仅在**定格 / 黏土动画 / CCTV 伪纪录**模板中可以打破此规则（用"三脚架固定拍摄，零镜头浮动"替代）

---

## 规则 4：永远加"声音"那一行

```
声音：不需要配乐，仅保留同期声。
```

如果场景有标志性环境音，**显式枚举**（不要让 AI 猜）：

- 暴雨场景：雨声、雷声、金属摩擦声、能量低频嗡鸣
- 室内情感：奶狗呜咽、雨打窗、牵引绳扣声、院子里的风
- 地下管廊：脚步回声、金属摩擦、等离子嗡鸣、虫子爪甲声、喘息
- 废墟末日：风沙穿堂、碎屑落地震动、远处金属嘎吱

### 例外

- **音乐 MV / 舞蹈编舞**模板：这里才应该有音乐。写"踩点节拍"而非"配乐渐强"。
- **产品广告**模板：可加品牌音效或产品特有声音。

## 规则 5：人物 / 装备 / 战衣段必须至少 2 处瑕疵描述

Mx-Shell 反复强调：**"过于完美，就假。适当地保留缺陷不是坏事。"**

### 候选词表

**面部**：
- 保留轻微面部瑕疵 / 面部伤口、纱布、血渍 / 嘴角有血渍 / 淤青
- 眼底有疲惫纹 / 眉间细纹 / 嘴角微裂 / 灰白口鼻（动物）

**装备 / 服装**：
- 磨损掉漆 / 关节油污 / 细微划痕使用痕迹明显 / 战损痕迹触目惊心
- 袖口磨出毛边 / 鞋面有泥渍 / 肩缝微开线 / 拉链缺齿

**状态**：
- 战衣整体远非平整 / 部分单元故障般闪烁 / 一道旧伤被重新撕开
- 磨旧项圈 / 灰白口鼻 / 爪上干泥

**环境**（可计入场景段瑕疵）：
- 碎屑满地 / 锈迹斑驳 / 断裂管道 / 剥落墙皮 / 裂缝穿过地面

**自检方法**：输出里数瑕疵词，少于 2 处 → 加。

### 情感叙事中的瑕疵 = 一致性锁

在情感叙事（萌宠/亲情/离别）中，每个主体 ≥2 处瑕疵锚点不仅是真实感要求，更是**一致性锁**：
- 狗：磨旧项圈 + 灰白口鼻 + 爪上干泥 → 在每个镜头中这些锚点钉死"是同一只狗"
- 人：擦伤膝盖 → 旧疤 → 疲惫纹 → 在时间推进中这些锚点钉死"是同一个人"

**建议**：先生成第一镜和最后一镜，锁定主体外观后再补中间镜头。

## 规则 6：单镜头变身 / 史诗 / 情感段的结尾不要堆特效

### 变身 / 史普段结尾

不要写：光芒万丈 / 爆炸特效 / 胜利姿态 / 凌空一跃 / 镜头炸开

**默认结尾模板**：
> "没有台词，没有爆炸，没有光芒万丈。只有 {{主角}} {{动作}}，{{环境细节}}。"

例子：
- "只有身穿不完整战衣的人站在原地，风吹过战场硝烟，远处天空划过陨石。"
- "只有暴雨持续打在能量场上，被瞬间汽化的水雾环绕着主角。"

### 情感叙事结尾

不要写：闪回蒙太奇 / 镜头光晕 / 屏幕爱心 / 配乐拉满 / 慢推泪脸

**克制 = 催泪**。用"缺席"和"空位"替你哭：
- 空门槛上的旧项圈
- 磨旧的狗窝旁一片落叶
- 被风吹动的空牵引绳
- 茶几上残留半杯水的旧杯子

> 给「缺席」，而不是「对缺席的反应」。

## 规则 7：避开 IP 词 + 模型拦截提示

### 基本原则

不要照搬具体 IP 名：仮面ライダー / 高达 / 钢铁侠 / 假面骑士 / 卡莎 / MJ / 黑客帝国 / 漫威 / 复仇者联盟 / WALL-E / 辐射 ...

### 替代写法

| ❌ IP 写法 | ✅ 替代写法 |
|---|---|
| "参考钢铁侠" | "原子朋克未来主义复古风格" |
| "迈克尔·杰克逊舞蹈" | "1980 年代标志性街舞动作风格（卡点转头/耸肩/太空步/压帽子顶胯 wave）" |
| "BLACK SUN 美学" | "暗黑写实战损美学" |
| "《机器人总动员》" | "60 年代原子朋克 + 后末日废土" |
| "《辐射》" | "核战后废土复古未来主义" |
| "高达" | "真人大小的拟人作战机甲" |
| "仮面ライダー" | "实战型穿戴式装甲变身" |

### 用户坚持要用 IP 名

照写用户要求的 IP 名，但**末尾必须加**一行提示：

> ⚠️ 这里用了 IP 名（{IP名}），目标模型可能拦截，建议替换为描述性写法或删除部分标点试试。

同时给出 1-2 个**替换方案**，让用户在被拦截时有备选。

**型号+数字组合更易被拦**（如 mark42）：额外提示"型号数字可以删，仅保留描述性写法"。

## 规则冲突优先级

当多条规则看似冲突时，按以下优先级处理：

| 优先级 | 规则 | 说明 |
|---|---|---|
| 1 | 用户明确要求 | 用户坚持用 IP 名时照写+提醒（规则 7 第二优先级例外） |
| 2 | 规则 1（禁空泛词） | 任何描述都必须可产生具体画面，这是底线 |
| 3 | 规则 6（结尾留白） | 情感叙事中尤其重要，克制 > 煽情 |
| 4 | 规则 5（瑕疵） | 完美 = 假，但瑕疵数可灵活（≥2 即可） |
| 5 | 规则 3（呼吸感） | 可在定格/CCTV 模板中打破 |
| 6 | 规则 4（声音行） | 可在 MV 模板中打破 |
| 7 | 规则 2（摄影机型号） | 组合可自由选择，只要有 |
| 8 | 规则 7（避 IP 名） | 用户明确要求时照写+提醒 |

### 典型冲突场景

- **规则 3 vs 定格动画**：定格/黏土模板中，"呼吸感"改为"三脚架固定拍摄"—— 页面跳帧是定格的美学特征
- **规则 4 vs 音乐 MV**：MV 模板中可以有音乐，但写"踩点 120BPM 电子节拍"而非"配乐渐强"
- **规则 5 vs 产品广告**：广告中的产品应当完美，瑕疵加在环境/人物而非产品上
- **规则 6 vs 动作片高潮**：打斗终结段可以有"决定性一击"，但不用"光芒万丈/爆炸"。用物理效果："刀刃划过关节，火花飞溅，虫体倒地抽搐"
