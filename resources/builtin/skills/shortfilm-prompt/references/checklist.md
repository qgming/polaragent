# 30 秒自检清单（分层版）

> 写完提示词后，按这个核对再交付。
> 分三层：A 层必过（缺一不交），B 层按题材选检，C 层按模型选检。

---

## A 层 · 核心结构（必须全过，缺一不交）

- [ ] **5 段结构齐全**：核心主题 / 人物设定 / 氛围画质 / 运镜规则 / 分镜
- [ ] **有摄影机型号 + 镜头型号**（硬规则 2，查 [camera-lenses.md](camera-lenses.md)）
- [ ] **有"如呼吸般的镜头浮动"完整句式**（硬规则 3，不可简化）
- [ ] **有"声音：不需要配乐，仅保留同期声"**（硬规则 4，显式枚举环境音）
- [ ] **至少 2 处瑕疵描述**（硬规则 5，数瑕疵词）
- [ ] **结尾不堆特效，留白**（硬规则 6，不出现"光芒万丈/爆炸/胜利姿态/闪回"）
- [ ] **没有空泛词**（硬规则 1，不出现"完美/震撼/史诗/帅气/4K/质感拉满/科技感/温馨/感人"）
- [ ] **没有具体 IP 名 OR 有则末尾加提示**（硬规则 7）

**A 层 8 条全过 → 继续 B/C；不过 → 修改后再检。**

---

## B 层 · 题材适配（按视频类型选检）

### 变身 / 充能类

- [ ] 分镜用写法 A（按秒切片），每段有时间标记
- [ ] 结尾用留白模板（"没有...没有...只有..."）
- [ ] 触发词不写 IP 名
- [ ] 特效写成物理反馈而非纯视觉叠加

### 多分镜叙事类

- [ ] 分镜用写法 B（按镜头切片），每个分镜含 4 件套：景别 / 构图 / 运镜手法 / 画面内容
- [ ] 4-8 个明确编号的分镜
- [ ] 包含空镜或过肩镜头
- [ ] 不堆情绪空洞词（"温馨/感人/治愈"→ 换具体动作/画面）

### 分段视频（充能 + 打斗等）

- [ ] 明确建议分两段独立生成
- [ ] 两段共享同一套氛围与画质设定（确保剪辑后不出色差）
- [ ] 打斗段写"拒绝游戏 CG 感"
- [ ] 给出后期剪辑建议

### 情感叙事（亲情 · 萌宠 · 离别）

- [ ] **全片锁一档调色**（不是每个镜头换滤镜/调色）
- [ ] **结尾克制**：用"空位/遗留物"催泪，不堆闪回/配乐渐强/慢推泪脸
- [ ] **每个主体 ≥2 处瑕疵锚点**（同时是一致性锁：磨旧项圈/灰口鼻/爪上泥 等）
- [ ] **建议先生成第一镜和最后一镜**锁定主体外观，再补中间
- [ ] 时间靠季节 + 光线推进，而非靠字幕/解说

### 产品广告类

- [ ] 产品本身无瑕疵（瑕疵加在环境/人物而非产品上）
- [ ] 有分秒 beat 精确控制
- [ ] 产品材质/光泽有具体描述（非"高级感"）

### 竖屏短剧类

- [ ] 前 2 秒有钩子（反常规画面 / 紧张悬念 / 意外元素）
- [ ] 正反打镜头结构
- [ ] 有悬念切点

---

## C 层 · 模型适配（按目标模型选检）

- [ ] **时长在目标模型支持范围内**（如 Seedance 消费版 ≤10s，可灵 ≤10s/15s，Sora Pro ≤25s）
- [ ] **语言与目标模型偏好匹配**（Sora/Runway/Pika → 英文；Wan → 偏中文；其余 → 中英皆可）
- [ ] **反向提示词已按模型分流正确处理**（见 [negative-prompts.md](negative-prompts.md)）
- [ ] **末尾给目标模型兼容性建议**（1-2 句，如 Seedance 避 IP / Veo 用纯名词反向 / Runway 只写正向）
- [ ] **IP 相关提示已加**（如需要）
- [ ] **平台差异已注明**（如豆包 App vs 即梦网页的时长/反向框差异）

---

## 评分规则

| 层 | 达标条件 |
|---|---|
| A 层 | 8/8 全过（缺一不交） |
| B 层 | 对应题材的子项全过 |
| C 层 | 对应模型的子项全过 |

**通过标准**：A 层 8/8 + B 层对应子项全过 + C 层对应子项全过 = **合规交付**。
