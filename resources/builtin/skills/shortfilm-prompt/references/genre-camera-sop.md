# 类型片运镜 SOP

> 按视频类型决定怎么运镜。
> 每个类型标注了：推荐时长、适合镜头组合、运镜要点、呼吸感是否保留、特殊规则打破说明。

## 各类型详细说明

### 15 秒单镜头变身

**运镜核心**：一镜到底，从特写入手→环绕→大全景拉远。

- 开场：仰拍人物侧面特写，极缓向前推进
- 激活：镜头以极缓速度环绕至正面胸甲核心
- 展开：护盾/变身完全展开后环绕至全身大全景，逐渐拉远
- 呼吸感：全程保留，"手持拍摄，全程保持极其轻微的、如呼吸般的镜头浮动"
- 角度变化：侧 30° → 正面 → 俯拍/全景，不超过 3 个视角变化

### 武器充能 + 打斗

**两段独立**，各自有完整 5 段结构。

**充能段（10s）**：
- 固定中景→武器特写→全身激活全景
- 运镜偏缓，让充能过程有时间感

**打斗段（15s）**：
- 快速近景切换→慢动作终结→短暂喘息全景
- 运镜节奏紧凑，写"动作符合真实物理定律，拒绝游戏 CG 感"
- 角度快速切换但不晃动

### 多分镜剪辑叙事

**4-5 个分镜必备要素**：
- 每个分镜写 4 件套：景别 / 构图 / 运镜手法 / 画面内容
- 至少 1 个空镜（建立场景/时间推进/情绪停顿）
- 至少 1 个过肩镜头（建立人物关系/对望感）
- 景别变化有节奏：远-近-远-近，不要全近或全远

### 情感叙事（亲情 · 萌宠 · 离别）

**运镜节奏**：比其他类型慢 30%。
- 缓推慢移：室内场景用极缓推镜头，2-3 秒才推近一个身位
- 固定空镜：故居/空门槛/旧项圈 —— 固定镜头 3-5 秒不动，让时间自己讲故事
- 手持近景：人与动物互动时用手持近景，轻微浮动传递温度
- **不要快切、不要闪回蒙太奇、不要慢推泪脸**

### 产品广告

**运镜两极**：稳到极致 + 微距极致。
- 产品全貌：稳定器平滑拍摄，零浮动，旋转展台式环绕
- 材质细节：微距推进，极浅景深，修复化纹理反光
- 使用场景：手持跟拍，轻呼吸感
- ⚠️ 呼吸感用"轻度"版本："稳定器平滑拍摄，保持极微弱的镜头呼吸感"

### 竖屏短剧

**运镜 = 节奏控制器**。
- 前 2 秒钩子：特写/俯拍/反常规角度，一秒抓眼球
- 正反打：对望/争执场景用标准正反打，切点在台词句末
- 悬念切：关键信息前切走，观众脑补
- 节奏公式：`钩子(2s) → 正常叙事 → 悬念切 → 反转 → 收尾`

### 赛博城市 / 氛围环境片

**两种运镜路线**：
- **一镜穿越**（写法 A）：仿 FPV 一镜飞过城市，持续推进，霓虹掠过镜头
- **多分镜蒙太奇**（写法 B）：建筑单片 + 街道运动 + 霓虹特写 + 城市全景

### 食物 ASMR

**运镜极简**：少动多看。
- 微距定焦：焦段拉到最近，景深极浅，看食材纹理
- 俯拍固定：料理全程俯拍，烹饪动作取代镜头运动
- 慢动作：关键瞬间（溅油/倒酱/切开）用慢动作
- ⚠️ 呼吸感极微："镜头有极轻微的触感微颤，如同手接触桌面的振动"

### 拟人动物 VLog

**仿自拍运镜**：
- 自拍广角：仿前置摄像头自拍画面，略带桶形畸变
- 平视跟拍：与动物视线齐平，不是俯拍
- 稳定手持：呼吸感保留但偏轻，符合 VLog 手持感

### 电影预告片

**递进式剪辑**：
- 静态全景 → 中景介绍 → 快切递进 → 高潮慢推 → 标题卡
- 递进加速：前 1/3 每镜 3-4s，中间 2s，后 1/3 每镜 0.5-1s

### 定格 / 黏土动画

**⚠️ 打破呼吸感（硬规则 3 例外）**：
- 写"三脚架固定拍摄，零镜头浮动，画面有定格动画特有的轻微跳帧感"
- 不写"手持拍摄"—— 定格的质地靠跳帧表现
- 运镜只在分镜之间切换，单镜头内零运动

### 自然 / 风景延时

**⚠️ 打破呼吸感（硬规则 3 例外）**：
- 写"三脚架固定拍摄，零镜头浮动，时间以云影与光线移动感知"
- 色调锁一档（同情感叙事原则）：全片一种自然调色，不要每段换滤镜
- 时间感知靠自然元素：云影移动 / 光线角度变化 / 水面颜色渐变

### CCTV / 伪纪录恐怖

**⚠️ 打破多条规则**：
- 呼吸感：写"固定监控机位，无镜头运动，画质劣质 480p 以下"
- 不加摄影机型号：写"模拟 CCTV 安防监控画面，固定广角鱼眼，时间码水印"
- 故意"粗劣"：低帧率 / 过曝 / 拖影 / 色彩失真 / 时间码跳动
- 这是唯一"越假越对"的类型

### 动漫 → 真人写实

**媒介过渡运镜**：
- 匹配剪辑：2D 版和真人版用相同构图/景别切换
- 过渡可加：画面边缘出现 2D 线稿覆盖→真实化/2D 色块渐褪→真人

### 音乐 MV / 表演

**⚠️ 声音规则（硬规则 4）例外**：
- 这里才该有音乐，写"踩点 120BPM 电子节拍"而非"配乐渐强"
- 卡点剪辑：镜头切换精确踩在节拍上
- 全身跟拍：拍舞者时从头顶到脚底完整可见
- 环绕：高潮段用环绕镜头

### 运动高速慢镜

**运镜 = 时间变形**：
- Phantom 高帧率感：写"模拟 Phantom 高速摄影机 4000fps 慢动作"
- 决定性瞬间：1-2 秒真实速度 → 慢动作 → 真实速度
- 背景/前景运动模糊：主体清晰，环境模糊

### 时尚大片 / 编辑式

**运动即主角**：
- 无剧情要求，不需要叙事弧线
- 抽帧感：写"细微的抽帧/跳帧感，24fps 中丢失 2-3 帧"
- 剪影：写"半剪影，面部靠背光勾勒而非正面光"
- 环境即配饰：不用写"时尚感"，写具体材质反光

### 旅拍 Vlog

**手持蒙太奇**：
- 第一人称跟拍 + 手持重呼吸感 + 环境空镜交替
- 地标建筑用稳定全景，街巷用快切手持
- ⚠️ 呼吸感偏重："手持拍摄，保持明显的镜头呼吸感和接触感"

### 无人机 / FPV 航拍

**一镜连贯**：
- 运镜即内容：起→穿越→到达→全景，线型叙事
- 写 FPV 特定术语："穿越式 FPV 一镜，低空掠过→急速爬升→俯瞰全景"
- ⚠️ 呼吸感微颤："模拟无人机微颤，非手持式"

### 科幻太空 / 失重

**失重物理运镜**：
- 一切运动极缓：写"0.1x 速度漂浮，符合零重力物理"
- 悬浮物体取代参造物：散落的水珠/工具/文件悬浮缓慢旋转
- 真空静音感：声音段写"真空环境，无声音传播，仅偶尔有舱壁振动低频"

### 汽车广告

**反光即语言**：
- 车身反光拍摄：写"车身漆面反射天空/城市灯光，镜头追着反光走"
- 追车跟拍：写"跟车侧面平行拍摄，长焦压缩感，背景虚化"
- 内饰沉浸：前座视角，通过挡风玻璃看路

### 舞蹈编舞

**身体即镜头**：
- 全身可见：至少 70% 时间全身在画
- 身体卡拍：写"动作精确卡在节拍上，关节运动与音乐对齐"
- 环绕：高潮段用低位环绕，拍摄者跟随旋转
