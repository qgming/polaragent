## 写法提示

1. **摄影机 + 镜头要同时出现**：单独写 `IMAX` 不够，必须有 `+ Panavision C 系列 35mm f4` 这样的完整组合
2. **写在「氛围与画质」段的「视觉基调」行**：
   ```
   视觉基调：变形宽荧幕电影质感。模拟 IMAX 胶片摄影机，
           搭配 Panavision C 系列镜头（焦段 35mm，光圈 f4）。
   ```
3. **可以微调参数**：用户要求特定焦段效果时，可以调整焦段和光圈数字，但摄影机+镜头型号不变
4. **不要编造不存在的组合**：上表 8 组是经过验证的，不要自己发明
