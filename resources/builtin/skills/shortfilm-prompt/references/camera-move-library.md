## 使用指南

1. **从 5 段式第 4 段「运镜规则」中引用**：不要写"运镜很有动感"，写"极缓推近 + 低位环绕 180° + 退步拉远"
2. **组合使用**：一个视频通常组合 2-3 种运镜，不要全篇只写一种
3. **写清楚速度**：`极缓` / `匀速` / `快推 0.3 秒` —— 速度词 + 秒数 = 精确
4. **与呼吸感配合**：大部分运镜都在"手持拍摄 + 呼吸感浮动"的基础上叠加
5. **特殊运镜查上方**：CCTV/定格/延时等"打破呼吸感"的运镜在固定机位编号 38-41
