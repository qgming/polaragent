# 探索性分析与图表参考

## 探索性数据分析（EDA）

EDA 是任何分析项目的第一步：在正式建模/下结论之前，先识别数据中的一般模式、异常值和意料之外的特征。理解离群点出现在哪、变量之间如何关联，能帮你设计出有意义的分析。

**常见误区**：直接跳进处理与建模，而没有先理解数据的来源、含义和局限——这会导致误读与错误分析。

**EDA 实操步骤：**

1. **理解数据背景**：不仅知道有哪些变量，还要理解数据是在什么场景下采集的、由什么业务过程产生、有哪些潜在局限或偏差。
2. **看单变量分布**：用直方图查看每个变量的分布、范围、集中度与离群。
3. **看变量间关系**：用散点图和相关系数考察成对变量之间的关系。
4. **定位离群与缺失**：判断异常是真实现象还是录入错误，决定保留/修正/剔除。

## 图表选择

| 目的 | 推荐图表 | 说明 |
| --- | --- | --- |
| 趋势 / 时间序列 | 折线图 | 连续数据随时间变化 |
| 分类对比 | 柱状图 | 离散类别之间比较 |
| 占比 / 构成 | 饼图 | 类别建议 ≤6–7 个 |
| 关系 / 相关 | 散点图 | 两变量之间的关系 |
| 分布 / 频次 | 直方图 | 单变量分布、频率分析 |

## 图表清晰度与诚实性

- 标题、坐标轴标签清楚，单位与口径写明。
- 柱状图坐标轴一般从 0 起，避免视觉夸大差异。
- 多序列时配图例；选用色盲友好的配色。
- 减少冗余装饰，突出要传达的信息。
- **不做误导性截断**：截断坐标轴、挑时间窗、改口径都可能制造假象。

## 分析技术速览

- **描述统计**：均值、中位数、众数；标准差与方差；四分位与百分位；最小/最大/极差。
- **趋势分析**：移动平均、增长率、季节性识别、离群点识别。
- **相关分析**：Pearson 相关、Spearman 秩相关；注意相关不等于因果。

## 数据预处理

- 处理缺失值（剔除或合理填补，并说明所用方式）。
- 识别并审慎处理离群点。
- 必要时对数值做标准化。
- 把日期字符串统一转为日期类型，口径统一。
