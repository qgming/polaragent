# 设计配色系统

本文档详细说明如何建立科学的配色系统，避免"安全但平庸"的配色。

## 配色基础理论

### 色彩三要素

**1. 色相（Hue）**
- 定义：色彩的基本外观（红、橙、黄、绿、蓝、紫）
- 色轮位置：0-360度
- 用途：确定色彩基调

**2. 饱和度（Saturation）**
- 定义：色彩的纯度/鲜艳程度
- 范围：0-100%
- 效果：
  - 高饱和度：鲜艳、活力、年轻
  - 低饱和度：柔和、成熟、优雅
  - 0饱和度：灰色

**3. 明度（Lightness/Brightness）**
- 定义：色彩的明暗程度
- 范围：0-100%
- 效果：
  - 高明度：轻盈、清爽
  - 低明度：沉稳、严肃

### 色彩模型

**HSL（推荐用于设计）**
```
H (Hue): 0-360° 色相
S (Saturation): 0-100% 饱和度
L (Lightness): 0-100% 明度

示例：
hsl(220, 90%, 60%) = 鲜艳的蓝色
hsl(220, 30%, 60%) = 柔和的蓝色
hsl(220, 90%, 90%) = 浅蓝色
```

**HEX/RGB（用于实现）**
```
HEX: #RRGGBB (如 #3B82F6)
RGB: rgb(59, 130, 246)

转换工具：
- coolors.co
- colorhexa.com
```

## 反 AI Slop 配色原则

### ❌ 要避免的"AI风格"配色

**1. 企业蓝 + 安全灰**
```
❌ 常见Slop组合：
主色：#007BFF（Bootstrap蓝）
辅助色：#6C757D（中性灰）
背景：#F8F9FA（浅灰）

问题：
- 千篇一律，毫无个性
- 像模板网站
- 用户无法建立品牌记忆
```

**2. 彩虹渐变**
```
❌ 过度使用渐变：
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

问题：
- 2016年流行，现在过时
- 降低文字可读性
- 显得廉价
```

**3. 高饱和度暴力撞色**
```
❌ 不和谐的高饱和度组合：
红色 #FF0000 + 绿色 #00FF00 + 蓝色 #0000FF

问题：
- 视觉刺激过强
- 无法长时间观看
- 不专业
```

### ✅ 有品味的配色策略

**1. 选择有态度的主色**
```
不要选：
❌ #007BFF（Bootstrap蓝，太常见）
❌ #28A745（Bootstrap绿，太常见）

可以选：
✅ #2563EB（深邃的蓝，冷静专业）
✅ #EC4899（粉红，活力创新）
✅ #F59E0B（琥珀色，温暖可靠）
✅ #8B5CF6（紫罗兰，神秘高端）
✅ #10B981（翠绿，自然清新）
```

**2. 降低饱和度**
```
从高饱和度调整到中低饱和度：

❌ hsl(220, 100%, 50%) = 过于鲜艳
✅ hsl(220, 70%, 50%)  = 柔和但有力
✅ hsl(220, 50%, 50%)  = 克制优雅
```

**3. 建立层级系统**
```
不要只有1-2个颜色，要建立完整色阶：

主色蓝色示例：
50:  #EFF6FF (最浅，背景)
100: #DBEAFE
200: #BFDBFE
300: #93C5FD
400: #60A5FA
500: #3B82F6 (主色)
600: #2563EB (深一级)
700: #1D4ED8
800: #1E40AF
900: #1E3A8A (最深，文字)

用途：
- 50-200: 背景、悬停
- 500: 主按钮、链接
- 600-700: 悬停状态
- 800-900: 文字、图标
```

## 配色方案类型

### 1. 单色调（Monochromatic）

**定义：** 一个色相的不同明度和饱和度变化

**优点：**
- 和谐统一
- 容易掌握
- 永不出错

**示例：**
```
主色：蓝色 (220°)
色阶：
hsl(220, 90%, 90%) - 浅蓝背景
hsl(220, 80%, 70%) - 浅蓝元素
hsl(220, 70%, 50%) - 主蓝色
hsl(220, 80%, 35%) - 深蓝按钮
hsl(220, 90%, 20%) - 深蓝文字
```

**适用场景：**
- 专业工具
- B2B产品
- 需要简洁统一的设计

---

### 2. 类比色（Analogous）

**定义：** 色轮上相邻的颜色（通常相隔30-60度）

**优点：**
- 和谐柔和
- 比单色丰富
- 视觉舒适

**示例：**
```
中心色：蓝色 (220°)
类比色：
- 青色 (180°)
- 蓝紫色 (260°)

配色方案：
主色：hsl(220, 70%, 50%) 蓝色
辅色1：hsl(180, 60%, 45%) 青色
辅色2：hsl(260, 60%, 55%) 蓝紫色
```

**适用场景：**
- 自然主题（森林、海洋）
- 柔和氛围
- 艺术类网站

---

### 3. 互补色（Complementary）

**定义：** 色轮上相对的颜色（相隔180度）

**优点：**
- 高对比
- 视觉冲击强
- 引人注目

**示例：**
```
主色：蓝色 (220°)
互补色：橙色 (40°)

配色方案：
主色：hsl(220, 70%, 50%) 蓝色（60-70%面积）
强调色：hsl(40, 90%, 55%) 橙色（10-20%面积）
中性色：灰色（20-30%面积）
```

**注意事项：**
```
❌ 不要：
- 大面积使用互补色（视觉疲劳）
- 高饱和度互补色紧挨（视觉震动）
- 互补色做文字背景（可读性差）

✅ 应该：
- 主色大面积，互补色小面积点缀
- 降低其中一个的饱和度
- 用于强调、按钮、图标
```

**适用场景：**
- 需要突出重点
- 运动、活力类品牌
- Call-to-Action 按钮

---

### 4. 分裂互补（Split-Complementary）

**定义：** 主色 + 互补色两侧的颜色

**优点：**
- 比互补色柔和
- 比类比色丰富
- 平衡感好

**示例：**
```
主色：蓝色 (220°)
互补色：橙色 (40°)
分裂互补：
- 黄橙色 (20°)
- 红橙色 (60°)

配色方案：
主色：hsl(220, 70%, 50%) 蓝色
辅色1：hsl(20, 80%, 60%) 黄橙色
辅色2：hsl(60, 70%, 55%) 红橙色
```

---

### 5. 三角配色（Triadic）

**定义：** 色轮上等距120度的三个颜色

**优点：**
- 活泼丰富
- 平衡和谐
- 视觉兴趣强

**示例：**
```
经典三角：
- 红色 (0°)
- 黄色 (120°)
- 蓝色 (240°)

现代三角：
主色：紫罗兰 (280°)
辅色1：柠檬绿 (40°)
辅色2：珊瑚红 (160°)

配色方案：
主色：hsl(280, 60%, 55%) 紫罗兰（60%）
辅色1：hsl(40, 70%, 60%) 柠檬绿（25%）
辅色2：hsl(160, 65%, 50%) 珊瑚红（15%）
```

**适用场景：**
- 儿童产品
- 创意工作室
- 活力品牌

## 配色检查清单

**设计完成后逐项检查：**

- [ ] **主色有个性**（不是#007BFF蓝或#28A745绿）
- [ ] **色阶完整**（至少5-7个明度变化）
- [ ] **对比度达标**（文字与背景对比度≥4.5:1）
- [ ] **饱和度适中**（不是100%纯色）
- [ ] **语义色清晰**（成功/警告/错误/信息）
- [ ] **中性色系统**（灰色色阶完整）
- [ ] **无障碍友好**（色盲模式下仍可区分）
- [ ] **一致性**（同一状态颜色统一）

---

## 总结

有品味的配色要点：
1. **避免常见组合**：不用Bootstrap默认色
2. **降低饱和度**：从100%降到60-80%
3. **建立系统**：完整的色阶而非单一颜色
4. **克制使用渐变**：能不用就不用
5. **无障碍优先**：对比度达标
6. **测试色盲模式**：确保可区分
