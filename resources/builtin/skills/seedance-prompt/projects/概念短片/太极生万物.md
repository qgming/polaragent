# 太极生万物 — 东方玄幻创世

---

## 版本A：30秒水墨CG创世史诗（双段拼接）

**主题**：太极 → 两仪 → 四象 → 万物 → 归一，东方哲学的视觉化创世演绎
**总段数**：2段
**建议比例**：16:9（2.35:1 宽银幕更佳）
**整体节奏**：混沌酝酿（慢）→ 天地裂变（爆发）→ 万物涌现（渐快）→ 归于太极（慢）

### 全局锁定语句

- 全片保持中国水墨美学与3D国漫CG融合的统一视觉风格，风格不漂移，色调与质感前后一致
- 全片保持一致的水墨流体物理特性，墨液的浓淡、流速、扩散方式前后统一
- 同一场景保持一致环境、光线、背景与空间关系，不突然切换

### 统一负面提示词

- 禁止风格漂移，禁止突然偏色，禁止光线突变，禁止新增任何现代元素，禁止出现文字、字幕、LOGO、水印

## 版本对比与建议

| 对比项 | 版本A（太极创世30秒） | 版本B（挥毫创世15秒） |
|--------|----------------------|----------------------|
| 时长 | 30秒（2段拼接） | 15秒（单段） |
| 叙事方式 | 抽象哲学（太极→万物） | 具象动作（落笔→生长） |
| 视觉高潮 | 太极炸裂+四象爆发（两次高潮） | 墨触纸扩散+2D→3D蜕变 |
| 制作成本 | 较高（两段+迭代） | 较低（单段即可） |
| 模型难度 | 中高（分段衔接+抽象概念理解） | 中（具象动作+风格统一） |
| 传播性 | 史诗感强，适合B站/视频号 | 创意感强，适合全平台 |

### 推荐策略

1. **先试版本B**：单段15秒，积分消耗少，能快速验证水墨CG风格的效果
2. **如果版本B效果好** → 说明模型对水墨风格理解到位，再上版本A的30秒史诗
3. **如果版本B效果一般** → 根据实际问题调整提示词后重试，不急于上30秒

### 迭代方向

- 如果墨液流体效果不够好 → 加强流体动力学描述的具体性，增加"墨液黏度""扩散速度"等物理参数
- 如果风格不够统一 → 把锁定语句放到提示词更前面的位置
- 如果画面太满/太乱 → 删减万物生长阶段的元素数量，少即是多
- 如果3D立体感不够 → 加入更多景深描述（"近景山石清晰远景山峦被墨雾虚化"）
