# 穿越万象 EP02：空间穿越

## 视频提示词

**主题**：一镜到底从地球仰望星空穿越到火星表面
**时长**：15秒  |  **比例**：16:9（建议 2.35:1 宽银幕）

---

### 版本一：星空穿越（硬核科幻）

#### 提示词

```
15秒一镜到底空间穿越，8K超高清+杜比视界HDR，UnrealEngine5渲染，院线级CG画质，全程不切镜头，

0-3秒：Low Angle仰拍，一个穿宇航服的身影孤独站在地球旷野上，银河横贯天际，漫天繁星清晰可见，星光在头盔面罩上倒映，风吹动地面的草丛，环境音只有风声和微弱的呼吸声；

4-7秒：Extreme Crane Up + Zoom In镜头从人物极速上升穿入星空，穿过大气层，大气散射的蓝色光晕在身后逐渐消散，地球缩小为一颗蓝色圆球，太空中星尘粒子如雪花般扑面而来，寂静无声的太空深处远处星云发出幽蓝微光；

8-12秒：Smooth Dolly Forward在星际间穿行，掠过几颗小行星，表面陨石坑纹理清晰可见，远处火星的红色球体逐渐变大填满画面，镜头穿过火星稀薄的大气层，红色尘埃弥漫，地表的奥林匹斯山轮廓在沙尘中若隐若现；

13-15秒：镜头落在火星红色荒漠表面，Slow Pan水平回转，地球作为一颗小小的蓝色亮点悬挂在火星暗橙色天空中，荒凉而壮美，远处沙丘绵延至地平线。

一镜到底全程不切镜头。

光影：地球段银河漫射冷白光+草地月光冷辉（光源层），大气散射柔化星光边缘（光行为层），冷蓝底调（色调层）；火星段太阳直射暖橙光（光源层），红色沙尘漫射柔化远景（光行为层），暗橙红底调+天空深蓝高光（色调层）。禁止：任何文字、字幕、LOGO或水印。
```

#### 首帧建议

> **首帧图片生成提示词**：一个穿白色宇航服的身影站在地球旷野上仰望星空，银河横贯天际，Low Angle仰拍视角，星光在头盔面罩上反射，周围是低矮的草丛被风吹动，冷蓝色调，8K超高清科幻写实CG

#### 设计意图

这版走硬核科幻路线，用光影三层结构做了地球冷蓝→火星暖橙的自然过渡。"地球缩小为蓝色圆球"和"回头看地球变成小亮点"形成首尾呼应，强化空间穿越的震撼感。

---

### 版本二：瞳孔星际（微观切入）

#### 提示词

```
15秒一镜到底空间穿越，8K超高清，ARRI ALEXA摄影机质感+UnrealEngine5渲染CG，全程不切镜头，

0-4秒：Extreme Close-Up极致特写，一只人类眼睛的瞳孔占满画面，瞳孔中倒映着漫天星空，虹膜纹理如同星云的旋涡结构，一滴泪水在眼角折射出星光，浅景深只聚焦瞳孔；

5-8秒：Smooth Zoom In持续推进潜入瞳孔内部，黑色瞳孔扩大为深邃的太空，虹膜纹理过渡为真实的星云旋臂，Dolly Forward在星云中穿行，气体和尘埃被引力拉伸成壮丽的光带，蓝紫色星云发出幽光；

9-13秒：穿过星云，火星的红色球体从星尘中浮现，Smooth Dolly Forward镜头靠近火星表面，穿过稀薄大气层，红色沙暴在地表翻涌，镜头降落在一片被风蚀的红色岩石峡谷中，峡谷壁上的地质层纹理清晰；

13-15秒：镜头在峡谷中Slow Crane Up缓缓升起，升出峡谷，火星地平线上一轮小小的太阳正在落下，天空从暗橙渐变为深蓝，孤寂而辽阔。

一镜到底全程不切镜头。禁止：任何文字、字幕、LOGO或水印。
```

#### 设计意图

从瞳孔潜入宇宙——"眼中有星辰大海"的具象化表达。虹膜纹理→星云旋臂是一个非常自然的视觉过渡（两者本身就形似），比直接从地面飞上太空更有艺术感和哲学意味。

---

### 提示词解析

| 对比项 | 版本一（星空穿越） | 版本二（瞳孔星际） |
|--------|-----------------|-------------------|
| 穿越起点 | 地球旷野（宏观） | 人类瞳孔（微观） |
| 情绪基调 | 壮阔、孤独、史诗感 | 内省、哲学、诗意 |
| 视觉亮点 | 地球缩小+火星着陆 | 虹膜→星云的形态过渡 |
| 技术重点 | 光影三层结构控制色调过渡 | 微观到宏观的尺度连续变换 |

> **迭代建议**：版本一画面信息密度更高、场景跨度更大，对模型要求较高，如果第一次效果不理想，可以尝试减少中间星际穿行的细节，让模型集中精力做好"地球→太空→火星"三个关键阶段的过渡。
