## 一、像素到3D加长（原㉑）

**思路**：第一段建立像素世界 + 镜头推进；第二段材质膨胀、落地 3D，动作不断。

### 段1（≤15 秒，建议 10 秒）

```
9:16竖屏，8-bit像素游戏风格，一个像素小人在红蓝方块平台上奔跑跳跃躲避障碍，镜头从远景平稳推进跟随角色，屏幕边缘有复古扫描线与轻微色差，8-bit芯片音乐感节奏，禁止文字水印
时长约10秒
```

### 段2 — 视频延长（5 秒）

```
将@视频1延长5秒：镜头继续推近到角色面部与身体，像素方块像充气般膨胀变得圆润，色块逐块替换为PBR真实材质——金属反光布料褶皱皮肤质感，最终为UE5级高清3D写实游戏画面，角色跑跳动作与上一段连贯不中断，色调从饱和像素转为自然光影
```

> 若首段只生成 5 秒：可把段1改为 5 秒，段2 延长 10 秒，总 15 秒。

## 六、AI脱口秀（先文本后画面）

### 6.1 工作流

1. **脚本**：用任意大模型写 60~120 秒台本（梗密、有包袱、可过审），分「开场 / 三个梗 / callback 收尾」。
2. **人设**：固定一个虚拟形象（建议 **3D 卡通或动物拟人**），避免真人脸争议。
3. **视频**：每段 ≤15 秒，按台词拆段，用 **视频延长** 串联；或单段 15 秒内快节奏剪辑风（需提示词写清「跳切/伪一镜」）。
4. **声音**：提示词内写台词 +「音色参考@音频1」；后期在剪映替换真人口播也可。

### 6.2 示例脚本骨架（可自行替换梗）

**人设**：穿西装的河狸 3D 卡通，手持话筒，站在小剧场聚光灯下。

| 段落 | 台词示例 | 画面提示（≤15s/段） |
|------|----------|----------------------|
| 开场 | "大家好，我是练习时长两年半的……河狸。" | 河狸整理领结，聚光灯扫过，观众席剪影鼓掌 |
| 梗1 | "AI 说我周报写得像诗——散文诗，因为没人看得懂。" | 河狸摊手，身后屏幕闪过乱码变诗句 |
| 梗2 | "老板让我对齐颗粒度，我对齐了——颗粒度就是对不齐。" | 河狸假装用放大镜对准空气 |
| 收尾 | "谢谢各位，我的尾巴比我的段子还硬。" | 河狸鞠躬，尾巴挡住话筒，笑声音效 |

### 6.3 首段视频提示词（复制即用）

```
9:16，3D卡通喜剧，一只穿迷你西装的河狸站在小剧场舞台中央手持立式话筒，暖色聚光灯，观众席为虚影剪影，河狸整理领结清嗓子，台词："大家好，我是练习时长两年半的河狸。" 音效：轻笑声、远处咳嗽、话筒轻敲，禁止文字与LOGO
时长约12秒
```

**延长示例**：

```
将@视频1延长10秒：河狸单手叉腰另一手比划，吐槽AI写周报像散文诗，身后LED屏快速闪过乱码与诗句切换，观众剪影前仰后合，台词："AI说我周报像诗——散文诗，因为没人看得懂。" 禁止文字
```
