## 一、小人国加长版（20-30秒，多段拼接）

### 工作流

1. 围绕一个主题设计 4-6 个分镜（每个5秒）
2. 每个分镜先生成图片（9:16竖屏）
3. 每张图片分别生成5秒视频
4. 剪映拼接 + 配乐 + 音效

### 主题一：小人国·年夜饭（30秒，6段）

**分镜1 — 备菜**（图片提示词）
```
微观世界微距摄影，俯瞰巨大的砧板，七八个穿围裙的微缩小人用双人锯切割巨大的白萝卜，锯末般的萝卜碎屑飞溅，旁边几个小人用小推车运走切好的食材，厨房暖光，微缩景观风格，高清细腻
```
视频动态：`小人们协力锯切萝卜，碎屑飞溅，推车缓缓移动`

**分镜2 — 包饺子**
```
微观世界微距摄影，一张巨大的饺子皮铺在案板上如同一张白色床单，四个微缩小人站在饺子皮边缘合力将皮对折，两个小人用绳索固定捏褶，一个小人站在馅料山顶指挥，暖黄厨房光，微缩景观风格，高清

视频动态：`小人合力折叠饺子皮，指挥官比划手势，动作自然流畅`

**分镜3 — 炒菜**
```
微观世界微距摄影，巨大的铁锅中油花翻滚冒着热气，三个微缩小人站在锅沿用长铲翻炒食材，火焰从锅底窜出照亮小人的脸，一个小人用消防水管般的调料瓶往锅里挤酱油，烟火气十足，微缩景观风格，高清

**分镜5 — 摆盘**
```
微观世界微距摄影，巨大的白瓷盘上，十几个微缩小人在搬运和摆放各种菜肴——饺子、鱼、年糕、腊肉，有的用小叉车运菜有的在指挥摆放位置，盘子边缘有小人在雕萝卜花做装饰，俯拍视角，暖光，微缩景观风格

**分镜6 — 团圆**
```
微观世界微距摄影，巨大的圆桌上菜品已摆好，十几个微缩小人围坐在用火柴盒做的凳子上举杯碰杯，桌中央一个巨大的火锅冒着热气，远处有微缩的红灯笼和春联，温馨团圆氛围，暖色调，微缩景观风格，高清

## 二、ASMR 视觉系列

> ASMR 的核心不是声音，是"看着就觉得舒服"。即梦目前声音控制有限，所以重点做**视觉ASMR**——画面本身就让人产生满足感和放松感。
>
> **反馈更新（真人向 + 音效 + 台词）**：已单独整理 **[11-用户反馈延展.md §五](11-用户反馈延展.md#五asmr真人向音效台词多类型)**——含性感挑逗、安眠入睡、吃播咀嚼、雨夜情绪等类型；每条带**台词示例 + 音效清单**。平台对**写实真人正脸**敏感，该文档用肩颈/手/唇下/剪影/3D数字人等方式规避，并说明可配合 `@音频1` 做音色或环境音参考。

### 路线A：手作ASMR（微距+过程满足感）

**陶艺拉坯**
```
极致微距特写，双手在陶轮上拉坯，湿润的陶土在手指压力下缓慢成型为碗的形状，指纹压入泥土的纹理清晰可见，泥浆从指缝溢出缓缓流下，陶轮匀速旋转，背景纯黑只有一束侧光，慢镜头，ASMR般的视觉满足感
```

**墨锭研磨**
```
极致微距特写，一块墨锭在砚台上缓缓研磨画圆，墨锭底部与砚台接触处渗出浓黑墨液，墨液沿砚台纹路缓慢流淌汇聚成墨池，研磨的节奏缓慢均匀如同冥想，砚台石材纹理粗粝质感清晰，自然侧光，安静沉浸
```

**金缮修复**
```
极致微距特写，一双手用细毛笔蘸取金色漆液，沿着一只碎裂茶碗的裂纹缓缓描画，金漆填入裂缝形成精美的金色纹路如同血管，每一条裂纹都被金色赋予了新的生命，慢镜头，柔和侧光，修复过程极度治愈
```

### 路线B：自然ASMR（环境+沉浸感）

**雨滴打荷叶**
```
微距特写，雨滴击打碧绿荷叶表面的瞬间，水珠在荷叶疏水表面弹跳滚动如同水银，多颗小水珠汇聚成一颗大水珠后沿叶脉滚落坠入池塘，荷叶微微震颤，慢镜头120帧，自然光，绿色调
```

**蜂蜜倾倒**
```
微距特写，金色蜂蜜从勺子边缘缓缓倾泻而下，蜂蜜拉出一条完美的粘稠细线后落在烤面包表面缓慢摊开，面包的每一个气孔都被蜂蜜缓慢填满，全程极致慢镜头，暖色侧光，背景纯黑，视觉满足感拉满
```

### 路线C：人物ASMR（3D动漫 / 局部真人）

> 即梦平台限制**写实真人正脸**素材。可选：**3D国漫/CG数字人**；或**真人但只拍肩颈、手、唇下至下巴、背影、剪影、侧脸虚焦**。需要更稳的听感时，提示词里写清台词与音效，并可加「音色参考@视频1/@音频1」。

**3D角色ASMR直播间**
```
3D动漫渲染风格，一位精致的二次元女主播坐在高端ASMR设备前，面前是专业的双声道麦克风，她用手指轻轻敲击麦克风表面，指甲与金属网面接触的微距细节清晰，柔和的暖色台灯光，直播间氛围，画面安静温柔
```

**3D角色掏耳**
```
3D国漫CG渲染风格，一位温柔的古风女子面带浅笑轻轻靠近镜头，如同靠近观众的耳朵，她手持一根精致的玉质耳勺缓缓伸向镜头，极致浅景深只聚焦她的手和耳勺，背景虚化为朦胧的中式屏风和烛光，安静私密的氛围
```

### 路线D：真人向氛围 ASMR（台词 + 音效，详见文档11）

以下为方向索引，完整可复制提示词见 **[11-用户反馈延展.md §五](11-用户反馈延展.md#五asmr真人向音效台词多类型)**：

| 类型 | 要点 |
|------|------|
| 性感挑逗 | 肩颈锁骨+手指刮麦，气声台词，低噪底 |
| 安眠入睡 | 羽毛扫耳、揉捏减压球，引导式呼吸台词 |
| 吃播咀嚼 | 俯拍食物，唇齿局部，脆响咀嚼音 |
| 雨夜情绪 | 雨声+翻页+笔尖，可无台或一句收束 |
| 低音剪影播报 | 男性剪影+虚构新闻梗，幽默放松 |

## 三、动作触发场景变化（最强2秒hook）

> 核心逻辑：角色做了一个极其日常的小动作 → 整个世界因此剧变。反差越大越抓人。

### 提示词

**打喷嚏→吹飞一切**
```
一位穿连帽卫衣的年轻人站在秋天的街头，猛地打了一个大喷嚏，喷嚏的气流像飓风一样把周围所有东西都吹飞——树叶路人雨伞报纸全部被吹向远方，只有他一个人站在原地，3D动漫风格，夸张喜剧效果
```

> **加长·奇葩版**：玻璃震碎、货架倒塌、甜甜圈套脖、懵逼看镜头——见 **[11 §四](11-用户反馈延展.md#四打喷嚏奇葩加长原)**（两段拼接 15 秒）。

**击掌→世界定格**
```
两个人在篮球场击掌的瞬间，冲击波从掌心爆发向四周扩散，冲击波经过的地方一切都被冻结定格——篮球悬停在半空、跳跃的观众定格在腾空状态、水花凝固成冰晶，只有这两人还在动，3D动漫渲染，慢镜头
```

**弹指→季节切换**
```
一只手在画面中央弹响手指，弹指的瞬间整个画面的环境从白雪皑皑的冬天瞬间切换为樱花盛开的春天，树枝上的雪瞬间变为粉色花瓣纷飞，地面积雪化为绿草如茵，天空从灰白变为湛蓝，电影质感，慢镜头
```

**吹蒲公英→变星空**
```
微距特写一个人轻轻吹一朵蒲公英，绒毛种子飘散到空中后每一颗种子变成一颗发光的星星，星星越飘越远越飘越高最终填满整个画面变成一片浩瀚星空，从微距到宏观的尺度变化，梦幻色调，慢镜头
```

**拍桌子→画面碎裂**
```
一只手用力拍在桌面上，拍击的瞬间整个画面如同玻璃屏幕一样从拍击点向四周裂开碎片，碎裂的画面碎片脱落后露出背后一个完全不同的世界——从办公室碎裂后露出星空宇宙，碎片在太空中飘浮，科幻风格
```

**翻书页→场景穿越**
```
特写一只手翻开一本旧书的书页，书页翻过的瞬间画面跟随书页转动，翻过后整个场景从现代书房变成了书中描述的古代战场，盔甲骑兵在远处冲锋扬起尘土，书页的边缘仍然可见框住画面如同一扇窗户，史诗古风
```

> **扩展公式**：[日常小动作] + [超自然级别的连锁反应] = hook。动作越小、反应越大、反差越强。

## 四、经典作品改编（附版权规避策略）

### 版权规避原则

1. **不使用任何角色名、作品名、公司名**——提示词中写"一位东方武者"而非"孙悟空"
2. **改编视觉风格而非复制内容**——把"一个场景的氛围"转换为另一种风格
3. **用"致敬""风格参考"代替直接复刻**——描述视觉特征而非具体IP
4. **原创角色+经典场景类型**——自己设计角色放入"类似"的场景
5. **如果要发布到平台**——建议只做风格转换和氛围致敬，避免任何可辨识的IP元素

### 路线A：风格转换（最安全）

把某种影视风格整体转为另一种风格，不涉及具体IP。

**赛博朋克→水墨画**
```
一座赛博朋克风格的雨夜城市全景，霓虹灯光在雨水中倒映，这个画面如同被施了魔法般从城市边缘开始逐渐变为中国水墨画风格，霓虹变为灯笼光晕建筑变为飞檐楼阁雨滴变为墨点，两种风格在画面中并存
```

**像素游戏→3D写实**
```
一个像素风格的8-bit游戏画面，角色在像素方块组成的世界中奔跑跳跃，镜头逐渐推近，像素方块开始膨胀变得圆润细腻，色块填充为真实材质纹理，最终整个画面从像素变为高清3D写实游戏画面，角色动作保持一致
```

> **加长版（两段拼接）**：先纯像素跟拍，再延长段完成材质膨胀落地 3D——见 **[11 §一](11-用户反馈延展.md#一像素到3d加长原)**。

### 路线B：经典场景类型改编（安全）

不复制具体作品，而是改编"场景类型"。

**武侠对峙→萌宠版**
```
两只穿着迷你古装的猫咪在竹林中对峙，竹叶飘落，一只橘猫握着牙签剑摆出起手式，对面的黑猫单爪背后冷酷凝视，风吹动它们的小披风，突然两猫同时弹跳交锋在空中对砍，Q版武侠风格，慢镜头
```

> **加长版**：对峙蓄力 + 空中对砍 + 落地呆萌反差——见 **[11 §二](11-用户反馈延展.md#二武侠猫咪加长原)**。

**太空科幻→蒸汽朋克**
```
一艘飞船的驾驶舱内部，但不是光滑的科幻金属而是布满铆钉齿轮和铜管的蒸汽朋克风格，巨大的机械仪表盘上指针颤动，船长戴着飞行护目镜拉下一个黄铜操纵杆，窗外是星空但星星由齿轮和发条构成，蒸汽喷射
```

### 路线C：名画活化（安全，公共领域）

> 古典名画版权已过期，属于公共领域，可以安全使用。

**清明上河图活起来**
```
清明上河图风格的长卷画面，水墨工笔画中的人物开始有了微妙的动作——河上的船夫摇橹划水，桥上的行人缓步走动，沿街小贩招手揽客，茶楼二楼有人推窗观景，画面保持工笔画的纸质纹理和墨色质感，微风吹动酒旗飘扬
```

> **只是动起来不够看**：要「出事」——虹桥推人落水、醉汉拽塌酒旗、马车受惊滚梨等脑洞与多段蒙太奇，见 **[11 §三](11-用户反馈延展.md#三清明上河图脑洞原)**。

**星月夜（梵高风格）流动起来**
```
一幅梵高星月夜风格的画面，天空中旋涡状的云层和星星开始缓缓旋转流动，金色的月亮散发脉动的光晕，远处小镇的窗户亮起暖光，柏树在夜风中轻轻摆动，所有笔触保持油画肌理质感但都在活动，如同画中世界苏醒
```

**千里江山图化为3D**
```
以千里江山图的青绿色调为基础，平面的山水画开始膨胀生长为3D立体地形，山峰从纸面上拔起，河流开始流淌反光，云雾在山谷中弥散，一群白色仙鹤从画中飞出穿过云层，从2D古画到3D仙境的连续变化，中国风极致美学
```
