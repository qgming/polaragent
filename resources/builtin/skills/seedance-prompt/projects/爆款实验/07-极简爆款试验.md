# 极简爆款试验 — 一个hook，一个瞬间

> 设计原则：每个概念只做一件事，前2秒即高潮，不讲故事，纯视觉冲击。

---

## 01：触碰变黄金（5秒 · 材质魔术）

> 手指触碰玫瑰的瞬间，花瓣从尖端开始迅速结晶为纯金。简单到极致，视觉冲击最直接。

```
5秒极致微距特写，8K超高清，Octane物理渲染，
一根手指轻轻触碰一朵鲜红玫瑰的花瓣尖端，
触碰的瞬间——金色从指尖接触点爆发式蔓延，
花瓣以肉眼可见的速度从鲜红活体转化为纯金属质感的黄金，
金色结晶波沿着花瓣纹脉精确扩散如同电路被点亮，
每一条花瓣纤维都变成精密的金丝浮雕，
慢镜头120帧/秒捕捉红色与金色交界线推进的每一帧，
交界处有微小的金色粒子飞溅如同微型焊花，
最后整朵玫瑰完全化为黄金，金属表面反射出周围环境的扭曲倒影，
一滴露水从金色花瓣上滚落击碎在桌面上发出清脆金属音。
黑色背景，单侧柔和暖光，浅景深只聚焦花朵。
禁止出现文字、字幕、LOGO、水印。
```

**首帧图提示词**：极致微距特写，一根手指即将触碰一朵鲜红玫瑰花瓣尖端，手指距花瓣仅1毫米，花瓣上有一滴露水，纯黑背景，单侧暖光，8K微距摄影，浅景深

---

## 02：咖啡杯中的微缩城市（8秒 · 尺度错位）

> 俯拍看一杯咖啡，咖啡表面是一座正在运转的微缩城市。咖啡的蒸汽就是城市的云。

```
8秒尺度幻觉短片，8K超高清，Octane物理渲染+流体动力学模拟，
Top-Down俯拍一杯冒着热气的咖啡，白色咖啡杯占画面中央，
咖啡表面不是普通液面——而是一座完整的微缩城市俯瞰图，
深褐色的咖啡液面构成城市的地表和道路网络，
奶泡拉花的白色纹路是城市中蜿蜒的河流与高速公路，
极微小的灯光在咖啡表面闪烁如同城市的万家灯火，
微缩的摩天大楼从咖啡液面上隐约凸起形成天际线，
咖啡蒸汽升腾形成城市上空的云层和薄雾，
一根搅拌勺缓缓伸入画面搅动咖啡——
城市的灯光跟随旋涡流动重组但不消失，
如同一场温柔的城市地震后万物重新归位。
俯拍固定机位，温暖侧光，咖啡馆氛围的柔和暖色调。
禁止出现文字、字幕、LOGO、水印。
```

**首帧图提示词**：正俯拍Top-Down视角，一杯冒着热气的咖啡在白色杯中，咖啡表面呈现出微缩城市的俯瞰图——深褐液面是道路，奶泡是河流，极微小灯光闪烁，咖啡蒸汽如云层，温暖侧光，8K超高清

## 03：碎裂重组蝴蝶（6秒 · 物理反常）

> 一个玻璃球坠落摔碎，碎片在半空悬停，然后重组成一只振翅的玻璃蝴蝶飞走。

```
6秒物理魔术短片，8K超高清，流体动力学+破碎物理模拟，工业光魔级VFX，
一个透明玻璃球从画面顶部坠入，
撞击黑色镜面地板的瞬间炸裂成上百块大小不一的玻璃碎片——
慢镜头120帧/秒捕捉碎裂的每一帧，碎片向四面八方飞溅，
撞击点溅起微小的玻璃粉尘如钻石粉末在光线中闪烁，

碎片飞散到最远点的瞬间——突然全部悬停在半空一动不动，
静止0.5秒制造悬念，

然后所有碎片开始逆向飞回中心但不是复原为球体——
碎片在空中旋转重新排列拼合成一只精致的玻璃蝴蝶形态，
最后一块碎片归位的瞬间蝴蝶翅膀开始扇动，
透明玻璃翅膀折射出彩虹色的棱镜光斑，
玻璃蝴蝶振翅飞出画面顶部，留下一串缓缓坠落的玻璃粉尘光点。
纯黑背景，单束顶光直射，镜面地板反射。
禁止出现文字、字幕、LOGO、水印。
```

**首帧图提示词**：纯黑背景中，一个完美的透明玻璃球悬浮在画面上方即将坠落，球体折射出彩虹棱镜光斑，下方是黑色镜面地板映出球体倒影，单束顶光直射，8K超高清

## 04：水面倒影是另一个世界（8秒 · 意外反转）

> 平静湖面的倒影不是现实的镜像，而是一个完全不同的世界（极光星空）。

```
8秒超现实反转短片，8K超高清，ARRI ALEXA摄影机质感，
Static Shot固定机位拍摄一片平静如镜的湖面，
画面上半部是真实的黄昏天空——暖橙色晚霞、几朵散云、远处山脉剪影，
一切看似普通的风光摄影，

但画面下半部的湖面倒影——不是天空的镜像，
倒影中是一片深邃的极光星空：绿紫色北极光在倒影中舞动流淌，
银河横贯倒影中的天幕，无数星辰在水面之下闪烁，
流星划过倒影中的星空在水面留下一道微弱的波纹涟漪，
真实世界的一片落叶飘落到湖面——
叶片接触水面的瞬间在倒影中变成一颗坠落的流星拖着光尾消散，
涟漪从接触点向外扩散同时扰动了倒影中的极光使其波动闪烁。
固定机位，上半部暖橙黄昏光，下半部倒影中冷蓝绿极光光。
禁止出现文字、字幕、LOGO、水印。
```

**首帧图提示词**：固定机位风光摄影，平静如镜的湖面将画面一分为二，上半部是暖橙色黄昏天空和远山剪影，下半部湖面倒影中不是天空的镜像而是深邃的绿紫色极光星空和银河，上暖下冷的极致对比，8K超高清，ARRI摄影机质感

## 05：墨入水绽放（5秒 · 极致满足感）

> 一滴墨水落入清水，在水中绽放成一朵完美的牡丹花形态。纯粹的流体之美。

```
5秒流体艺术短片，8K超高清，Octane物理渲染+流体动力学模拟，
纯白色背景中一个透明玻璃容器装满清水，
一滴浓黑墨水从画面上方坠入水中——
入水瞬间墨水炸开成蘑菇云状的黑色烟团，
慢镜头120帧/秒极致慢放，
墨水在水中的扩散不是随机的混沌——
而是精确地绽放成一朵盛开的牡丹花形态，
每一片花瓣都是墨水的流体卷曲形成的涡旋结构，
花蕊处墨水最浓呈深黑色向花瓣边缘渐变为淡灰如同水墨画的浓淡渲染，
花瓣持续舒展到最大绽放的瞬间画面定格——
如同一幅悬浮在水中的完美水墨牡丹。
纯白背景，均匀柔和散射光，无阴影，水中漂浮的墨粒如花粉般细腻。
禁止出现文字、字幕、LOGO、水印。
```

**首帧图提示词**：纯白背景，一个透明玻璃容器装满清水，一滴浓黑墨水刚刚触及水面的瞬间定格，墨水刚开始在水中形成蘑菇云形态的初始绽放，慢镜头定格感，均匀柔光无阴影，8K超高清

## 试验策略

| 编号 | 概念 | 时长 | 核心hook | 推荐首试 |
|------|------|------|----------|----------|
| 01 | 触碰变黄金 | 5秒 | 材质魔术 | ★★★ |
| 02 | 咖啡微缩城市 | 8秒 | 尺度错位 | ★★★ |
| 03 | 碎裂重组蝴蝶 | 6秒 | 物理反常 | ★★☆ |
| 04 | 水面倒影异世界 | 8秒 | 意外反转 | ★★★ |
| 05 | 墨入水绽放 | 5秒 | 极致满足感 | ★★★ |

**建议**：01和05最简单直接，先试这两个验证效果。都是单一主体+单一动作+微距/特写，给模型的指令最清晰。

**关键区别**：这批提示词和之前的最大不同——
- 时长从15秒压到5-8秒，模型不需要管理长时间的画面变化
- 没有叙事，没有多段分镜，就一个动作从头到尾
- 每个概念用一句话就能说清："触碰变黄金""咖啡里有城市""碎了变蝴蝶"
- 先生成首帧图，用首帧图锚定画面起点，比纯文本可控得多
