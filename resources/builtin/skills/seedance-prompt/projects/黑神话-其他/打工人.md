# 黑神话·打工人

> **版本**：v2.0 · 2026-05-01
> **设计思路**：对标 `黑神话系列.md` 潘金莲原案的写法 —— ≤600 字、朴素描述、信任 AI。
> **类型**：单品爆款（非系列）· 当代职业 × 3A 反差梗
> **结构**：1 张主菜单图 + 1 张实机 UI 图 + 1 段 15 秒 Seedance 视频

---

## 💡 写作原则（未来迭代时记得）

- 不要超过 500 字，更不要追加补丁
- UI 用"左上角：..."这种朴素描述，不用像素数值约束
- 用"对标黑神话悟空/潘金蓮"做风格锚定，胜过堆砌"Octane/工业光魔"
- 轻度磨损感：皱衬衫、咖啡渍、蔫盆栽、汗渍、磨损工牌 —— 不是废墟
- 信任 AI 的视觉直觉，给它"感觉"不是"规范"
- 打工人露脸风险提示：眼神看向窗外，禁对视镜头

## 📋 生图抽卡策略

```
Step 1：主菜单图生成 3-5 次择优
       （重点看：LOGO 江湖体飞白 + 主角眼神不对视 + UI 克制）
Step 2：实机图生成 3-5 次择优
       （重点看：UI 完整 + 背影姿态 + 办公室轻度磨损感）
Step 3：两张图作为首尾帧喂给 Seedance 2.0
```

**挑图的三条心法：**

1. **看 LOGO**：「打工人」三字有没有飞白/干笔/粗细反差？带不带疲惫倔强气质？
2. **看主角**：眼神是不是看向窗外？有没有"活着的打工人"感（非游戏建模）？
3. **看氛围**：办公室够不够暗（背景压暗）？有没有咖啡渍、磨损、汗渍等生活痕迹？

## ⚠️ 打工人特殊风险提示

因为打工人是**主角露脸**场景（外卖员有头盔遮挡），生图时要特别注意：

- **眼神对视镜头** → 最常见翻车点。重生成直到眼神看向窗外
- **面部过于写实真人感** → 要求"CG 游戏立绘质感"而非"真人照片感"
- **画面过亮** → 日光灯通杀办公室 → 重生成直到背景压暗
