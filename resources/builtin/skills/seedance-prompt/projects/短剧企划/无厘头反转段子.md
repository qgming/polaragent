## 使用指南

### 快速上手
1. 选一个概念 → 复制「首帧图生成提示词」到即梦生成首帧图
2. 选满意的首帧图 → 在视频提示词前加 `@图片1为首帧，` 再粘贴视频提示词
3. 生成视频 → 挑选满意版本

### 调整建议
- **节奏太快？** → 减少分镜数量，把4段改为3段
- **风格不对？** → 替换开头的风格关键词（如"3D卡通"改"2D手绘"）
- **反转力度不够？** → 加大前后反差，让"正常"部分更正常，"荒诞"部分更荒诞
- **想加对白？** → 在对应时间段加入 `角色（情绪）："台词"` 格式的对白

### 质量自检
- [x] 所有提示词为中文
- [x] 无写实真人面部素材
- [x] 时间戳覆盖完整15秒
- [x] 每段包含画面+镜头+音效
- [x] 10个概念风格各异（动物拟人×3、日常生活反转×2、赛博荒诞×2、人际关系反转×1、民俗混搭×1、古风荒诞×1）
- [x] 均附带首帧图生成提示词
