## 3. 抖音味专属：v3 草案需补的概念

抖音爆款 ≠ 普通凝视模式视频。**抖音味需要在凝视模式基础上叠加 4 个特征**：

### 3.1 前 2 秒"立刻能识别的钩子"
不能是"缓缓推进"，必须是**第 0.5 秒就有视觉/动作上的辨识点**：
- 主角第一个动作要"明显"
- 画面构图要"立刻把美少女放在视觉重心"
- BGM 在第 0 秒就响起（不可渐入）

### 3.2 节拍点意识（即使没有真的卡 BGM）
- 主角的 4 个自然小动作要呈现"节奏感"——不是均匀分布，而是 **2s+1.5s+1.5s+1s** 的递快节奏
- 最后一个动作落在第 5-6 秒之间，作为"想看第二遍的钩子"

### 3.3 居家暖光氛围 ≠ 偷拍业余感
本 case 与 case-07 地铁偷拍的关键区别：
- 地铁是**偷拍**——画质要业余、握持抖、构图歪
- 厨房抖音爆款是**自拍 vlog**——画质要"看起来很自然但其实精心摆放"，**比偷拍干净一档**
- 4 层画质退化中第①层平台压缩保留，但第③层偷拍特征改为"自拍轻微痕迹"

### 3.4 主角"知道镜头在但装作没看见"
- 凝视模式纯版本：主角完全不知情
- 抖音爆款版本：**主角知道镜头在但表演成"自然状态"**——这是抖音爆款的隐藏特征
- 体现方式：4 个动作中最后一个动作是"轻微整理头发/微笑/看一眼镜头方向"——给观众"她在为镜头表演但很自然"的感觉

## 5. 第 2 轮：完整 Seedance Prompt（候选 A · 凝视模式 + 抖音味 v3.1）

```

她是这条视频的核心主角是抖音"早晨vlog"标签下最让观众想停留2秒的那个女孩。

主角必须做厨房早晨自然会做的事（6秒内必须明显发生4个动作呈递快节奏）：
- 0—2秒：左手从橱柜拿出陶瓷马克杯放到流理台上动作明显能看到拿杯放杯过程，
  右手同时把刚煮好的咖啡壶端起来（不是只做出拿杯样子是真的有完整拿杯放杯+端壶动作）；
- 2—3.5秒：把咖啡从壶里倒进马克杯，倒的时候头部微低看着倒入的咖啡液面注视2秒，

窗户从右侧透入早晨柔和日光（注意：日光不刺眼是温柔的早晨光线），
顶灯关着仅靠日光（**禁止厨房出现其他人入画**仅允许主角一人独处时光）。
```

## 8. 与 v3 草案的契合度 + 反哺

本 prompt 是**凝视模式叠加抖音味的 v3.1 实验**，**为 v3 草案补充了 3 个新概念**：

### 反哺 v3：抖音味强化模板（路径 C 子模式 v3.1 提案）

凝视模式在 v3 草案下已经定型，但**抖音爆款场景需要进一步细化**。建议 v3 草案下次更新时增加这条：

```markdown
### 凝视模式·抖音味强化子模式（v3.1 实验）

适用场景：
- 用户说 "抖音很火"、"抖音爆款"、"小红书爆款"、"vlog 风格"
- 主角是 KOL/网红/美少女向，半身入镜为主
- 居家场景或日常生活场景（厨房/卧室/咖啡馆/通勤等）

四个特征：
1. **前 2 秒钩子**：第 0.5 秒就有视觉/动作辨识点 + BGM 立刻响起不可渐入
2. **递快节拍**：4 个动作的时长分布是 2s→1.5s→1.5s→1s 而不是均匀分布
3. **自拍干净 ≠ 偷拍业余**：4 层画质退化的第③层"现场瑕疵"换成"自拍轻微痕迹"——构图轻微歪、轻微稳定抖动、轻微反光，但不要握持狂抖、不要构图严重歪
4. **主角知道镜头但装作没看见**：最后一个动作必须有"眼神飘向镜头方向 0.3-0.5 秒 + 微笑暗示"——这是抖音爆款的隐藏特征，与纯凝视模式的"完全不知情"区分开

独处场景的"群演替代"：
独处场景没有群演，凝视模式的"段内具体写动作"原则改为"环境细节互动"——
- 灶台水壶余烟/流理台绿植随空气微动/前景热气朦胧光晕/远处小细节锚点
- 这些环境细节充当"凝视模式的叙事填充物"，强化"独处的诗意"
```

### 反哺 v3：合规风险新场景

| 场景类型 | 合规风险 | 处理 |
|---------|---------|------|
| 半身美少女居家 vlog | 中度（30-50%） | 禁止低胸/抹胸/吊带、禁止下蹲找东西的角度、保持宽松居家服 |
| 厨房半身入镜 | 中度 | 同上 + 禁止过于贴近镜头特写 |

下次实测后可验证：
- "自拍干净版" 4 层退化模型能否区分于"偷拍业余版"
