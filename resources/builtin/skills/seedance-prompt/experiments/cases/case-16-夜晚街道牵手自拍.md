## 6. 真实感修订（2026-06-03 主理人反馈后 · v2 真实感版）

> **主理人实跑 A/B/C 三版后的反馈**（三点共性问题）：
> 1. **灯光打得太亮**——整体是夜晚，人身上受光不该这么亮（出片像影棚补光，不像夜路）
> 2. **太高清**——不像手机拍出来的（缺手机弱光实拍的粗糙感）
> 3. **动作/神态/走动不自然**——表演感重，需要更松弛生活化

**根因**：原 v3.1 走极简版，**没写画质退化模板 + 没约束夜景光感**，Seedance 默认就给"干净高清 + 均匀打亮"。修订 = 补三个核心模块。

### 6.1 三个核心修订模块（可直接替换进任意一版 prompt）

**① 夜景低照度光感（治"灯太亮"）**：
```
夜晚低照度真实光感：人物只被路灯橘黄余光从侧面/背后微弱照到，脸和身体大部分处在柔和暗部里，
明暗对比明显、暗部不补光，严禁影棚式均匀打亮、严禁人物比夜晚环境更亮、严禁面部高光过曝，
呈现夜晚手机弱光下偏欠曝的真实感。
```

**② 手机夜拍画质退化（治"太高清"）**：
```
画质：夜晚手机随手拍质感——弱光高ISO彩色噪点与暗部色块、夜间自动曝光与白平衡轻微漂移偏暖橘、
握持与走动带来的轻微抖动和动态模糊、对焦呼吸、手机广角轻微畸变与暗角。
禁用4K高清禁用电影级稳定禁用影棚布光禁用调色精致禁用画面锐利，
要明显的手机弱光实拍颗粒粗糙感，不要高清不要干净。
```

**③ 自然动作神态声明（治"不自然"）**：
```
动作与神态自然生活化、去表演感：走路步频与重心自然、回头是随意一瞥而非刻意摆拍定格、
微笑松弛不僵硬、手部与头发动作随走动自然发生，避免夸张造型和过度对镜头表演。
```
