## 1. v1 反馈分析

**主理人反馈**：v1 角色打闹程度不够热烈、反应不够强烈，整体不够"皮"。

**v1 问题诊断**：
- 3 个穿越互动的动作和反应都偏"克制"（甩眼镜/敲肩膀/吐舌头）
- 其他 6 位角色全程是静态背景"等级数字闪烁"——浪费了画面
- 缺少"被打闹波及→连锁反应"的群体氛围

**v2 修复方向**：
1. **3 个穿越互动的动作和反应都升级 1 个等级**（更有挑衅感+更夸张）
2. **被穿越的角色反应更强烈**（推回去/挣扎/笑到躲）
3. **全员加入"看戏反应"**（其他 6 位也吃瓜、笑、翻白眼）
4. **加镜头小晃动**——画面跟着打闹有轻微震动，强化"活过来"感

## 2. 完整 Seedance Prompt（v2·再皮一点）

```
8秒赛博朋克角色卡集体打闹活过来了，1:1 方形。

参考@图片1 9 宫格角色卡的整体布局和每个角色的外貌服装等级图标作为视觉参考，**保留 9 宫格布局/圆形角色框/职业图标/名字等级/星级**这些游戏 UI 元素不变。视频在静态卡牌墙基础上让 3 组角色突破自己的圆形边框做**热烈打闹互动**——剩下 6 位也都活过来做"看戏反应"不做静态背景。

0-1.5秒：**Glitch 偷袭 Vex** ——中上 Glitch（绿头发戴眼镜女）一把摘下自己的绿色眼镜手伸出圆形边框使坏地朝右上 Vex（粉头发女）扔过去眼镜挂到 Vex 鼻梁上，**Vex 瞪大眼睛伸手推眼镜回去同时朝 Glitch 翻了个超大白眼**，Glitch 在自己格子里大笑捂嘴。同步看戏反应：左上 Riptide 转头哈哈笑、左中 Lifeline 推眼镜看戏。

1.5-3.5秒：**Crusher 揉 Echo 头发** ——中间 Crusher（黄色男）一只手臂突破圆形边框朝右伸过去**一把抓住 Echo（白头发男）的脖子用力揉他的头发把头发揉得乱七八糟**，**Echo 挣扎着假装挥拳捶 Crusher 一下又笑又骂**做出"吃我一拳"的搞笑姿势。同步看戏反应：左中 Lifeline 笑到肩膀抖、右下 Sledge 翘起大拇指起哄。

3.5-6秒：**Onyx 偷袭 Lifeline 自拍** ——左下 Onyx（粉色女）整个上半身翻进左中 Lifeline（紫色女戴眼镜）的格子里**贴着 Lifeline 的脸嘟嘴比 V 做自拍鬼脸+吐舌头+眯眼**强行抢镜，**Lifeline 受惊大笑双手捂脸躲到格子角落**还在格子里小幅推 Onyx 回去。同步看戏反应：右下 Nova 笑到掉眼泪擦泪、Vex（已戴 Glitch 眼镜）摆出"我也要拍"的姿势凑过来。

6-8秒：**全员混战收束** ——9 个格子里所有 9 个角色同时做出活泼夸张反应（笑/翻白眼/挥手/比 V/捂嘴），整个 9 宫格变成一个吵闹的合影，最后 0.5 秒画面凝固在所有人朝镜头扮鬼脸的状态作为高潮收束。

镜头单一全景定格但**画面随打闹节奏轻微抖动晃动**（不是稳定机位，模拟"活过来的卡牌墙"震动感），霓虹光晕和粒子效果在 9 个格子里持续呼吸式明灭，电子鼓点+游戏 UI 提示音效+角色们的笑声/惊呼声/"哈哈哈/喂！/嘿！"等环境音贯穿。

禁止任何额外文字字幕水印（保留角色卡内的名字等级图标），禁止角色面孔过度精修保持插画风格。
```

## 3. 设计意图：v2 三大升级

### 升级 1：3 组互动动作和反应都升 1 级

| | v1（克制）| **v2（再皮一点）**|
|---|------|------|
| Glitch + Vex | 摘眼镜甩进 Vex 格子，Vex 露出疑惑笑容 | **使坏地扔眼镜挂鼻梁 + Vex 翻超大白眼推回去 + Glitch 大笑捂嘴** |
| Crusher + Echo | 拳头敲一下 Echo 肩膀，Echo "什么鬼"表情 | **一把抓 Echo 脖子用力揉头发揉乱 + Echo 挣扎假装挥拳还嘴** |
| Onyx + Lifeline | 钻进 Lifeline 格子吐舌头，Lifeline 受惊往后缩 | **贴脸嘟嘴比 V 自拍+吐舌+眯眼抢镜 + Lifeline 大笑双手捂脸躲角落+小幅推 Onyx 回去** |

### 升级 2：全员看戏反应（不做静态背景）

每段 2 秒互动都给周围至少 2 位角色一个"看戏反应"：
- 第 1 段：Riptide 哈哈笑 + Lifeline 推眼镜看戏
- 第 2 段：Lifeline 笑到肩膀抖 + Sledge 翘大拇指起哄
- 第 3 段：Nova 笑到掉眼泪擦泪 + Vex 凑过来"我也要拍"

→ **9 个角色全员有戏不浪费**，整体画面信息密度高 N 倍。

### 升级 3：第 6-8 秒混战收束

- v1 是"画面凝固在 Onyx 吐舌头瞬间"——只有 1 个焦点
- v2 是 **"9 个格子全员朝镜头扮鬼脸做合影"** —— 9 个焦点同时爆发，更适合爆款收束

### 升级 4：镜头小晃动 + 笑声起哄音效

- v1 是"镜头单一全景定格"——稳定但缺乏临场感
- v2 是 **"画面随打闹节奏轻微抖动晃动"**（模拟"活过来"震动）+ **角色们的笑声/惊呼声/哈哈哈/喂/嘿** 环境音 → 立体的"卡牌墙活过来集体打闹"

## 4. 三层动作分级落地（v5 方法论 ✓）

| 层级 | 落地内容 |
|------|---------|
| 第 1 层 持续微动 | 9 位等级数字呼吸式明灭 + 霓虹光晕呼吸式明灭 + 粒子效果——全程不停 |
| 第 2 层 被聚焦回应 | 6 位看戏反应（笑/翻白眼/挥手/比 V/捂嘴/掉眼泪/起哄）+ 3 组主互动 |
| 第 3 层 大动作 | ✅ 这里**允许大动作**——因为路径 D 9 宫格本质是 UI 卡牌不是真实物理空间，"活过来打闹"就是要打破物理 |

**关键差异**：路径 D 的 9 宫格 ≠ 路径 C 的多人室内场景。9 宫格是 UI 卡牌墙，"穿越格子"本身就是非物理动作，所以**不适用"防物理崩坏 3 条铁律"**——这里就是要让动作夸张挑衅有戏。
