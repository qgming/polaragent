## 3. 抖音味专属：v3.1 四特征落地（沿用 case-08）

### 3.1 前 2 秒"立刻能识别的钩子"
- 第 0 秒：暖光门廊光打在主角侧脸（已是首帧视觉钩子）+ BGM 立刻响起
- 第 0-1 秒：主角眼神已经在"看向画面右侧外"做出小笑——画面立刻有"她在和谁说话"的悬念感

### 3.2 节拍点意识（8s 版的递快节奏）
- 8s 内 4 个动作分布：**2s + 2s + 2s + 2s** 较均匀（vs case-08 的 6s 递快 2+1.5+1.5+1）
- 8s 容纳"侧目看同伴 → 转回看镜头 → 抬手撩头发 → 嘴角微笑端起咖啡/凑近镜头"的完整节奏

### 3.3 自拍干净 ≠ 偷拍业余
- 沿用 case-08 的"自拍轻微痕迹"：构图轻微歪、轻微稳定抖动、自拍手臂左下隐约入画
- 与 case-07 地铁偷拍区分：画质要"看似随手但其实精心摆放"

### 3.4 主角"知道镜头在但装作没看见"
- 8s 版的隐藏特征：第 5-6 秒"转回镜头眼神飘到镜头方向 0.5 秒+嘴角微扬"——抖音爆款标志性瞬间
- 第 7-8 秒可以选择"端起手中物品/嘴角加深微笑/微微歪头"作为收束

## 6. 设计意图（v3 八条原则 + 抖音味 4 特征对照）

| v3 原则 | 本 prompt 落地 |
|---------|---------------|
| ① 图作框架参考 | "参考@图片1...但不强制卡任何一帧"标准句式 |
| ② 凝视 3-4 自然小动作 | 4 个动作完整分布（侧目→转回→撩发→微笑歪头），8s 均匀节奏 2s+2s+2s+2s |
| ③ 日常常识审计 | "远处路人面孔必须虚化模糊不可识别" + "雨后街景细节自然演化" |
| ④ 4 层画质退化 | **自拍干净版**：抖音压缩→手机夜景暖光→自拍轻微痕迹（左下角自拍手臂袖口）→广角变形 |
| ⑤ 群演段内具体写 | 暖色灯笼闪烁/远处路人虚化剪影/偶尔车头灯划过/雨后湿气光晕——独处场景用环境细节代替群演 |
| ⑥ 主角行为正向+反向否定 | 4 个动作每个都加反向否定锁定（不是面无表情看远处/不是机械转头/不是夸张甩头/不是夸张大笑）|
| ⑦ 镜头运动锁死 | Very Slow Push In + 否定句穷举 + 自拍手臂高位视角声明 |
| ⑧ 跨人物互动 | N/A，主角独处（同伴在画外）。改用"画外互动事件"——主角与画外同伴的眼神交流构成"有人陪伴的独处氛围"|

| 抖音味 4 特征 | 本 prompt 落地 |
|--------------|---------------|
| 前 2 秒钩子 | 第 0-2 秒侧目看同伴方向小笑——立刻有"她在和谁说话"的悬念感 |
| 节拍点意识 | 4 个动作 2s+2s+2s+2s 均匀治愈节奏（8s 版比 6s 略放缓） |
| 自拍干净 ≠ 偷拍业余 | 第③层"自拍轻微痕迹"明确"左下角自拍手臂袖口" |
| 主角知道镜头但装作没看见 | 第 4-6 秒"转回镜头眼神锁定但不死板" + 第 6-8 秒"嘴角微扬+眼神短暂飘向同伴方向"——抖音爆款隐藏特征 |

**前 2 秒生死线**：第 0 秒侧目看同伴方向 + 嘴角小笑已显示 + BGM 同步响起（不可渐入）。

**核心 hook**：第 6-8 秒"嘴角微微上扬+眼神短暂飘向画外同伴方向"——抖音爆款的"她在等谁/和谁一起出门"的悬念感。

**画质真实感**：抖音/小红书 vlog 自拍的"治愈的随性感"——不是偷拍那种业余手抖，是精心摆放但看起来不刻意。

**二刷钩子**：观众看完会想"她在等的同伴是谁/为什么笑得这么好看"——情绪治愈系的"想再看一遍"钩子。

**传播预期**：抖音/小红书爆款标签 "韩系夜游" / "夜游 vlog" / "等同伴的瞬间" / "城市夜晚小确幸" / "韩剧女主角既视感"。

## 7. 微调建议

如果想调整：
- **想要更纯凝视模式不要"对镜头微笑"**：把第 6-8 秒改成"眼神继续看右侧同伴方向+轻微闭眼笑"，去掉抖音味特征 4
- **想要更短 6s 版**：去掉"撩发"动作，3 个动作完成（侧目→转回→微笑歪头）
- **想要更长 10s 版**：加一个"举起手中咖啡杯/手机靠近嘴边"的动作放在第 6-8 秒之间，最终微笑收束放到 8-10 秒
- **改成 vlog 视角不是自拍**：去掉"自拍手臂袖口痕迹"+ 镜头机位改为同伴帮拍的略低角度
- **想去掉合规风险**：保留"远处路人面孔虚化"声明 + 保持黑色高领外套包裹严实即可

## 8. 与 case-08 的对比 + 方法论沉淀

| 维度 | case-08（厨房 6s） | case-09（韩系夜游 8s） |
|------|-------------------|----------------------|
| 场景 | 室内独处 vlog（厨房煮咖啡） | 室外街边 vlog（夜游等同伴） |
| 节奏 | 6s 递快 2+1.5+1.5+1（前 2 秒强 hook） | 8s 均匀治愈 2+2+2+2（前 2 秒已有悬念感作 hook）|
| 群演替代 | 环境细节锚点（咖啡热气/绿植/小猫公仔） | 环境细节锚点（暖色门廊灯/路人虚化/车头灯/雨后湿气）|
| 抖音味第④特征 | 第 5-6 秒"飘镜头 0.5 秒微笑暗示" | 第 4-6 秒转回镜头锁定 + 第 6-8 秒"嘴角微扬+飘向同伴方向"|
| 画质退化第③层 | 居家自拍痕迹（橱柜玻璃反光） | 夜游自拍痕迹（自拍手臂袖口入画+湿润路面反光）|

**v3.1 抖音味强化模板验证**：本 case 进一步验证 case-08 提出的"凝视模式·抖音味强化"子模式可迁移到不同场景（室内→室外、独处→画外同伴互动），4 特征均成立。

**新增方法论建议**：8s 版的"4 个动作均匀分布"vs 6s 版的"递快节奏"，可作为凝视模式·抖音味强化的两套节奏模板：
- **6s 递快**：适合需要强 hook 抓住前 2 秒的场景
- **8s 均匀**：适合本身画面已有视觉悬念的场景（如"等同伴"、"窗边看风景"）
