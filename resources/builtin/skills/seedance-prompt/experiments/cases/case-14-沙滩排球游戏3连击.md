## 2. 完整 Seedance Prompt（v2·缩短开场 + 双向互动 + 风格连贯）

```
15秒沙滩排球手游一局完整玩家体验串联，16:9 横屏。

参考@图片1@图片2 两张游戏截图作为完整视觉风格参考（包括日光沙滩场景+游戏 CG 渲染感+UI 元素布局），**@图片3 仅作为最后一段鱼跃救球的动作姿态参考，不迁移图3 的黄昏色调和心形粒子特效**——整段视频统一为日光沙滩明亮基调风格连贯。**完整保留游戏 UI 元素**（小地图/任务栏/按键提示/血条/技能图标/比分/对话框）让画面看起来就是真实在玩这款游戏，UI 不要消失也不要变成水印。

0-3秒：@图片1 场景。沙滩日光下两位女角色出现在玩家面前——**Yuna（粉短裤）抬手朝玩家挥手并跑动两步走近镜头喊"快来！"**，**Mina（蓝运动服）笑着跟过来抓住玩家手腕（玩家视角看到 Mina 的手抓住自己手腕）轻轻往球场方向拉**，对话框"You in? Let's play a round!" 弹出，玩家视角第 1 个选项 "Sounds good." 高亮 0.3 秒后快速确认。

3-8秒：@图片2 场景硬切。比赛激烈进行，玩家视角第三人称跟随队友 Yuna 完成扣球瞬间，Mina 在另一侧场地预备接球，比分 Team A 11 vs Team B 8 显示，Match Point 提示出现，技能图标闪烁，沙地激起细沙粒，对方球员防守跳起。

8-15秒：场景延续日光沙滩明亮基调（**不切黄昏不加心形粒子**），高潮鱼跃救球瞬间——**Yuna 和 Mina 同时奋力鱼跃救球的极致动作姿态参考@图片3 的姿态**（仅参考动作不参考色调），背景仍是日光沙滩+蓝色海平面，画面动感线条+沙粒飞溅特效，连击数和角色卡 UI 高亮显示，最后 1 秒定格在球被精准救起的高光瞬间。

镜头模拟手游玩家第一/第三人称切换（如同手持手机玩游戏），整段日光沙滩明亮基调时间感统一（不做日光→黄昏过渡）。

游戏 BGM 节奏感强烈的电子运动配乐贯穿，加叠击球声+沙地脚步声+得分提示音效+Yuna/Mina 的女声呼喊（"快来！""Nice!""Got it!"）。

禁止任何额外文字字幕水印（仅保留游戏内 UI 文字），禁止角色面孔细节过度精修保持游戏 CG 渲染感。
```

## 3. 设计意图：v2 三大修复（按主理人反馈）

### 修复 1：开场提速 + 双向互动

| | v1 错误 | **v2 修复**|
|---|------|------|
| 开场时长 | 0-5 秒（占 1/3 时长）| **0-3 秒**（占 1/5 时长）|
| 互动方向 | 玩家"缓缓注视她们"（单向凝视）| **Yuna 招手喊"快来！" + Mina 抓玩家手腕往球场拉**（双向互动）|
| 选项节奏 | "高亮闪烁后确认"（慢）| **"高亮 0.3 秒后快速确认"**（快） |
| 心动 hook | 选项闪烁 | **Mina 抓手腕 + Yuna 喊"快来！"**——直观可爱的"游戏开始"信号 |

### 修复 2：第三段保持风格连贯

| | v1 错误 | **v2 修复**|
|---|------|------|
| 整体色调 | 日光（前 5s）→ 突切黄昏（后 5s）| **全程日光沙滩明亮基调统一**（无时间过渡）|
| 粒子特效 | 心形粒子（图 3 风格）| **沙粒飞溅特效**（与日光沙滩自洽）|
| @图片3 用途 | 整体风格迁移 | **仅参考"鱼跃救球的动作姿态"**，不参考色调和粒子 |

### 修复 3：女声呼喊增强游戏沉浸感

加入 Yuna/Mina 的女声呼喊（"快来！""Nice!""Got it!"）—— 强化"和女角色一起玩游戏"的互动感，弥补 v1 玩家全程沉默的"枯燥感"。

### 时间分配重新

| 阶段 | v1 | **v2** |
|------|-----|------|
| 邀请互动 | 5 秒 | **3 秒**（缩短，节奏紧凑） |
| 比赛进行 | 5 秒 | **5 秒** |
| 鱼跃救球高潮 | 5 秒 | **7 秒**（延长，高光值得展开） |
