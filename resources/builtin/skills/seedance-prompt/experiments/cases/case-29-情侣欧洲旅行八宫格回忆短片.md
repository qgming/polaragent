## 4. 设计意图

- **情绪母题黏合剂**："温柔怀旧的情侣夏日旅行回忆" + "仿佛记忆逐一复活" → 8 个差异场景被统一为一段共同记忆，硬切违和转为回忆跳跃感。
- **语义转场流**：以…开场→流转到→切到→转入→移向→接着→随后→最后收尾，不卡死秒数，模型自控节奏。
- **首尾呼应**：开头"记忆复活" ↔ 结尾"记忆般缓缓淡出"，情侣合照收尾 = 情感闭环 + 二刷钩子。
- **视觉收敛**：暖金 + 胶片颗粒 + 浅景深虚化统一调色 + 段间交叉溶解，弥合 8 格色温差异。
- **前 2 秒生死线**：开场即大笑 + 牛角包递镜，鲜活情绪抓人，不用"缓缓推进"。

## 7. v2 修订（2026-06-03 主理人反馈后）

> **主理人看 v1 成片后的 4 点反馈**：
> 1. **尺寸**：原图每个宫格本身是竖屏构图 → 成片应走 **9:16 竖屏**（v1 跟随横图跑成横屏）
> 2. **内容塞太满**：15s 装 8 格太挤 → **只取第 1、4、5、6、8 格**（5 格），其余忽略
> 3. **切换太像相册**：v1 的交叉溶解（渐隐渐现）太像电子相册 → 改 **干脆硬切**
> 4. **BGM 太平淡**：太安静 → 改 **活泼运动、有节奏感**的音乐

**抽取的 5 格映射**：
| 新段 | 原格 | 场景 |
|------|------|------|
| 1 | 上1 | 清晨咖啡馆·大笑递牛角包 |
| 2 | 上4 | 室内镜面·白衬衫自拍 |
| 3 | 下1 | 酒店窗边俯拍·两人的脚+泳池 |
| 4 | 下2 | 夜晚街市·漫步回眸 |
| 5 | 下4 | 情侣脸贴脸合照（收尾）|

**关键改动**：D-3 默认用"交叉溶解 + 治愈 BGM"营造回忆感，但本次主理人要"轻快卡点"风格 → 改为 **硬切 + 活泼 BGM + 卡拍**（更接近抖音旅行混剪而非文艺回忆 MV）。这是 D-3 的一个**子变体：轻快混剪版**（vs 原文艺溶解版）。
