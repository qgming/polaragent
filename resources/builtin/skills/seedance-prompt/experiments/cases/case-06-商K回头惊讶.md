## 3. 范式选择 + Hook 候选

**Hook 范式**：反应模式（高戏剧瞬间，按子节拍展开）

### 候选 A：回眸·瞳孔骤缩版（错愕 → 错愕峰值 → 表情微落定）
- Hook 范式：反应模式
- 演化轨迹：6s 内 [前 1.5s 无意识闲聊]→[1.5-2.5s 听到叫声肩膀微紧]→[2.5-4s 回头眼神变化（瞳孔骤缩 + 嘴微张）]→[4-6s 错愕峰值后嘴角微动表情落定]
- 时序角色：图作中间关键帧（第 3 秒抵达图的画面状态）
- 传播因子：**眼神戏 + 错愕的"被定格" + 周围闺蜜的反应链**——网友会重看眼神部分
- 适用度：★★★★★（图本身就是为这一刻而存在的）

### 候选 B：回头·叫名喊话版（带"诶"音效设定）
- Hook 范式：反应模式
- 演化轨迹：[前 1s 主角闲聊]→[1-1.5s 画面外朋友喊名+主角听到的瞬间]→[1.5-3s 回头]→[3-5s 错愕表情 + 嘴部张开"诶？"]→[5-6s 收束转笑]
- 时序角色：图作中间关键帧（第 2.5 秒）
- 传播因子：**音效驱动的 hook**——"诶？"这个叫声本身就是抖音爆款的常用钩子
- 适用度：★★★★

→ **默认选 A**（眼神戏更适合反应模式的精细子节拍）

## 4. 第 2 轮：完整 Seedance Prompt（候选 A · 反应模式 v3）

```
6秒商K包厢内被叫到一瞬的回头错愕反应，9:16竖版，

参考@图片1的主体外貌服装+商K包厢氛围，图作为第3秒的中间关键帧——
0—1.5秒：主角坐在卡座一侧无意识地和左侧朋友闲聊，手举着饮料杯随节奏轻晃，
身体放松微微随BGM节拍点头，眼神看向左前方对话者；

1.5—2.5秒：画面外画右侧传来朋友叫她名字的声音，主角肩膀先微紧（屏息0.4秒），
头部开始向右后方转动，眼神还未到位但已经开始游移；

2.5—3秒：画面演化抵达@图片1所示状态——头部完全转向画面右后方，
瞳孔骤缩（瞳孔轻微缩小+眼角微张），嘴部从闭合状态轻微张开（约5毫米开口）；

3—4.5秒：错愕反应展开按子节拍——眉头微挑0.5秒+嘴部张开幅度增加到"诶？"的口型1秒，
眼神在错愕中聚焦到画面外叫她的人方向，整个面部呈现"被叫到的微惊讶"状态；

4.5—6秒：错愕情绪落定，嘴角开始微微上扬出现"原来是你"的笑意预兆，
肩膀放松回原位，画面收束在"惊讶过去转入笑意"的过渡瞬间。

环境同步演化：包厢内身边朋友必须有可见微动具体写——
左侧朋友看到主角被叫开始侧身往主角方向看一眼然后笑出声、
右后方叫主角的朋友身影模糊但能看到正举着手伸过来轻拍桌面、
前景桌面酒杯随握持动作传到桌面有轻微震动、
远处包厢深处其他卡座的朋友剪影各自举杯交谈不静止。

主角行为正向夸张+反向否定双重锁定：
- 回头动作必须明显能看出从"无意识"到"被叫到"的转折（不是平静转头是真的被吓到一下）
- 瞳孔骤缩要有明显视觉变化（不是表情没变化是眼睛真的有戏）
- 嘴部"诶？"口型必须张开能看到内部（不是抿唇微动是真的张开）

画质：手机闪光夜场抓拍的双重画质退化：
①微信短视频低码率压缩感、画面细节磨平、暗部色块、霓虹彩色色噪、压缩边缘抖动；
②夜场极弱光下闪光灯硬光过曝+室内中等ISO粗粒+霓虹白平衡漂移；
③霓虹绿紫反射次级光源+举杯握持随节拍微抖+霓虹打到主角面部时短暂过曝0.3秒；
④手机自拍广角畸变+画面边缘色散+主角脸部边缘轻微鱼眼变形。
禁用4K高清禁用Cinematic禁用电影级禁用稳定器质感禁用调色精致禁用工业渲染禁用画面锐利禁用清晰画质。

镜头单一缓慢推进Very Slow Push In（6秒内仅放大约15%，聚焦主角面部），
不可切镜不可拉远不可上升下降不可切角度，全程单一机位。

风格锁定：画质必须呈现明显的手机夜场闪光抓拍感不可呈现4K锐利画质不可呈现电影级稳定画质，
现场BGM贯穿全片不可寂静不可渐入不可中段才出现，
禁止任何文字、字幕、LOGO、水印，禁止主角着装出现透视/侧胸/紧身贴体特写镜头。
```

**字符数**：约 1380 字符（≤2000，余量充足）

## 5. 设计意图（v3 八条原则对照）

| 原则 | 本 prompt 落地 |
|------|--------------|
| ① 图作框架参考 | "图作为第 3 秒的中间关键帧"——反应模式下例外保留时序锚点机制（v3 第三原则的例外条款） |
| ② 凝视 3-4 自然小动作 | N/A，反应模式不适用此原则 |
| ③ 日常常识审计 | "回头动作物理可行不是同时面向两个方向"+周围朋友不可全静止 |
| ④ 4 层画质退化 | 微信压缩→闪光硬光→霓虹反射+握持抖→自拍广角畸变 |
| ⑤ 群演段内具体写 | 左侧朋友/右后方叫人朋友/前景桌面/远处包厢分别写 |
| ⑥ 主角行为正向+反向否定 | 三条独立否定锁定：转折/瞳孔/嘴部 |
| ⑦ 镜头运动锁死 | Very Slow Push In + 否定句穷举 |
| ⑧ 跨人物互动 | 叫主角名字 = 跨人物互动事件作为视频起点叙事骨架 |

**反应模式专属（第六原则）**：错愕子节拍展开 1.5s（屏息 0.4s + 头转动 1s + 瞳孔骤缩 + 嘴部张开），符合"惊吓 / 一惊"≥1.2s 的物理时间常数。

**前 2 秒生死线**：第 1.5 秒就有"画面外叫名"作为听觉钩子，第 2.5 秒抵达图的状态，6 秒内完整闭合。

**核心 hook**：第 3 秒的瞳孔骤缩 + 嘴部"诶？"——这是抖音爆款的标准眼神戏 hook。

**画质真实感**：商 K 包厢的夜场+霓虹+手机闪光自拍质感，不可被默认电影级稳定渲染。

**二刷钩子**：第 4.5-6 秒"错愕过渡到笑意"的微表情，会让观众想重看一遍确认是不是真的转笑。

**传播预期**：抖音/小红书都适合，标题方向"突然被叫到的我.jpg" / "这一秒的眼神戏" / "原来是你"。

## 6. 微调建议

如果想调整：
- **节奏更快**：把回头时间从 [1.5-2.5s] 压缩到 [1-1.8s]，错愕峰值放在第 2.5 秒
- **想要"诶？"音效更显**：把候选 B 的"画面外朋友喊名"显式写进画面段落（"听到朋友喊'XXX！'"，注意单引号）
- **想去掉酒桌道具降低合规风险**：把 prompt 改成"咖啡厅/餐厅"场景，画质退化层②③同步改为白光餐厅
- **改时长**：5s 适合更紧凑，8s 适合多展开错愕→笑意过渡
- **改成凝视模式**：不推荐，图本身就是反应瞬间，强行凝视会失去戏剧 hook
