# experiments — 研发与实测日志

这个目录记录 **Seedance 2.0 提示词方法论是怎么一步步踩坑迭代出来的**，是 `references/` 里那些方法论的「原始证据链」。不影响 Skill 运行，可独立阅读。

> 注：本目录原名 `handoff`（最初用于工作交接），现已演变为研发 / 实测记录，故更名为 `experiments`。

## 目录说明

| 路径 | 内容 |
|:---|:---|
| `summary.md` | 实测方法论总结（含即梦平台版权审查完整 SOP） |
| `cases/` | 30+ 个图生视频实测案例，每个含完整的 v2→v3→v4→v5 迭代轨迹 |
| `templates/` | 写 case / summary 用的记录模板 |

## 怎么读

- 想看**方法论结论** → 直接读 `summary.md`
- 想看**某类场景怎么调** → 翻 `cases/` 对应案例（编号大致按时间顺序）
