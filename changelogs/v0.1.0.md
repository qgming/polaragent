# PolarAgent v0.1.0

这是 PolarAgent 的第一个公开发布版本。这个版本的目标是提供一个本地优先、可扩展、能真正参与日常工作的桌面 AI Agent 工作台，把对话、工具调用、知识库、浏览器控制、桌面控制和多助手协作放到同一个应用里。

## 首发亮点

- 发布 PolarAgent 桌面应用，支持 Windows、macOS 和 Linux。
- 提供本地优先的 AI Agent 工作台，会话、配置、助手、团队和知识库数据默认保存在本机。
- 支持多模型供应商配置，可接入兼容 OpenAI 接口的模型服务以及其他已适配供应商。
- 支持普通对话、团队 Agent 协作、技能系统、工具管理、知识库问答和 MCP 扩展。
- 新增 Browser Use，让 Agent 可以连接真实 Chrome 浏览器会话，处理需要登录态的网页任务。
- 新增 Computer Use，让 Agent 可以通过 Windows UI Automation 观察并操作桌面应用。
- 支持 GitHub Releases 分发，并接入 Electron 官方更新服务相关能力。

## Agent 与对话

- 支持创建多个助手，并为不同助手配置模型、提示词、技能和工具权限。
- 支持流式输出，让模型响应更接近实时对话体验。
- 支持会话历史、本地保存、会话切换和全局会话搜索。
- 支持自动生成会话标题，降低长期使用时的整理成本。
- 支持 Markdown 内容渲染，包括代码块、表格、链接、列表等常见格式。
- 支持 Mermaid 图表、数学公式和代码高亮，适合研发、分析、写作和学习场景。

## 团队 Agent 协作

- 支持创建由多个助手组成的团队，让不同角色围绕同一任务协作。
- 支持团队会话和团队消息历史，方便回看协作过程。
- 支持团队监控面板，用于观察团队成员的行动轨迹和任务进展。
- 支持团队投票、团队控制和待办更新等工具，适合方案评审、需求拆解、复杂问题讨论和多角度决策。

## 工具系统

- 内置文件操作能力，包括读取、写入、追加、目录列表和文件状态检查。
- 内置 Shell 命令执行能力，可用于本地开发、脚本运行和环境诊断。
- 支持网页搜索，可配置 Tavily、Exa、Serper、SearXNG、Brave 等搜索服务。
- 支持网页读取和内容提取，方便从网页中获取结构化信息。
- 支持图片生成、语音识别、语音合成等多模态相关能力。
- 支持工具权限控制，让不同助手按需启用或禁用工具。

## Browser Use

- 新增 Browser Use 设置页和运行时能力。
- 支持通过 Chrome 扩展连接真实浏览器标签页。
- 支持保留本机浏览器 Profile、Cookie 和登录态，适合处理后台系统、控制台、表单页面和需要账号登录的网站。
- 支持查看扩展连接状态、可操作标签页数量、端口配置和错误状态。
- 支持从应用内导出 Browser Use 扩展文件夹，再在 Chrome 的 `chrome://extensions` 中加载。
- 支持为浏览器动作配置超时时间和动作后等待时间。

## Computer Use

- 新增 Computer Use 设置页和运行时能力。
- 支持通过 Windows UI Automation 读取桌面窗口和控件结构。
- 支持观察 UI 树、控件名称、控件类型、位置和窗口信息。
- 支持截图辅助，让模型在需要时获得视觉上下文。
- 支持点击、双击、移动、拖拽、滚动、输入文本、按键和批量动作。
- 支持列出窗口、激活窗口、聚焦控件、等待界面变化和查询元素详情。
- 支持常驻 Worker 配置，减少连续桌面动作的启动成本。

## 知识库

- 支持创建和管理多个本地知识库。
- 支持添加 TXT、Markdown、PDF、Word 等常见文档。
- 支持文件级管理、删除、重建和兼容性检查。
- 支持基于语义检索的知识库问答，让模型回答更贴近项目资料。
- 支持在对话中选择启用指定知识库，适合项目文档、产品资料、学习笔记和工作资料整理。

## MCP 与技能生态

- 集成 Model Context Protocol，支持配置外部 MCP Server。
- 支持发现 MCP Server 暴露的工具，并在 Agent 工具系统中调用。
- 支持内置技能、自定义技能和技能市场相关能力。
- 技能可为助手提供稳定的方法论、操作规程和上下文模板，适合沉淀专业工作流。

## 桌面体验

- 提供基于 Electron 的跨平台桌面应用。
- 提供无边框标题栏、侧边栏、窗口控制和任务监控面板。
- 支持亮色、深色和跟随系统主题。
- 支持设置页面集中管理模型、偏好、知识库、搜索、音频、图片生成、Browser Use、Computer Use 和关于信息。
- 支持本地数据目录管理，方便备份、迁移和排查配置。
- 关于页面支持检查更新，并通过更新弹窗展示新版本号和更新日志。

## 发布与更新

- Windows 提供 NSIS 安装包，并提供 Squirrel 产物用于 Electron 官方自动更新服务。
- macOS 提供 DMG 和 ZIP。
- Linux 提供 AppImage、deb、rpm 和 tar.gz。
- GitHub Release 正文会使用 `changelogs/v0.1.0.md` 作为本版本更新日志。
- 应用内更新弹窗会读取 GitHub Release 更新日志并展示给用户。

## 适合的使用场景

- 日常 AI 对话、资料整理、文档写作和知识问答。
- 研发辅助，包括代码理解、命令执行、技术调研和方案设计。
- 多角色协作，包括产品、研发、设计、分析等角色的模拟讨论。
- 浏览器任务，包括资料检索、网页阅读、后台系统操作和表单处理。
- 桌面自动化，包括窗口观察、控件操作和重复性 GUI 流程。

## 已知说明

- Computer Use 当前主要面向 Windows 桌面环境。
- Browser Use 需要先导出并加载 Chrome 扩展后才能控制真实浏览器标签页。
- 自动更新主要面向 Windows 和 macOS；Linux 版本会随 GitHub Release 提供安装包。
- 首发版本仍在持续打磨，欢迎通过 Issue 反馈问题和建议。
