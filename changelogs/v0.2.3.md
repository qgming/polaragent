# 更新日志

## [0.2.3] - 2026-06-17

### 🎉 新功能

#### 系统托盘与后台运行

- **系统托盘图标**：应用启动后自动在 Windows 状态栏右侧显示托盘图标，提示应用正在运行
- **右键托盘菜单**：包含「显示主窗口」和「退出软件」两个选项，退出时彻底关闭应用
- **左键托盘恢复**：点击托盘图标即可恢复并聚焦主窗口
- **关闭到托盘保留后台**：点击窗口关闭按钮（X）默认隐藏窗口到托盘，AI 任务可继续在后台运行，而非直接退出
- **偏好设置开关**：
  - 「关闭时最小化到托盘」：默认开启，关闭窗口时保留后台运行；关闭后点击 X 直接退出整个软件
  - 「启动时隐藏到托盘」：默认关闭，开启后应用启动不显示窗口，仅托盘图标可见
- **关闭按钮文案动态化**：关闭到托盘模式下显示「关闭」，直接退出模式下显示「退出」

#### 技能压缩包安装

- **从 .zip 安装技能**：技能安装对话框新增「从本地压缩包安装」选项，支持选择 .zip 文件直接安装 Skill
- **压缩包文件选择器**：新增 `dialog:pick-zip-file` 系统文件选择对话框，筛选显示 .zip 格式文件
- **安装对话框 UI 优化**：Git 安装和压缩包安装的输入区域独立布局，压缩包安装使用只读路径输入 +「选择文件」按钮

#### 技能卸载功能实现

- **卸载自定义技能**：实现 Skill 卸载逻辑，调用主进程删除技能目录并从内存中移除
- **保护内置技能**：内置技能（builtin）禁止卸载，操作时提示「无法删除内置技能」

### 🔧 优化

- **文件路径权限放开**：移除工具路径的工作目录前缀校验限制，AI 可访问用户有权限的任意文件路径，提升灵活性
- **托盘图标自动适配**：复用应用主图标（build/icon.ico），自动 resize 到 16x16 适配 Windows 托盘标准尺寸
- **配置变更实时生效**：关闭到托盘开关切换后无需重启应用，下次点击关闭按钮立即生效
- **退出时清理托盘**：应用退出时主动销毁托盘图标，防止 Windows 上图标残留

### 📝 技术细节

- 新增 `electron/lib/tray.cjs`：托盘核心逻辑（Tray 创建、上下文菜单、isQuitting 退出标志管理）
- 修改 `electron/lib/windows.cjs`：`createMainWindow` 注册 `close` 事件拦截器，支持 `_skipShow` 内部选项
- 修改 `electron/lib/app-paths.cjs`：新增 `readSettingCloseToTray` / `readSettingStartInSystemTray` 同步配置读取
- 修改 `electron/main.cjs`：新增 `before-quit` / `will-quit` 生命周期处理
- 修改 `electron/ipc/skills.cjs`：新增 `installSkillFromZip` 和 `uninstallSkill` 实现
- 修改 `src/types/config.ts`：`Settings.window` 新增 `closeToTray: boolean` 字段
- 修改 `src/ai/tools/tool-context.ts`：移除工作目录前缀校验，简化路径解析逻辑
