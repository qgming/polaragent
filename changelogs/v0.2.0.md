# 更新日志

## [0.2.0] - 2026-06-15

### 🎉 新功能

#### 办公文档创作模块

- **Word 文档生成**：支持通过 AI 工具创建 .docx 文件，使用 JSZip 在前端直接生成 OOXML 格式
- **PPT 演示文稿生成**：支持创建 .pptx 文件，先渲染 HTML 幻灯片再通过 Electron 截图打包为 PPTX，提供 12 种商务/创意风格模板（board、studio、data、pitch、magazine、ink、kraft、porcelain、forest、swiss-blue、lemon-grid、safety-orange）
- **PDF 文档生成**：支持创建 .pdf 文件，通过 Electron printToPDF 导出，提供 4 种专业排版模板（executive-brief、data-report、proposal、whitepaper）
- **统一 AI 工具接口**：新增 `create_office_document` 工具，AI 可通过单一接口创建 Word/PPT/PDF 三种格式文档

#### 文档附件理解

- **PDF 文档提取**：使用 pdfjs-dist 逐页提取 PDF 文本内容，Agent 可理解上传的 PDF 附件。（不支持图片型PDF）
- **Word 文档提取**：使用 mammoth 提取 DOCX 纯文本内容，Agent 可理解上传的 Word 附件
- **文档文件选择器**：对话输入工具栏新增"文档 (PDF/Word)"附件类型，支持选择 PDF 和 DOCX 文件

#### 文件预览

- **PDF 预览**：独立预览窗口支持 PDF 渲染，提供翻页、缩放、旋转、适合宽度等功能
- **Word 预览**：独立预览窗口支持 DOCX 文件预览，使用 mammoth 转 HTML 渲染
- **PPT 预览**：独立预览窗口支持 PPTX 文件预览，解析 XML 提取幻灯片文本和嵌入图片

#### 内置技能

- **pdf-generator 技能**：指导 AI 先设计语义化 HTML 再导出 PDF，含模板选择指南和页面布局策略
- **ppt-generator 技能**：指导 AI 先设计静态 HTML 幻灯片再导出 PPTX，含 12 种风格模板和多布局模式参考

### 🔧 改进

- 任务监控面板中的产物列表支持点击 PDF/DOCX/PPTX 文件直接预览
- AI Agent 工具摘要支持显示 `create_office_document` 的创建结果
- 附件芯片样式区分：文档附件使用橙色系，图片使用天蓝色系，文本/音频使用绿色系
- 工具权限审查新增 `create_office_document` 为默认允许的写入级工具
