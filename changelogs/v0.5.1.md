# 更新日志

## [0.5.1] - 2026-07-04

本次 0.5.1 是一次围绕界面细节、首屏体验与创作能力补强的补丁更新。除了修复深色模式显示异常、资源页面卡片与筛选区间距不一致、应用启动初期侧边栏短暂空白等问题外，还补入了面向短片/短剧写作的内置剧本技能，方便从创意、人物小传、故事梗概一路推进到场次分镜。

### 🎬 创作能力增强

- **新增内置 Skill `screenplay-storyboard`**：支持短片故事脚本、短剧项目包、场次分镜剧本、镜头级分镜四种核心产物，适合“短片剧本”“人物小传”“剧本摘要”“分镜剧本”“按案例扩写/改写”等场景
- **参考案例内置化**：将短片故事脚本案例与神话后传短剧案例沉淀到 Skill 的 `references/` 中，后续助手可按需读取并复用用户指定的结构、密度与文风
- **短剧制作师接入新技能**：内置 Agent `short-drama-producer` 新增启用 `screenplay-storyboard`，从原本偏图片资产与视频提示词的工作流，补齐到剧本策划与分镜写作链路

### 🎨 界面优化

- **知识库卡片重排**：知识库列表卡片改为更贴近定时任务卡片的结构，顶部操作区与信息区层次更清晰
- **移除知识库简介**：知识库卡片、新建弹窗、编辑弹窗统一移除“简介”字段，界面更简洁
- **筛选区间距统一**：技能、助手、工具、定时任务四个页面的筛选按钮/Tab 下方内容间距统一收敛为 `mt-3`，减少上下节奏不一致带来的松散感

### 🌙 深色模式修复

- **更新日志深色模式适配**：修复新版本弹窗中更新日志在深色模式下仍显示黑色文字的问题
- **统一 Markdown 主题变量**：HTML 格式的 release notes 现在复用项目现有的 markdown 预览主题，标题、正文、链接、列表与代码样式都能正确跟随深浅色切换

### ⚡ 启动体验优化

- **侧边栏加载态补全**：应用启动后，对话、项目、团队三个侧边栏 tab 在数据 hydrate 完成前显示加载中状态与旋转指示器
- **避免误判为空白**：修复左侧侧边栏在启动初期短暂空白、容易让人误以为没有数据的问题
- **会话 hydrate 状态显式化**：普通对话与团队会话 store 新增 hydration 生命周期状态，方便界面更准确地判断“正在加载”与“确实为空”

### 🐛 其他修复

- **版本元数据对齐**：同步更新应用版本号与锁文件顶层版本字段到 `0.5.1`
