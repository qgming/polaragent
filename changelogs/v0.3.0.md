# 更新日志

## [0.3.0] - 2026-06-18

### 🎉 新功能

#### 全局记忆：让 PolarAgent 越用越懂你

- **长期记忆系统**：新增全局记忆能力，PolarAgent 可以沉淀用户偏好、身份画像、长期目标、沟通风格、工作流习惯、工具偏好和历史纠正
- **全局记忆 + 项目记忆双层结构**：支持保存跨会话通用的用户记忆，也支持把技术栈、目录约定、项目规则等上下文限定在当前项目内，避免项目 A 的信息污染项目 B
- **自动写入记忆**：每轮助手回复完成后，后台自动从最近对话中提取值得长期保存的信息；没有可记内容时不会写入
- **显式工具召回**：记忆不会自动塞进每轮提示词，AI 只在需要了解偏好、身份、历史约定或项目上下文时通过 `search_memory` 主动检索，减少上下文污染
- **记忆管理工具**：新增 `search_memory`、`remember_memory`、`forget_memory` 三个 AI 工具，并加入「长期记忆」工具分组，可在工具页统一开关
- **本地向量检索**：记忆复用现有嵌入配置生成向量，支持相似度检索、TopK 返回、阈值过滤和向量去重合并
- **敏感内容过滤**：自动跳过密码、API Key、token、验证码、身份证号、银行卡等高风险内容，避免把敏感信息写入长期记忆

#### 全局记忆设置页

- **新增「全局记忆」设置面板**：设置页新增全局记忆入口，集中管理记忆总开关、自动写入、项目记忆、嵌入配置复用、敏感过滤和检索参数
- **记忆管理列表**：支持搜索、按范围筛选、按分类筛选、编辑、关闭、删除和重建索引
- **编辑记忆弹窗**：记忆编辑改为模态窗，使用通用下拉组件选择分类，体验与知识库和设置页其它控件保持一致
- **关闭而非归档**：记忆可通过开关关闭；关闭后仍保留在本地，但工具不会检索到这条记忆
- **索引状态可视化**：列表中已索引记忆显示绿色状态点，未索引和已关闭记忆显示清晰状态标签
- **更多默认分类**：记忆分类扩展为偏好、画像、项目、指令、纠正、沟通风格、工作流、工具偏好、长期目标、约束边界

### 🔧 优化

- **设置页左侧导航滚动**：设置页左侧导航支持独立上下滚动，入口变多后仍能稳定访问所有设置项
- **记忆行操作按钮优化**：记忆管理列表右侧操作改为参考技能页和工具页的「图标 + 文字」按钮，编辑、删除操作更直观
- **记忆重建语义优化**：重建索引会为全部记忆生成向量，关闭状态只影响检索，不会导致重新开启后缺少索引
- **项目记忆隔离优化**：记忆检索会结合当前工作目录生成项目 key，只返回当前项目相关的项目记忆
- **错误处理更温和**：自动写入失败不会影响正常聊天，只记录最近错误并在设置页展示
- **工具输出更克制**：记忆检索结果只返回必要摘要、scope、type、score、source 信息，避免大段记忆污染对话上下文
