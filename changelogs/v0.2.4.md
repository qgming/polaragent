# 更新日志

## [0.2.4] - 2026-06-18

### 🎉 新功能

#### 多语言英文界面

- **完整英文界面支持**：为应用主要页面、设置面板、对话界面、资源管理页面补齐英文翻译
- **多命名空间语言包**：新增并扩展 `agents`、`knowledge`、`skills`、`tools`、`team`、`tutorial` 等语言资源
- **语言即时切换**：保持现有语言切换机制，英文与简体中文界面可即时切换
- **语言选项真实名称**：语言选择器中始终显示 `简体中文` 与 `English`，符合用户对语言名称的直觉

#### 教程内容多语言化

- **教程页面英文翻译**：快速开始、对话功能、助手管理、技能使用、工具集成、团队协作、知识库配置、模型设置、Browser Use、Computer Use、使用技巧、常见问题均支持英文内容
- **教程导航本地化**：左侧教程分组、教程条目、返回应用按钮、提示标签全部接入语言包
- **结构化教程渲染**：新增统一的教程文章渲染器，教程内容从语言资源读取，便于后续继续扩展其他语言

### 🔧 优化

- **通用组件文案本地化**：补齐确认弹窗、全局搜索、错误边界、预览窗口、任务监控、工作区文件树、更新说明弹窗等公共组件翻译
- **对话体验文案本地化**：补齐工具栏、权限模式菜单、Agent 轨迹、消息渲染、语音录制、目标编辑与目标状态区域翻译
- **设置页文案本地化**：补齐偏好设置、模型设置、音频设置、图片生成、网络搜索、知识库、Browser Use、Computer Use、关于页面等设置项翻译
- **资源管理页面本地化**：补齐助手、技能、工具、团队、知识库页面的标题、空状态、操作按钮、编辑弹窗和状态提示
- **英文界面一致性提升**：清理英文模式下残留的用户可见中文，使界面语言更统一

### 📝 技术细节

- 新增 `src/i18n/locales/en-US/tutorial.json` 与 `src/i18n/locales/zh-CN/tutorial.json`
- 修改 `src/i18n/config.ts`：将 `tutorial` 加入命名空间列表
- 修改 `src/i18n/index.ts`：静态导入并注册教程语言包
- 新增 `src/components/tutorial/TutorialArticle.tsx`：统一渲染教程标题、段落、列表、提示和内联样式
- 精简 `src/components/tutorial/*Guide.tsx`：各教程组件改为对应教程文章入口
- 修改 `src/pages/TutorialPage.tsx`：教程导航改为使用 `tutorial` 命名空间
- 修改 `src/components/settings/PreferencesPanel.tsx`：语言选项改为从语言资源读取，并保持真实语言名称显示
